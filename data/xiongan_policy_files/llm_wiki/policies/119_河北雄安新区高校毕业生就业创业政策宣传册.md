## 政策简介

本政策旨在促进河北雄安新区高校毕业生多渠道就业创业，涵盖企业吸纳就业激励、基层就业支持、自主创业帮扶、能力提升培训等多个方面。政策提供一次性吸纳就业补贴、扩岗补助、社保补贴、创业担保贷款等多种资金支持，助力高校毕业生实现就业创业梦想。

## 支持内容

1. 一次性吸纳就业补贴：每人1000元。
2. 一次性扩岗补助：每人1500元。
3. 一次性新增就业补贴：新区注册服务型企业1000元/人，其他类型企业及参建企业500元/人。
4. 社会保险补贴：企业吸纳按实际缴纳社保费补贴（不含个人部分），最长3年；灵活就业最高不超过实际缴费2/3，最长2年。
5. 就业见习补贴：离校2年内未就业高校毕业生1900元/月，16-24岁失业青年为最低工资标准60%，留用率超50%提高10%，最长12个月。
6. 职业培训补贴：参照河北省目录及标准执行。
7. 职业技能鉴定补贴：参照全省收费标准执行。
8. 一次性求职创业补贴：每人2000元。
9. 一次性创业补贴：每个创业项目10000元。
10. 创业担保贷款：个人/合伙最高20万元，小微企业最高300万元，财政给予部分贴息。

## 申请对象

- 中小微企业（吸纳毕业年度或离校2年内未就业高校毕业生）
- 新区注册服务型企业、其他类型企业、参建企业（吸纳新区三县户籍高校毕业生）
- 人社部门认定的见习单位
- 小微企业（含社会组织）
- 开展职工岗位技能培训、技师培训、新型学徒制培训的企业
- 毕业年度高校毕业生
- 离校2年内未就业高校毕业生
- 登记失业的16-24岁青年
- 河北籍全日制大专以上离校2年内未就业高校毕业生
- 毕业学年及毕业5年内高校毕业生（自主创业）
- 困难毕业生（低保家庭、贫困残疾人家庭、脱贫家庭、特困人员、残疾、烈士子女、享受国家助学贷款）
- 脱贫家庭子女、城乡未继续升学应届初高中毕业生、农村转移就业劳动者、城镇登记失业人员

## 申报条件

### 硬性条件

- 一次性吸纳就业补贴：招用毕业年度高校毕业生，签订1年以上劳动合同，按规定缴纳社保，主体为中小微企业。
- 一次性扩岗补助：招用2022年1-12月毕业普通高校毕业生，签订劳动合同，缴纳失业保险费1个月以上。
- 一次性新增就业补贴：2022年5月1日-12月31日期间新招用新区三县户籍高校毕业生；服务型企业/其他类型企业需签订6个月以上合同并提供3个月工资凭证；参建企业需录入劳务实名制监管系统。
- 社会保险补贴（企业）：招用离校2年内未就业高校毕业生，签订1年以上劳动合同，缴纳社保，主体为中小微企业。
- 就业见习补贴：吸纳河北籍全日制大专以上2年择业期内离校未就业高校毕业生或16-24岁失业青年，按月发放基本生活费，主体为人社部门认定的见习单位。
- 职业培训补贴：企业新录用人员签订1年以上合同，1年内参加培训并取得证书；在职职工参加新型学徒制/技师培训并取得证书。
- 职业技能鉴定补贴：通过初次职业技能鉴定，取得职业资格证书或等级证书。
- 灵活就业社保补贴：毕业年度和离校2年内未就业高校毕业生，实现灵活就业，办理就业登记并缴纳社保。
- 一次性求职创业补贴：毕业学年有就业创业意愿并积极求职创业，属于低保、贫困残疾人、脱贫、特困、残疾、烈士子女或享受国家助学贷款的高校毕业生/中职毕业生。
- 一次性创业补贴：毕业学年及毕业5年内高校毕业生，初次创业，取得营业执照，正常运营6个月以上，非国家限制行业。
- 创业担保贷款：小微企业当年新招用符合条件人员人数达到一定比例。

### 方向性条件

- 就业见习单位留用率超过50%，补贴标准提高10%。
- 政策聚焦高校毕业生求职创业需求，提供不断线就业服务。

### 排除条件

- 一次性扩岗补助与一次性吸纳就业补贴政策不重复享受。
- 一次性新增就业补贴一年内同一人只可申领一次。
- 一次性新增就业补贴对象不包括从事个体经营或创办小微企业和在民政部门注册的社会组织人员。
- 一次性创业补贴国家限制的行业除外。
- 职业技能鉴定补贴不含培训合格证。

## 材料与流程

### 申报材料

- 一次性吸纳就业补贴：签订劳动合同证明、社保缴费证明。
- 一次性扩岗补助：劳动合同、失业保险缴费证明（免申即享或自行申请）。
- 职业培训补贴：培训机构开具的税务发票；技师/新型学徒制培训前备案需培训计划、人员花名册、劳动合同复印件。
- 社会保险补贴：社保缴费证明（通过人社一体化公共服务平台提供）。
- 就业见习补贴：基本生活费发放证明、人身意外伤害保险办理证明。
- 一次性新增就业补贴：服务型企业/其他类型企业需劳动合同、3个月工资发放记账凭证或银行代发工资对账单；参建企业需合同、考勤、3个月以上工资凭证（需录入系统）。
- 职业技能鉴定补贴：免费技能鉴定协议书、职业资格证书/等级证书。
- 一次性求职创业补贴：《求职补贴审核认定表》、申请人员名册、困难证明材料（低保、残疾、脱贫等）、学籍证明。
- 一次性创业补贴：营业执照、稳定经营6个月证明材料。

### 办理流程

- 一次性吸纳就业补贴：用人单位申请 -> 人社部门审核、公示 -> 支付至企业基本账户。
- 一次性扩岗补助：免申即享主动发放 或 企业自行申请 -> 经办机构审核 -> 支付。
- 职业培训补贴：培训前备案 -> 提供发票申请 -> 人社部门审核 -> 支付至企业和培训机构账户。
- 社会保险补贴：先缴后补 -> 登录平台提供证明 -> 审核、公示 -> 支付至基本账户。
- 就业见习补贴：线上/线下申请 -> 初审、公示 -> 财政部门审核 -> 拨付至基本账户。
- 一次性新增就业补贴：产业互联网平台注册发布岗位 -> 签订合同录入平台 -> 向人社部门申请。
- 职业技能鉴定补贴：签订免费协议 -> 考核机构提交申请 -> 审核通过直接拨付考核机构。
- 灵活就业社保补贴：线上/线下申请 -> 初审、公示 -> 财政部门审核 -> 拨付至社保卡账户。
- 一次性求职创业补贴：高校代为申请 -> 院校初审公示 -> 人社部门审核 -> 支付至社保卡或高校账户。
- 一次性创业补贴：经营6个月后12个月内登录平台申请 -> 审核、公示 -> 支付至社保卡账户。

## 匹配时需要补充的信息

- 企业是否属于中小微企业
- 企业注册地是否在雄安新区
- 企业是否为新招用高校毕业生
- 高校毕业生户籍是否为新区三县
- 企业是否已缴纳社会保险/失业保险
- 企业是否已录入雄安新区产业互联网平台或劳务实名制监管系统
- 企业是否为人力资源社会保障部门认定的见习单位
- 企业是否开展职业技能培训或鉴定
- 创业者是否为毕业5年内高校毕业生
- 创业者是否初次创业且正常运营6个月以上
