# 软件和人工智能企业现在能关注哪些雄安政策？

## 回答

**简短结论：**
目前（2026年5月）针对软件和人工智能企业，**暂无处于“当前可关注”申报窗口期的专项资金或项目政策**。2025年度的人工智能科技攻关项目已过期。建议重点关注**长期有效的产业指导意见**、**人才支持政策**以及**专精特新/高新技术企业认定**等通用类政策，并留意后续发布的实施细则。

### 推荐关注

1.  **数字经济与人工智能顶层设计（长期有效/未明确期限）**
    *   **政策：** 《关于全面推动雄安新区数字经济创新发展的指导意见》
    *   **适用性：** 明确支持人工智能算法头部企业、云计算、大数据等领域的金融科技企业及创新资源。
    *   **来源：** `policies/130_河北雄安新区管理委员会印发《关于全面推动雄安新区数字经济创新发展的指导意见》的通知` (Source 2, Source 9)
    *   **行动建议：** 该政策为定期征集新产品、新技术、新服务，建议企业定期在雄安新区产业互联网平台进行自评估或提交成果，争取纳入支持范围。

2.  **未来产业与独角兽/瞪羚企业培育（长期有效/未明确期限）**
    *   **政策：** 《雄安新区促进未来产业发展行动计划》
    *   **适用性：** 覆盖科技类企业、创新型中小企业、独角兽企业、瞪羚企业。软件和AI企业若符合高成长性特征，可关注此类认定带来的资源倾斜。
    *   **来源：** `policies/050_关于印发《雄安新区促进未来产业发展行动计划》的通知` (Source 1, Source 2)

3.  **人才聚集措施（长期有效/未明确期限）**
    *   **政策：** 《关于打造创新高地和创业热土聚集新人才的若干措施》
    *   **适用性：** 针对高端高新产业领域（含新一代信息技术）的技术研发等特殊岗位引进的急需紧缺人才。
    *   **来源：** `policies/086_中共河北雄安新区工作委员会 河北雄安新区管理委员会《关于打造创新高地和创业热土聚集新人才的若干措施》` (Source 2)

4.  **专精特新“小巨人”认定（当前可关注，但需确认资格）**
    *   **政策：** 《关于开展2026年专精特新“小巨人”企业认定和复核推荐工作的通知》
    *   **状态：** 当前可关注（线上填报截止2026年5月25日）。
    *   **适用性：** 虽然主要面向制造业，但“制造业和信息化相融合的企业”也可申报。纯软件企业需确认是否符合河北省专精特新中小企业前置条件。
    *   **来源：** `policies/003_关于开展2026年专精特新“小巨人”企业认定和复核推荐工作的通知` (Source 4, Source 7, Source 10)

### 暂不建议

1.  **2025年度人工智能科技攻关项目**
    *   **政策：** 《关于征集2025年度雄安新区本级科技攻关项目（人工智能领域）的通知》
    *   **原因：** 状态为**已过期**（征集截止2025年9月22日）。
    *   **来源：** `policies/023_关于征集2025年度雄安新区本级科技攻关 项目（人工智能领域）的通知` (Source 1, Source 2, Source 3)
    *   **注意：** 2026年度的同类项目（如中央引导地方科技发展资金雄安场景成果转化）目前显示为“延期”或“已过期”（Source 4, ID 005, 008），具体2026年新一轮AI专项申报时间**需回查原文或人工核实**，目前无明确开放窗口。

2.  **机器人产业专项（若企业非机器人领域）**
    *   **政策：** 《雄安新区支持机器人产业发展的若干措施》
    *   **状态：** 当前可关注（有效期至2026年12月31日）。
    *   **原因：** 仅适用于机器人产业链相关企业。若贵司为纯软件/AI算法且无机器人硬件或集成应用，则不适用。
    *   **来源：** `policies/054_雄安新区支持机器人产业发展的若干措施` (Source 5, Source 6)

### 需要补充的信息

为了更精准地匹配政策，请补充以下信息：
1.  **企业注册地：** 是否在雄安新区注册？（多数政策要求新区注册或实质性运营）
2.  **企业规模与资质：** 是否为科技型中小企业、高新技术企业或已入库专精特新培育库？
3.  **业务细分：** 是纯软件/AI算法，还是涉及智能硬件、机器人、智能网联汽车或北斗应用？（不同细分领域对应不同政策，如ID 009北斗案例征集虽已过期，但类似活动可能重启；ID 076智能网联汽车措施长期有效）。
4.  **人才需求：** 是否有引进高端紧缺人才的需求？（可匹配ID 086人才政策）

**建议下一步行动：**
*   登录**雄安新区产业互联网平台**，关注“科技项目综合服务平台”板块，等待2026年度新一轮科技攻关或场景应用项目的申报通知。
*   若企业符合“制造业+信息化”融合特征，立即准备**2026年专精特新“小巨人”**申报材料（截止5月25日）。
*   梳理企业核心技术，对照《数字经济创新发展指导意见》进行成果梳理，以备定期征集。
