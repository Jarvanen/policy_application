# 关于开展 2026 年度雄安新区“特岗特薪”特聘岗位申报工作的通知

各重点用人单位：

根据《雄安新区“特岗特薪”工作方案（试行）》（雄人才发〔2023〕1 号）相关规定，支持用人单位以“特岗特薪”方式引进事业发展需要的高层次和高技能人才，现开展2026 年度特聘岗位申报工作。有关事宜通知如下：

# 一、岗位设置及奖励标准

# （一）设岗条件

申请设置特聘岗位应同时符合以下条件：

1.用人单位应为新区党政机关或在新区依法注册登记、实质性运营且具有独立法人资格的企事业单位。对疏解央企总部及二三级企业或创新业务板块，新一代信息技术、现代生命科学和生物技术、新材料以及空天信息、绿色能源、金融科技等高端高新产业领域重点企业、科研院所、新型研发机构、重大创新平台等给予重点支持。
2.引进人才应为本单位事业发展所急需紧缺的高层次、高技能人才。一般应在本行业本领域取得重大基础研究进展、前沿技术突破、解决重大工程技术难题，或具有自主知识产权核心技术、

有较强科技创新成果转化能力，或在技术创新、工艺改造、产业优化升级、经济社会发展中作出突出贡献。

3.用人单位现有常设岗位体系、薪资水平无法满足所需引进相应层次人才需求，需设置首席科学家、首席技术官、首席信息官、首席架构师等特聘岗位。原则支持设置科技、产业、研发类关键岗位。

上述情形之外，根据工作需要和事业发展确需设置特聘岗位的，可“一事一议”。特聘岗位不受本单位常聘岗位总量、工资总额、最高等级和结构比例限制。事业单位特聘岗位设置程序按现行干部人事管理制度执行。

# （二）岗位类别

特聘岗位分为 A、B、C 三档。任职条件应高于本单位或本行业相应常设岗位的要求。

A 档特聘岗位聘用对象应处于本领域、本专业世界一流或国内顶尖水平，做出系统的、创造性成就和重大贡献，其研究成果、业绩成就已获得国内外学术界或业内的广泛认可和尊崇。

B 档特聘岗位聘用对象应处于本领域、本专业国际先进或国内领先水平，取得较为优秀的成果，显示出较强的业界影响力，其研究成果、业绩成就已获得国内外学术界或业内认可。

C 档特聘岗位聘用对象应在本领域、本专业有一定影响力，取得有一定影响力的研究成果、业绩成绩，或为具有发展潜力的

青年骨干人才。

聘用对象应符合相应档次岗位基本条件（见附件），结合其业绩成果、经济社会贡献、社会公认度等因素进行综合评价、最终认定。

# （三）奖励标准

新区党工委人才办根据聘用对象的岗位类别按用人单位给付薪资的50%\~100%给予奖励。

1.特聘 A 档人才按用人单位给付薪资的 100%给予岗位奖励；
2.特聘 B 档人才按用人单位给付薪资的75%给予岗位奖励；
3.特聘 C 档人才按用人单位给付薪资的50%给予岗位奖励。

应全职在雄安新区工作，聘期 3 年。人才在聘期内类别上升的，自认定之月起按新类别奖励标准发放。

# 二、实施步骤

# （一）岗位设置申请

新区党工委人才办会同新区产业、科技等部门，调研重点用人单位“特岗特薪”需求，支持重点用人单位引进高端人才。用人单位根据事业发展和工作需要，经单位集体研究后，向新区党工委人才办提出特聘岗位设置申请，明确岗位名称、设岗理由、岗位职责、聘期目标任务、薪资待遇、考核管理制度等。

# （二）岗位核准认定

接到用人单位申请后，新区党工委人才办会同行业主管部门进行审核，出具认定意见，并按程序报新区管委会核准。核准通过的，设岗单位可设置特聘岗位。对于材料不齐的，应告知设岗单位一次性补正，设岗单位应在 5 个工作日内提交补正材料，逾期不补正的自动视为放弃申报。

# （三）设岗单位推荐人选

设岗单位自主开展人才招聘，对岗位拟聘用人才材料加具推荐意见，连同相关佐证材料报新区党工委人才办。

# （四）资格审查

由新区党工委人才办对设岗单位推荐人选材料进行资格审查。审查通过的，列为相应类别聘用对象。

# （五）公示和签约

设岗单位应在收到结果反馈后 5 个工作日内，对本单位拟聘用对象进行公示，公示期为5 个工作日（有特殊规定的从其规定）。公示无异议或异议不成立的，由设岗单位和拟聘用对象签订协议（协议应明确岗位聘期、任务内容、约束性考核目标、退出方式、失信惩戒等相关事项），确定为相应岗位聘用对象，并报新区党工委人才办备案。

# （六）资金拨付

聘用对象正常薪资由用人单位按原渠道发放，新区党工委人才办根据考核评估结果按年度发放岗位奖励资金。岗位聘用后，自签订协议次月起享受相应类别岗位奖励，岗位奖励按实际履职月份计算，设岗单位应科学制定考核标准，对聘用对象的履约情况进行年度考核。任期届满的，设岗单位应对聘用对象的任期履约情况开展绩效评估。年度考核结果和任期绩效评估结果应向新区党工委人才办备案。年度考核不合格的，停发其当年度岗位奖励。连续两次年度考核不合格的，由用人单位对聘用对象进行解聘，终止其岗位奖励。任期绩效评估不合格的，停发最后一年年度岗位奖励，不得再列为聘任人选。

# 三、其他事项

1.集团企业的总部、分公司和子公司之间或内部的人员调动，不纳入岗位奖励范围；各类用人主体之间，存在层级管理关系、投资关系、连锁经营关系等其他利害关系的情形，其人员流动不纳入岗位奖励范围。
2.聘用对象主动提出解聘或更换工作单位并经设岗单位同意的，终止其岗位奖励，按当年度实际履职时间计算奖励；聘用对象未经设岗单位同意更换工作单位，或存在其他重大问题的，终止其岗位奖励，不再发放当年度岗位奖励。
3.设岗单位原则上应在每年 11 月底前，形成本单位“特岗特薪”聘用人才岗位绩效自评报告，报送新区党工委人才办备案。

# 四、申报要求

申报单位结合本单位实际提出特聘岗位设置申请，填写《雄安新区“特岗特薪”特聘岗位申报表》，加盖单位公章。每个单位最多申报3个岗位，于2026年3月23日18:00前将电子版（word版及盖章后 PDF 扫描版，首页盖章+骑缝章）发送至邮箱dqrs603@163.com（文件名：特岗特薪+单位名称）。
