# 关于支持现代生命科学和生物技术产业发展的若干措施（试行）实施细则

# 第一章 总则

第一条 为贯彻落实《关于支持现代生命科学和生物技术产业发展的若干措施（试行）》（雄安办字〔2023〕51号）有关内容，制定本实施细则。

第二条 本实施细则规定的资金由雄安新区财政预算资金安排，由责任部门从部门年度预算资金列支。资金使用应当符合财政预算管理有关规定，坚持公开、公平、公正，确保安全和高效。支持资金要确保专款专用，不得重复申报补贴。

第三条 新区改革发展局根据本实施细则编制预算，组织项目评审，做好项目跟踪服务。新区财政支付中心负责专项支持资金拨付。

# 第二章 鼓励研发创新投入

第四条 支持在雄安新区实质运营的医疗机构、科研院所、企业等加大研发创新投入，对承担国家、河北省科技计划项目且符合雄安新区发展方向的，给予最高1:1的资金配套，每个项目年度支持金额最高为2000万元。

第五条 认定标准

（一）在雄安新区设立的医疗机构、科研院所、生物医药企业等独立法人单位，疏解医疗机构、高等院校的雄安新区院(校)区。

（二）承担国家、省级部门发布的科技计划项目，开展生物医药健康领域基础研究、成果转化、产业化、公共卫生安全、医疗健康数据等相关的研发创新。

（三）申请主体需在雄安新区有固定运营场所，工作人员在雄安新区缴纳社保或个税。

（四）上述单位成功申报国家级科技计划项目，新区按照实际获得专项经费1:1配套资金支持；对上述单位成功申报河北省级科技计划项目，新区按照实际获得专项经费1:0.5配套资金支持，最高支持金额为2000万元。上级资金及雄安新区配套资金合计不超过项目总经费。

# 第六条 申报材料及申报流程

# （一）申报材料

1. 申请项目报告、申请补贴资金申报表，加盖公章。
2. 获得国家、省级相关部门科技计划项目资金支持并提供认定证明材料（如文件、名单等）。
3. 申报主体在雄安新区运营场所证明及工作人员社保缴费证明或个人纳税证明。
4. 企业营业执照复印件，课题经费使用情况说明。

5.申请资金承诺书。
6.其他需要提供的资料。

# （二）申报流程

1. 申报。申报单位应根据本实施细则以及雄安新区产业互联网平台发布的申报通知（每年不少于一次），编写申请材料并上传。
2. 审核。新建片区管委会及三县行业主管部门对企业申请材料进行初审，并对报送材料的合规性、真实性负责，在规定的时间内将推荐企业齐全的申请材料和推荐意见上报雄安新区改革发展局。
3. 复核。雄安新区改革发展局应于收到材料后 10 个工作日内完成复核，对复核通过的，形成拟认定名单进行公示，公示期不少于 5 个工作日。
4. 认定。公示无异议的，雄安新区改革发展局按程序拨付支持资金。

# 第三章 支持创新平台建设

第七条 对国家医学中心、国家中医药医学中心、国家临床医学中心等国家级创新载体，予以最高 3000 万元资金支持。

# 第八条 认定标准

（一）取得国家有关部委认定的国家医学中心、国家中医药医学中心、国家临床医学中心等国家级创新载体批复。

（二）按经批复的建设方案完成实际固定资产投资额，原则上按完成固定资产投资额不高于 $30\%$ 且不超3000万给予一次性资金支持。

# 第九条 申报材料及申报流程

# （一）申报材料

1. 申请项目报告、申请补贴资金申报表，加盖公章。
2. 获得国家有关部委认定为国家级创新载体并提供认定证明材料（如文件、名单等）。
3. 会计师事务所出具的项目实际固定资产投资专项审计报告（含分年度投入明细及发票汇总表）。
4. 申请资金承诺书。
5.其他需要提供的资料。

# （二）申报流程

1. 申报。申报单位应根据本实施细则以及雄安新区产业互联网平台发布的申报通知（每年不少于一次），编写申请材料并上传。
2. 审核。新建片区管委会及三县行业主管部门对企业申请材料进行初审，并对报送材料的合规性、真实性负责，在规定的时间内将推荐企业齐全的申请材料和推荐意见上报雄安新区改革发展局。
3. 复核。雄安新区改革发展局应于收到材料后 10 个工作日

内完成复核，对复核通过的，形成拟认定名单进行公示，公示期不少于5个工作日。

4. 认定。公示无异议的，雄安新区改革发展局按程序拨付支持资金。

第十条 对新认定的国家级、省级的创新平台（学科重点实验室、企业重点实验室、技术创新中心、企业技术中心、工程研究中心、工业设计中心），按照实际获得国家、省级专项经费1:1配套资金支持，配套资金原则上不超过1000万元。

# 第十一条 认定标准

（一）创新平台属于现代生命科学与生物技术产业领域且平台主体为雄安新区独立法人机构或疏解医疗机构、高等院校的雄安新区院（校）区。
（二）自政策印发以来新认定的国家级、省级的创新平台，包括学科重点实验室、企业重点实验室、技术创新中心、企业技术中心、工程研究中心、工业设计中心等。

第十二条 本条政策采用“免申即享”方式，由新区行业主管部门自国家或省级资金下达后按1：1配套。

# 第四章 加快产业生态布局

第十三条 对独立自建产业化基地，且当年新增固定资产投资额（土地购置费用除外，下同）超过1亿元，按照新增固定资产投资额20%的比例，给予不超过2000万元的资金奖励，单个

项目补贴年限不超过3年。

# 第十四条 认定标准

（一）产业化基地主要用于提供药品和医疗器械等现代生命科学和生物技术领域技术研发、小试中试、成果转化、产业化等功能。
（二）申报单位应为雄安新区独立法人单位，并为产业化基地产权的实际拥有人。
（三）申报当年新增固定资产投资额超过1亿元，包括生产厂房、生产设备、生产辅助设备、研发和检测设备等资产投资（不含土地购置费用）。
（四）按照申报当年新增固定资产投资额 20% 的比例，给予不超过 2000 万元的资金奖励。
（五）自申报当年起，单个项目补贴年限不超过3年。

# 第十五条 申报材料及申报流程

# （一）申报材料

1. 申请项目报告、申请补贴资金申报表，加盖公章。
2. 企业营业执照复印件、会计师事务所出具的专项审计报告，加盖公章（含企业年度财务报表、采购合同及清单、发票明细等）。
3. 项目立项或备案证明文件。
4. 国有土地使用权出让合同复印件。

5.申请资金承诺书。
6.其他需要提供的资料。

# （二）申报流程

1. 申报。申报企业应根据本实施细则以及雄安新区产业互联网平台发布的申报通知（每年不少于一次），编写申请材料并上传。
2. 审核。新建片区管委会及三县行业主管部门对企业申请材料进行初审，并对报送材料的合规性、真实性负责，在规定的时间内将推荐企业齐全的申请材料和推荐意见上报雄安新区改革发展局。
3. 复核。雄安新区改革发展局应于收到材料后 10 个工作日内完成复核，对复核通过的，形成拟认定名单进行公示，公示期不少于 5 个工作日。
4. 认定。公示无异议的，雄安新区改革发展局按程序拨付奖励资金。

第十六条 对在雄安新区新设立且实缴注册资本在1亿元以上的相关生物医药类企业，给予最高不超过1000万元的开办补贴，相关经费分3年按 $40\%$ 、 $30\%$ 、 $30\%$ 比例拨付。

# 第十七条 认定标准

（一）在雄安新区实缴注册资本在1亿元以上的具有独立法人资格的生物医药企业。

（二）按照不超过实缴注册资本 5%，给予最高不超过 1000 万元的一次性开办补贴，实缴注册资本以企业申报当月的注册资本金额为准。
（三）经认定后经费分3年按 $40\%$ 、 $30\%$ 、 $30\%$ 比例拨付，且分别不超过400万元、300万元、300万元。

# 第十八条 申报材料及申报流程

# （一）申报材料

1. 申请项目报告、申请补贴资金申报表，加盖公章。
2. 企业营业执照复印件、法定验资机构出具的验资报告，加盖公章。
3. 申请资金承诺书。
4.其他需要提供的资料。

# （二）申报流程

1. 申报。申报企业应根据本实施细则以及雄安新区产业互联网平台发布的申报通知（每年不少于一次），编写申请材料并上传。
2. 审核。新建片区管委会及三县行业主管部门对企业申请材料进行初审，并对报送材料的合规性、真实性负责，在规定的时间内将推荐企业齐全的申请材料和推荐意见上报雄安新区改革发展局。
3. 复核。雄安新区改革发展局应于收到材料后 5 个工作日内

完成复核，对复核通过的，形成拟认定名单进行公示，公示期不少于5个工作日。

4.认定。公示无异议的，雄安新区改革发展局按程序拨付支持资金。

# 第五章 推动生物医药产业园区建设

第十九条 对新被认定为国家级、省级、新区级生物产业示范基地（包括产业园区等），自认定当年起，按照年度运营经费20%给予补贴，每年不超过500万元。

# 第二十条 认定标准

（一）申报单位应为新被认定为国家级、省级、新区级生物产业示范基地、产业园区运营主体单位。
（二）新区级生物产业示范基地需具备专业园区管理机构，具备完善的供水、供电、供热、供气等公共基础设施，具备污水、危废合规处置能力等；载体空间原则上不少于10000平方米，其中生物医药实验室及制造空间不少于50%，生物医药企业占比不少于80%。
（三）运营费用是指维持示范基地正常运转所必需的成本费用支出，包括水电费、维修费、培训及科创服务费用等。

# 第二十一条 申报材料及申报流程

# （一）申报材料

1. 申请项目报告、申请补贴资金申报表，加盖公章。

2. 运营主体营业执照复印件、会计师事务所出具的专项审计报告，加盖公章。
3. 申请资金承诺书。
4. 国家级、省级、新区级生物产业示范基地（包括产业园区等）认定材料。
5.其他需要提供的资料。

# （二）申报流程

1. 申报。申报企业应根据本实施细则以及雄安新区产业互联网平台发布的申报通知（每年不少于一次），编写申请材料并上传。
2. 审核。新建片区管委会及三县行业主管部门对企业申请材料进行初审，并对报送材料的合规性、真实性负责，在规定的时间内将推荐企业齐全的申请材料和推荐意见上报雄安新区改革发展局。
3. 复核。雄安新区改革发展局应于收到材料后 10 个工作日内完成复核，对复核通过的，形成拟认定名单进行公示，公示期不少于 5 个工作日。
4. 认定。公示无异议的，雄安新区改革发展局按程序拨付奖励资金。

第二十二条 对入驻基地的相关企业，连续3年按照年租金50%给予资助，每家企业每年资助金额最高不超过500万元。

# 第二十三条 认定标准:

（一）在新区正常开展经营活动的企业。
（二）与国家级、省级、新区级生物产业示范基地（包括产业园区等）等运营主体签署承租合同并入驻。
（三）满足以下任意一项的生物医药类企业：

1. 团队有一人（含）以上获得国家、省、市级人才评选称号；
2. 拥有 1 项（含）以上生物产业产品发明专利，或取得产品临床试验许可，或持有产品注册证并实现产业化；
3. 申报上年度营业收入达到 300 万元（含）以上。

（四）自申报核准后连续三年给予实际租用办公研发或生产性场地租金 50% 补贴，每家企业每年资助金额最高不超过 500 万元。
（五）租金补贴按照“先缴后补”的方式，补贴期从实际缴纳租金日期起开始计算，房租滞纳金、管理费不纳入补贴范围内。
（六）享受租金资助的办公、研发、生产场地在资助期内不得对外转租转售，连续一个月无人办公或生产的，停止给予房租补贴。
（七）本政策补贴与新区其他如主题楼宇、孵化器、总部经济等办公租金补贴政策不叠加享受。

# 第二十四条 申报材料及申报流程

# (一) 申报材料

1. 申请项目报告、申请补贴资金申报表，加盖公章。
2. 申报企业营业执照复印件，加盖公章。
3. 运营方与承租方租赁合同、租金发票、付款证明等扫描件。
4. 申请资金承诺书。
5.其他需要提供的资料。

# （二）申报流程

1. 申报。申报企业应根据本实施细则以及雄安新区产业互联网平台发布的申报通知（每年不少于一次），编写申请材料并上传。
2. 审核。新建片区管委会及三县行业主管部门对企业申请材料进行初审，并对报送材料的合规性、真实性负责，在规定的时间内将推荐企业齐全的申请材料和推荐意见上报雄安新区改革发展局。
3. 复核。雄安新区改革发展局应于收到材料后 5 个工作日内完成复核，对复核通过的，形成拟认定名单进行公示，公示期不少于 5 个工作日。
4. 认定。公示无异议的，雄安新区改革发展局按程序拨付奖励资金。

# 第六章 支持发展数字医疗

第二十五条 鼓励数字医疗产品在新区内有条件的医疗机构先行先试，加快应用智能可穿戴设备、人工智能辅助诊断和治疗系统等智慧服务软硬件，按当年采购总金额(含设备购置及服务)的 $10\%$ 予以资助，单个产品资助最高不超过300万元，单个机构每年资助最高不超过500万元。

# 第二十六条 认定标准

# 第三十九条 申报材料及申报流程

# （一）申报材料

1. 申请项目报告、申请补贴资金申报表，加盖公章。
2. 申报企业营业执照复印件加盖公章。提供会计师事务所出具的专项审计报告（含合同、付款凭证、发票等证明材料）。
3.药品注册批件、医疗器械注册证或首次获得美国食品药品监督局（FDA）、欧洲药品管理局（EMA）、欧洲共同体（CE）、日本药品医疗器械管理局（PMDA）、世界卫生组织（WHO）、国际药品认证合作组织（PIC/S）、新药临床试验批文（IND）、美国药物主文件（DMF）、欧洲药典适应性证书（CEP）、日本药物主文件（MF）注册、认证的证明材料。
4. 国外销售的证明材料包含但不限于销售清单、销售合同、销售发票、转账凭证、出口报关单等。
5.申请资金承诺书。
6.其他需要提供的资料。

# （二）申报流程

1. 申报。申报企业应根据本实施细则以及雄安新区产业互联网平台发布的申报通知（每年不少于一次），编写申请材料并上传。

2. 审核。新建片区管委会及三县行业主管部门对企业申请材料进行初审，并对报送材料的合规性、真实性负责，在规定的时间内将推荐企业齐全的申请材料和推荐意见上报雄安新区改革发展局。
3. 复核。雄安新区改革发展局应于收到材料后 10 个工作日内完成复核，对复核通过的，形成拟认定名单进行公示，公示期不少于 5 个工作日。
4. 认定。公示无异议的，雄安新区改革发展局按程序拨付奖励资金。

# 第八章 加快公共服务平台引进

第四十条 加快在新区培育引进组建新药研发合同外包服务机构（CRO）、合同定制研发生产机构（CDMO）、生产合同外包服务机构（CMO）、合同销售组织（CSO）等公共服务平台，按经评审核定项目总投资的20%给予资助，最高不超过5000万元。

# 第四十一条 认定标准

1. 申请单位应为在雄安新区具有独立法人资格生物医药企事业单位。
2. 生物医药产业公共服务平台是指以降低企业研发、生产、经营、销售等环节成本和风险，提升产品开发质量，完善产业研发支撑环境为目标，已建成运营的面向生物医药中小企业提供服务的技术平台。对外提供创新资源共享、科技研发、工程化研究、

产业化等环节中的共性、专业性技术服务与信息服务等。生物医药产业公共服务平台包括但不限于 CRO、CDMO、CMO、CSO、医疗大数据临床研究应用中心、检验检测中心、转化医学中心、临床医学研究中心、一致性评价平台、中试服务平台、生产服务平台、药物非临床安全性评价平台等。

3.建成并投入使用，完成固定资产投资额达到500万元以上的且设备投入占比不低于60%，具备为生物医药企业提供专业服务能力。
4. 在新区建设公共服务平台，经专家评审符合条件的，按平台项目总投资的 20% 予以资助，最高不超过 5000 万元。

# 第九章 鼓励特医食品产业发展

第四十三条 新获批国家注册的保健食品、特殊医学用途配方食品，首次在雄安新区生产结算后，按该品种年营业收入的5%予以奖励，单个企业每年最高不超过1000万元，连续奖励3年。

# 第四十四条 认定标准

（一）申请单位应为在雄安新区具有独立法人资格生物医药企业。
（二）取得特医食品注册证在新区开展产业化生产，产品首次生产订单结算，且产值及营收结算在雄安新区。

（三）按该品种申报上年营业收入的 5% 予以奖励，多个品种的，可累加奖励，但单个企业每年最高不超过 1000 万元。
（四）自申报当年起连续3年给予奖励，每年需提交上年产品销售数据，单个品种产品3年后不再奖励。

# 第十章 推动药物临床应用和研究

第四十六条 雄安新区的注册药企在中国境内完成I—III期临床试验并获得上市许可的创新药，鼓励新区医疗机构按照“随批随进”的原则直接使用，给予其最高不超过实际使用产品金额3%的奖励，单个医疗机构奖励每年合计最高不超过500万元，且不纳入医疗机构药占比和耗占比的考核范围。

# 第四十七条 认定标准

（一）申请单位应为在雄安新区具有独立法人资格的二级以上公立、私立医疗机构，疏解医疗机构的雄安新区院区。
（二）每年更新发布雄安新区创新药指导目录，新区医疗机构“随批随进”采购目录内药品的，纳入支持范围。
（三）经认定后，按该品种采购金额的 $3\%$ 予以奖励，多个品种的，支持可累加，单个医疗机构每年最高奖励不超过500万元。
