## 正文

关于做好2024年第三批高新技术企业申报工作的通知

三县工科数局，各有关单位：

按照河北省高新技术企业认定管理工作领导小组办公室《关于开展2024年度高新技术企业申报认定工作的通知》（冀高认办〔2024〕3号）要求，本年度受理企业申报材料截止日期为9月18日，请三县工科数局高度重视并积极组织辖域内企业开展申报认定工作。

一、2024年申报情况

2024年第一批高企申报共推荐140家企业，第二批推荐51家企业，工信厅暂未组织评审。

二、申报要求

1.申报企业需具备河北省科技型中小企业资质，或已在平台申报相关资质。

2.申报企业需在国家“高新技术企业认定管理工作网”（http://www.innocom.gov.cn/）提交电子版申请；根据预审意见修改完成后，纸质资料按照申报材料内容要求（附件1）逐项打印、装订成册，并留存备查，纸质版材料需与电子版材料一致。

3.请三县工科数局对2024年需复审高企通知到位，名单见附件2，如因企业经营状况、搬迁等原因未提交复审的需出具情况说明，见附件3。

三、注意事项

申报书封皮统一选择河北省认定管理机构。此外，为落实《进一步完善高新技术企业认定管理的若干措施》（冀科高函〔2019〕91号），规范全省相关科技服务机构的服务行为，在各企业申报汇总表中需注明科技服务机构参与企业申报材料编写情况。三县工科数局请于9月19日前将汇总表加盖公章后和电子版一并发送至邮箱：gfjggfwz@126.com。

<span class="mark">附件1</span>

**<span class="mark">申报材料内容要求</span>**

**<span class="mark">1.高新技术企业认定申请书</span>**

<span class="mark">提供高新技术企业认定管理工作网生成打印的《高新技术企业认定申请书》。</span>

**<span class="mark">2.企业营业执照</span>**

<span class="mark">提供加盖公章的营业执照。</span>

**<span class="mark">3.申请书封皮</span>**

<span class="mark">提供高新技术企业认定管理工作网生成打印的高新技术企业认定申请书封皮，法定代表人签字并加盖企业公章。</span>

**<span class="mark">4.知识产权（IP）相关资料</span>**

<span class="mark">知识产权需逐项填写，提供材料如下：知识产权证书（已发放）及最近一次缴费证明，授权通知书及缴费收据等；反映技术水平的证明材料；相关专管机关出具的变更证明（通过转让、受赠、并购获得的知识产权）、协议和备案证明等。纸件目录需标明知识产权编号和知识产权名称（编号同《知识产权汇总表》一致）。</span>

**<span class="mark">5.企业人员相关资料</span>**

<span class="mark">提供企业职工和科技人员情况说明，包括在职、兼职和临时聘用人员人数、人员学历结构；科技人员花名册及其劳动关系证明（社保证明、工资个税申报表、劳动合同均可）等。</span>

**<span class="mark">6.研究开发活动（RD）相关资料</span>**

<span class="mark">近三年研发活动佐证材料，包括科技项目立项报告、结题报告或验收报告、委托开发或合作开发协议等。</span>

**<span class="mark">7.近三年研发费用专项审计报告或鉴证报告</span>**

<span class="mark">近三年研发费用专项审计报告或鉴证报告（需按照正文要求赋码），包括近三年研发项目情况表、近三年研究开发费用支出汇总表、近三年研发费用结构明细表及明细表编制说明（专项报告中应明确说明申报企业研发经费辅助账建立及执行情况）；审计单位营业执照、执业证书、资格证书，专项审计机构证明事项告知承诺书或可证明专项审计机构符合条件的相关材料等，须赋码。</span>

**<span class="mark">8.上年度高新技术产品/服务（PS）相关资料</span>**

<span class="mark">上年度高新技术产品（服务）佐证材料，包括企业高新技术产品（服务）的关键技术和技术指标的具体说明、相关的生产批文、认证认可和资质证书、产品质量检验报告、用户使用报告、销售发票等。</span>

**<span class="mark">9.近一年高新技术产品（服务）收入专项审计报告或鉴证报告</span>**

<span class="mark">近一年高新产品（服务）收入专项审计报告或鉴证报告（需按照正文要求赋码），包括近一年的高新产品（服务）收入明细表及编制说明；审计单位营业执照、执业证书、资格证书，专项审计机构证明事项告知承诺书或可证明专项审计机构符合条件的相关材料等，须赋码。</span>

**<span class="mark">10.科技成果转化证明材料</span>**

<span class="mark">转化结果证明材料包含以下内容：</span>

<span class="mark">新产品：产品生产许可文件，如生产批文（如药品等），销售合同、订单、销售发票、检测报告、企业标准等一种或多种材料；</span>

<span class="mark">新服务：服务发票、合同、设计图纸、软件运行界面、作为第三方检验检测认证和标准化服务技术企业出具的检验报告等；</span>

<span class="mark">新设备：设备说明书、设备备案文件、发票等；</span>

<span class="mark">新技术应用：推广应用证明、解决产品性能质量或生产效率问题情况等；</span>

<span class="mark">样品/样机：检测报告、毒理报告、用户报告、说明书等；</span>

<span class="mark">其他证明材料等。</span>

**<span class="mark">11.研究开发组织管理水平证明材料</span>**

<span class="mark">已制定企业研究开发的组织管理制度、建立研发投入核算体系、编制研发费用辅助账的企业，需提供《研发项目管理制度》《研发费用财务管理制度》《研发辅助账管理办法》等。</span>

<span class="mark">已设立内部科学技术研究开发机构并具备相应科研条件，与国内外研究开发机构开展多种形式产学研合作的企业，需提供研发机构建立证明材料、研发场地及设备清单和主要设备照片、产学研合作协议等。</span>

<span class="mark">已建立科技成果转化的组织实施与激励奖励制度，建立开放式的创新创业平台的企业，需提供《科技成果管理及激励制度》《创新创业技术管理办法》等。</span>

<span class="mark">已建立科技人员的培养进修、职工技能培训、优秀人才引进，以及人才绩效评价奖励制度的企业，需提供《科技人员培训制度》《职工技能培训制度》《人才引进措施》《研发人员绩效考核制度》等。</span>

**<span class="mark">12.国家或行业标准证明材料</span>**

<span class="mark">材料组织顺序须同《高新技术企业认定申请书》中的“企业参与国家标准或行业标准制定情况汇总表”一致；</span>

<span class="mark">参与国家标准或行业标准制定的企业，需提供标准证明材料，包含标准封面和显示有申报企业名称参与制定该标准的相关页面等。</span>

**<span class="mark">13.企业财务审计报表</span>**

<span class="mark">近三年企业财务审计报表，包含资产负债表、利润表、现金流量表等会计报表、会计报表附注、财务情况说明书、审计单位营业执照、执业证书、资格证书等。</span>

**<span class="mark">14.企业所得税纳税申报表</span>**

<span class="mark">近三年企业所得税年度纳税申报表，包括企业所得税年度纳税申报表填报表单、企业基础信息表、企业所得税年度纳税申报表主表及附表。</span>

**<span class="mark">15.证明事项告知承诺书</span>**

<span class="mark">提供法人签名并加盖公章的证明事项告知承诺书。</span>

**<span class="mark">16.其他附件</span>**

<span class="mark">其他资质证明：特殊领域、行业的企业，须提交相关管理部门开展相关技术服务、生产许可的文件或相关许可规定。</span>

<span class="mark">复审企业需提供高新技术企业证书</span>

<span class="mark">更名证明：企业如近三年涉及到名称变更，还需上传核准变更通知书等相关材料。</span>

<span class="mark">知识产权说明材料：当知识产权有多个权属人时，只能由一个权属人在申请时使用，由申报单位征求其他权属人同意后出具确认文件。</span>

<span class="mark">近三年研发活动总体情况说明（重点说明近三年研发工作总体思路、研发方向、所属技术领域、研发组织管理、成果数量及对企业技术进步、转型升级、创新发展的作用）。</span>
