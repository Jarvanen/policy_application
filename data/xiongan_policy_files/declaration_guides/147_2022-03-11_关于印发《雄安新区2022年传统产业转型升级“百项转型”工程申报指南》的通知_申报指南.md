# 政策简介

“百项转型”工程主要支持符合《河北雄安新区传统产业转移升级工作实施方案》及《河北省重点产业技术改造投资导向目录(2021-2022年)》的传统产业转型升级项目。围绕服装、制鞋、毛绒玩具、塑料包装、电线电缆等传统优势产业，改造提升传统产业竞争新优势，支持符合国家和省现行产业结构调整目录、技术改造投资导向目录，以及省级出台的行业规划、行动计划、实施方案等政策文件明确支持的方向和领域的重点项目。

# 奖励额度/补贴标准

1. **企业技术改造项目**：按项目已完成的固定资产投资额（包括但不限于购置生产设备、检测仪器、研发工器具、配套软硬件系统等，不含土地、建筑物投资），采取后补助方式给予不高于20%补助。
2. **企业数字化转型项目**：按照信息化软硬件购置、系统开发与服务等费用，采取后补贴方式给予30%资金支持，每个平台奖补金额最高不超过300万元。其中，新区级综合产业互联网平台支持不超过1个项目，行业级产业互联网平台中每个行业支持不超过1个项目。
3. **企业服务平台转型项目**：对企业在服务化转型已完成的固定资产投资额（包括但不限于购置设备、仪器、配套软硬件系统等，不含土地、建筑物投资），给予最高不超20%补贴。

# 注意事项

1. **重复享受限制**：已经获得国家、省、新区各级财政支持的同一项目，原则上不得再申报此次市级专项资金支持。
2. **材料保存**：资金申报资料需永久保存备查。
3. **真实性责任**：项目单位对专项资金申报材料内容的真实性负责。
4. **格式要求**：资金申请报告书脊标注“2022年+**县+申报单位名称+项目名称”，电子版正文为word格式，附件为pdf格式，表格为excel格式。
5. **发票要求**：进口设备需提供海关税务发票及相关资料。

# 申请对象

1. 在雄安新区注册、具有独立法人资格的雄安新区辖区内传统制造业企业。
2. 产业互联网平台服务机构。
3. 传统制造业企业的工业增加值、投资数据在本地入统，并在本地纳税。

# 申报条件

1. 项目实施单位在雄安新区注册、具有独立法人资格的雄安新区辖区内传统制造业企业，以及产业互联网平台服务机构。传统制造业企业的工业增加值、投资数据在本地入统，并在本地纳税。
2. 项目实施单位具备较强的技术开发、资金筹措、项目实施能力，成长性较好，具有良好的社会信用。
3. 申报项目实施地在雄安新区。
4. 申请财政资金奖补的应已办理完成备案(核准)及土地、环评等手续，并已开工建设。谋划项目可先列入项目库(填报附件7)，并注明谋划项目、何时完成备案开工，项目实质性完成投资后可再申请财政奖补。
5. 已经获得国家、省、新区各级财政支持的同一项目，原则上不得再申报此次市级专项资金支持。
6. 项目单位遵守国家法律法规，符合新区产业发展方向，企业经营状况良好、发展规划和目标明确，在经营管理、社会信用、安全生产、节能环保等方面无不良记录。
7. 开展两化融合管理体系贯标并通过评定的，被认定为省级“专精特新”中小企业、单项冠军、绿色工厂的，入统的规模以上工业企业，项目计划投资1000万元以上的项目，同等条件下优先纳入项目库。

# 申报材料

1. **资金申请报告**：严格按照编制要点编制（见附件1、2、3）。
2. **附件材料**：
   - 项目单位营业执照(组织机构代码证)复印件。
   - 项目核准或备案证复印件。
   - 上年度财务审计报告(复印件，加盖申请单位公章)；如申报时尚未完成审计报告编制，应提交①最近年度的财务审计报告，②由单位财务人员按审计报告要求编制的上年度《资产负债表》、《利润表》、《现金流量表》(法定代表人、财务负责人签字，盖公章)。
   - 三县工业和信息化主管部门出具的项目初审情况说明。
   - 项目固定资产设备投资完成明细表及对应发票复印件。
   - 项目单位同意税务机关提供设备、仪表仪器、软件产品、专利技术及设备改造等固定资产投资发票等相关资料的授权书。
   - 申报项目未获得省级专项资金支持情况说明（或国家、省、新区级）。
   - 项目单位对项目单行材料及附件真实性负责的声明。
3. **其他**：向所属地县级工信部门提供申报书及佐证材料等纸质文本。

# 办理流程

1. **网上填报**：各申报项目单位登陆“雄安新区产业互联网平台”(xatic.org.cn)选择“百项转型”项目申报栏目填报申报材料。
2. **纸质报送**：向所属地县级工信部门提供申报书及佐证材料等纸质文本。
3. **县级初审**：三县工信部门组织属地项目申报，核实项目进展情况，进行实地考察，筛选符合申报方向、条件成熟的项目，核查有关审批手续原件，推荐上报项目须逐个出具在建证明函。
4. **汇总上报**：三县工信部门统一汇总上报本地区推荐项目，将推荐文件、资金申请报告一式三份报送新区改发局(纸质版、电子版U盘)。
5. **评审**：新区改发局委托专家或第三方机构，对资金申请报告进行形式审查、专家评审，提出评审意见。

# 申报时间

1. 第一批申报截止日期为2022年4月30日。
2. 后续申报工作根据实际情况滚动开展。
3. 三县工信部门需于4月30日前将推荐文件、资金申请报告报送新区改发局。
