# 政策简介

为贯彻落实中共中央、国务院《关于支持河北雄安新区全面深化改革和扩大开放的指导意见》等相关文件精神，服务承接非首都功能疏解，优化新区营商环境，推动新区传统产业转型升级和高端高新产业发展，加大对新区企业改制、挂牌、上市的扶持力度，加快实现“五新”目标。重点支持新一代信息技术、现代生命科学和生物技术、新材料、高端现代服务业、绿色生态农业等产业，以及非首都功能疏解和三县传统产业转型升级企业，加快对“专精特新”企业、隐形冠军企业、小巨人企业和独角兽企业的挖掘、引入和培育。

# 奖励额度/补贴标准

1. **上市奖励**：
   - 对在上海、深圳、北京证券交易所完成首次公开发行股票并上市的企业，给予奖励资金300万元。
   - 对在境外主要证券交易场所主板完成首次公开发行股票并上市或回归A股上市的企业，给予奖励资金300万元。
2. **挂牌及融资奖励**：
   - 对在“新三板”基础层和创新层挂牌的企业，给予奖励50万元。
   - 对其挂牌1年内的首次融资额，按照不超过1‰的比例进行融资奖励，融资奖励金额每家企业不超过100万元。
3. **迁入企业奖励**：
   - 对2020年后迁入新区的上海、深圳、北京证券交易所上市企业以及基础层、创新层挂牌企业，视其区外政策享受情况，参考上述标准给予差额奖励。
4. **县域特色产业集群补助**：
   - 对进入辅导期后向河北证监局提供申报材料的县域特色产业集群企业，按上市挂牌过程中委托专业服务机构进行财务规范和股份制改造等所发生的服务费的40%给予补助。
   - 对企业规模在县域特色产业集群内前三（或已挂牌上市、拟辅导上市），且辐射带动作用强的集群龙头产业，当年购置研发、检测、试验等设备投入50万元以上，按照实际投资额，采取后补助的方式给予20%的补助，单个项目不超过200万元。

# 注意事项

1. **时间要求**：奖励对象为2020年起成功挂牌或上市的新区企业；迁入企业奖励针对2020年后迁入新区的企业。
2. **重复享受限制**：迁入新区的上市及挂牌企业，需视其区外政策享受情况，给予差额奖励。
3. **其他规定**：三县政府可以根据实际情况，研究制定本辖区内对于挂牌上市企业的奖励资金政策。

# 申请对象

1. **新区企业**：在雄安新区注册的企业。
2. **重点产业方向**：新一代信息技术、现代生命科学和生物技术、新材料、高端现代服务业、绿色生态农业等产业。
3. **特定企业类型**：
   - 非首都功能疏解和三县传统产业转型升级企业。
   - “专精特新”企业、隐形冠军企业、小巨人企业和独角兽企业。
   - 县域特色产业集群企业（特别是进入辅导期、集群龙头产业）。
   - 种子期、初创期的科技型小微企业。
4. **迁入企业**：2020年后迁入新区的上海、深圳、北京证券交易所上市企业以及基础层、创新层挂牌企业。

# 申报条件

1. **基础条件**：
   - 主营业务突出、竞争能力较强、盈利水平较好、发展潜力较大。
   - 治理结构完善、经营运作规范、具备核心竞争力和成长潜力。
   - 依法协助企业妥善处理股份制改造和挂牌上市过程中可能存在的土地房产权属不清晰、股权纠纷、税费缴纳、证照补办和行政许可不衔接等历史遗留问题。
2. **专项条件**：
   - **县域特色产业集群服务费补助**：进入辅导期后向河北证监局提供申报材料。
   - **设备购置补助**：企业规模在县域特色产业集群内前三（或已挂牌上市、拟辅导上市），且辐射带动作用强；当年购置研发、检测、试验等设备投入50万元以上。

# 申报材料

1. **县域特色产业集群企业**：向河北证监局提供申报材料（用于申请服务费补助）。
2. **设备购置补助**：需提供当年购置研发、检测、试验等设备的实际投资额证明（用于申请20%后补助）。

# 办理流程

1. **入库管理**：筛选符合条件的企业，逐步建立分层级、分阶段的挂牌上市后备企业资源库，实现动态管理、分类指导和精准服务。
2. **辅导培育**：
   - 新区管委会及三县政府安排专人给予“专业化的管家式服务”。
   - 联合证券监管部门、证券交易所、证券行业协会和中介机构，开展梯度培育和分类指导。
   - 协调国家有关部委，推动国家有关支持政策落地（如IPO“即报即审、审过即发”，新三板“专人对接、专项审核”“即报即审、审过即挂”）。
3. **服务支持**：
   - 职能部门配合出具合规性证明函件，缩短办理时限。
   - 为重点培育企业股权转让、投资立项、环保审批等事项开辟绿色通道。
4. **资金申请与拨付**：
   - 帮助协调申请省级财政专项补助资金。
   - 对符合条件的企业给予专项资金奖励或补助。
5. **信息报送**：三县政府每月3日之前向新区改革发展局报送上月辖内企业挂牌上市及项目储备有关情况。

# 申报时间

- **信息报送**：三县政府每月3日之前向新区改革发展局报送上月辖内企业挂牌上市及项目储备有关情况。
- **奖励起始时间**：2020年起成功挂牌或上市的企业。
