# 政策简介

为贯彻落实《河北省支持机器人产业发展若干措施》，在制造业领域遴选一批技术成熟、成效明显、易于推广的机器人产品和系统解决方案，推动“机器人化”生产装备向相关领域应用拓展，省工业和信息化厅组织开展2024年“机器人化”技术改造项目申报工作。省工信厅将编制机器人创新应用案例集，推广100项基于“机器人化”的智能制造和工业互联网创新发展重点项目。

# 奖励额度/补贴标准

对牵头参与重点项目的机器人相关企业，按“机器人化”项目金额的20%给予补贴，最高不超过500万元。

# 注意事项

1. **材料真实性**：申报单位对申报材料的真实性负责，并积极配合省、市工业和信息化部门及专家、会计师事务所做好现场核查工作，按要求如实提供原始资料。
2. **填报格式**：请用A4幅面编辑，正文字体为4号仿宋体，单倍行距。一级标题4号黑体，二级标题4号楷体。
3. **外文名词**：项目申报书中第一次出现外文名词时，要写清全称和缩写，再出现同一词时可以使用缩写。
4. **信用记录**：申报单位近3年无不良信用记录和重大违法行为。
5. **邮寄要求**：纸质版盖章材料（一式三份）邮寄时，信封上请注明“申报‘机器人化’技术改造项目”字样。

# 申请对象

1. **申报主体**：牵头参与“机器人化”技术改造项目的机器人相关企业（包括机器人生产企业、智能制造集成商、系统控制软件生产企业等）。
2. **关联主体**：“机器人化”技术改造应用单位。
3. **地域要求**：机器人相关企业和“机器人化”技术改造应用单位均应为河北省境内注册、具有独立法人资格的企业。
4. **行业方向**：以钢铁、装备、化工、食品、建材、轻纺等传统产业为重点。

# 申报条件

1. 申报单位为牵头参与“机器人化”技术改造项目的机器人相关企业。
2. 机器人相关企业和“机器人化”技术改造应用单位均应为河北省境内注册、具有独立法人资格的企业，且运营、财务、信用状况良好。
3. “机器人化”技术改造软硬件设备购置时间应在2023年3月2日以后，且申报时机器人软硬件设备已投入使用。
4. 项目方向：以机器人产品为核心，以装备智能化、生产线智能化和建设数字车间、智能工厂为主要方向的智能制造和工业互联网创新发展项目。

# 申报材料

1. **申报书**：河北省2024年机器人化技术改造项目申报书（见附件1，一式三份，PDF版和Word版）。
2. **推荐文件**：河北省2024年机器人化技改项目推荐表（见附件2，由市级主管部门出具）。

    - （一）机器人相关企业和机器人化改造应用单位的营业执照复印件（必备）；

    - （三）机器人相关企业与机器人化改造应用单位的项目实施/采购合同（必备）；
    - （四）机器人化技术改造设备（软硬件）清单表（格式见附件1-1，必备）；
    - （五）机器人化技术改造项目已投入资金凭据汇总表（格式见附件1-2）与上述已投入资金凭据汇总表顺序、内容对应一致的记账凭证、付款凭证、采购合同、软硬件设备购置发票复印件等（必备）；

    - （八）机器人相关企业专利、软件著作权等知识产权证明；
    - （九）申报材料真实性声明（详见附件1-4）；
    - （十）其他佐证材料。

# 办理流程

1. **企业申报**：申报单位向属地工业和信息化主管部门提交申报材料（一式三份，PDF版和Word版发至电子邮箱）。
2. **市级初审**：各地工业和信息化主管部门对申报材料的合规性、完整性、统一性进行形式审查，出具推荐意见，并于2024年4月18日前将审核通过的申报材料和推荐文件报至省工业和信息化厅。
3. **省级评审**：省工业和信息化厅组织专家对各市推荐的申报材料进行评审，重点关注技术先进性和应用示范性、市场需求和经济效益潜力等。
4. **现场核查**：对省级评审通过的项目，省工业和信息化厅将组织技术专家、会计师事务所等开展现场审计核查。
5. **公示发布**：根据评审和现场审计核查结果，研究核定最终通过审查的项目名单，在省工业和信息化厅官网进行公示，接受社会监督。对通过公示的项目给予确定，并正式行文公布。
