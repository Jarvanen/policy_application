# 政策简介
本细则旨在支持雄安新区科技企业高质量发展，对纳入新区科技企业库并在新区依法纳税的科技企业，在新区范围内银行机构申请的销售订单、应收账款等相关流动资金贷款给予贴息支持。贴息按照“先付后贴”、“免申即享”的方式实施，即企业按合同约定支付银行机构利息后，新区政府定期对符合条件的企业贴息。

# 奖励额度/补贴标准
1. **贴息比例**：按照贷款合同签订实际贷款利率的50%予以贴息。
2. **贴息期限**：最多不超过2年。
3. **贴息总额**：每户企业贴息总额不超过100万元。
4. **适用范围**：申请企业2022年5月17日至2024年5月17日之间发生的流动资金贷款（以提款日期为准）。

# 注意事项
1. **贷款类型限制**：贴息主要用于销售订单、应收账款等相关流动资金贷款。以个人名义贷款（包括企业法定代表人及配偶、企业主要股东、实际控制人等）用于经营的不列入贴息范围。
2. **排他性规定**：各家企业同时只可享受一项贷款贴息政策。
3. **资金拨付方式**：通过数字人民币钱包完成资金拨付。
4. **政策有效期**：本细则有效期至2024年5月17日。
5. **其他提醒**：贴息资金在预算安排总额下，视贴息进度和资金使用情况，新区改发局有权顺延贴息资金拨付或暂停、终止贴息申请。

# 申请对象
按照雄安新区改革发展局《关于〈促进“专精特新”企业和数字经济核心产业高质量发展的若干措施〉科技企业库入库及奖补政策的实施细则》(雄安改发〔2022〕233号)有关规定纳入新区科技企业库并在新区依法纳税的企业。

# 申报条件
1. 企业需纳入新区科技企业库。
2. 企业在新区依法纳税。
3. 贷款需在新区范围内银行机构申请。
4. 贷款用途为销售订单、应收账款等相关流动资金贷款。
5. 贷款发生时间在2022年5月17日至2024年5月17日之间（以提款日期为准）。

# 申报材料
1. 企业主体登录雄安新区产业互联网平台自主完善贷款信息。
2. 上传贷款合同等有关材料。
3. 完成上述信息录入后，企业无需再报送其他申报材料。
4. 银行需配合提供贷款贴息明细表（附件1）、付息有关证明材料等。

# 办理流程
1. **完善信息**：企业主体登录雄安新区产业互联网平台(http://www.xaiip.org.cn)，通过“用户中心”-“企业数据账户”-“贷款信息”或“惠企政策”-“科技企业高质量发展贷款贴息”自主完善贷款信息，并将贷款合同等有关材料进行上传。
2. **银行核准**：各银行定期对企业客户填写信息进行核对，核对无误的，提交产业互联网平台推送至三县审核部门进行审核；核准有错误的，请企业客户重新填写后提交。
3. **三县审核**：三县审核部门根据企业贷款信息协请对应贷款银行出具贷款贴息明细表、付息有关证明等材料，并聘请第三方会计师事务所开展审计工作。
4. **汇总及公示**：三县审核部门定期对本县审核通过的企业拟贴息情况进行汇总，填写《贷款贴息资金申请表》，请第三方审计机构出具综合审计报告，报县政府研究同意，同步在新区产业互联网平台上进行公示（不少于5日）。
5. **资金拨付**：三县责任部门根据县政府研究情况，形成明确书面意见，以县政府名义向新区改发局提起贴息申请，附《贷款贴息资金申请表》、综合审计报告等有关材料，新区改发局按程序通过数字人民币钱包完成资金拨付。

# 申报时间
贷款发生时间需在2022年5月17日至2024年5月17日之间（以提款日期为准）。政策有效期至2024年5月17日。
