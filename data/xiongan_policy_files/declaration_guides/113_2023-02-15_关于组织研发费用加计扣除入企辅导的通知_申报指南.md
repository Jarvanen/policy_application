# 政策简介

为了进一步促进雄安新区科技型企业持续加大研发投入，引导企业研发费用加计扣除优惠政策落地落实，助力科技创新支撑引领高质量发展，河北雄安新区管理委员会改革发展局委托河北中浩华财税服务集团股份公司为科技型企业免费开展研发费用加计扣除入户辅导。辅导主要事项包括：研发立项流程是否合规、研发投入是否正确、研发资料是否齐全真实完整、研发费用归集是否准确。

# 注意事项

1. **免费服务**：本次入企辅导为免费开展。
2. **保密要求**：服务机构需做好服务保障和保密工作。
3. **配合要求**：雄安集团、各相关单位在入企辅导期间需主动配合提供所需材料，做到政策应享尽享。
4. **后续安排**：后续根据企业需求和工作安排不定期开展辅导。

# 申请对象

1. 三县科技管理部门分管企业。
2. 雄安集团下属企业。
3. 各科技型企业（含附件1名单中的重点入户企业）。

# 申报材料

1. 2022年度研发项目立项决议及研发立项书。
2. 2022年度研发费用辅助账。
3. 2022年度用于研发的固定资产折旧表（包含资产名称、购置时间、折旧年限、原值、折旧、残值、净值等）、研发人员的工资及五险一金（分月提供）；如涉及到费用分配的，需提供分配方法及分配表。
4. 2022年研发直接投入中领用材料明细表，燃料动力分配明细表。
5. 涉及委托外部发生的研发费用的委托研发合同及技术合同登记表；涉及技术服务费的技术服务合同。
6. 2022年度财务报表（包括资产负债表、利润表、现金流量表等）。
7. 高新技术企业证书（如有）。

# 办理流程

1. **组织与对接**：三县科技管理部门、雄安集团积极组织分管企业、协调安排、宣传政策；河北中浩华财税服务集团股份公司主动对接。
2. **入企辅导**：服务机构入户开展辅导，企业主动配合提供所需材料。
3. **解答疑问**：服务机构耐心解答企业疑问，做好服务保障。

# 申报时间

2023年第一批入企辅导时间从2月16日-3月10日。
