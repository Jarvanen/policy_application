# 政策简介

为帮助新区受灾工业企业渡过难关，确保企业尽快复工复产，快速有效恢复市场主体活力，针对近期台风“杜苏芮”及外围云系给雄安新区带来强降雨、洪涝积水和大风天气导致辖区内部分企业受到不同程度影响的情况，特制定帮扶救助措施。

# 奖励额度/补贴标准

1. **原辅材料及成品浸水损失救助**：给予直接损失数额20%的现金救助，最高救助金额不超过50万元。
2. **设备损坏救助**：按新增设备实际购置金额的20%给予一次性救助，救助上限500万元。
3. **水电房租救助**：
   - **水电费**：按实际产生成本每家企业最高救助2.5万元。
   - **房租**：按照每天每平方米不高于1元救助。合同租赁期间，最高给予3个月救助，合同不足3个月据实救助且只对租赁合同中涉及的房屋建筑面积给予救助。房租救助每家企业最高救助5万元。
4. **财产保险补贴**：补贴金额不超过1000元。
5. **灾后重建专项贷款贴息**：贴息标准按照不高于贷款合同签订当日执行的中国人民银行公布的一年期贷款市场报价利率(LPR)予以贴息，每户企业的贴息总额不超过100万。

# 注意事项

1. **有效期**：本政策有效期至2023年12月31日。
2. **认定要求**：水电房租救助对象需为“经认定受灾且产生经济损失的工业企业”；低息贷款对象需为“经认定的受灾工业企业”。
3. **救助限制**：
   - 原辅材料及成品浸水损失救助针对“厂区库房积水漏水导致原辅材料及成品浸水而产生损失”的情况。
   - 设备损坏救助针对“厂房及生产车间等区域积水漏水导致设备损坏而产生损失”的情况。
   - 房租救助只对租赁合同中涉及的房屋建筑面积给予救助。

# 申请对象

受灾工业企业。

# 申报条件

1. 受近期台风“杜苏芮”及外围云系带来的强降雨、洪涝积水和大风天气影响。
2. 属于工业企业。
3. 根据不同救助项目，需满足以下具体情形之一：
   - 厂区库房积水漏水导致原辅材料及成品浸水而产生损失。
   - 厂房及生产车间等区域积水漏水导致设备损坏而产生损失。
   - 经认定受灾且产生经济损失（适用于水电房租救助）。
   - 购买灾后财产保险（适用于财产保险补贴）。
   - 因灾后重建而需要申请专项贷款（适用于贷款贴息）。
   - 经认定受灾（适用于灾后重建低息贷款）。

# 办理流程

1. **认定**：部分救助项目（如水电房租、低息贷款）需经过受灾认定。
2. **申请**：企业根据受损情况申请相应救助或贷款。
3. **救助/拨付**：
   - 现金救助、保险补贴、水电房租救助由相关部门给予救助。
   - 贷款贴息由相关部门实行贴息措施。
   - 低息贷款由金融机构引导中小银行提供。

# 申报时间

政策有效期至2023年12月31日。
