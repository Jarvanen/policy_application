# 政策简介

高技能人才培训基地建设项目是新时期加强高技能人才队伍建设的一项重要举措，旨在打造具有新区特色的新时代一流技术工人队伍、促进新区当地劳动力多渠道转移就业、提高就业质量。项目主要面向各类在职职工及其他有技能提升愿望的劳动者，开展高技能人才研修培训，同时承担技能人才评价、职业技能竞赛、高技能课程研发、高技能成果交流展示等任务。结合新区实际，按年度计划重点支持高技能人才培训基地建设，建设项目实施期限2年。

# 奖励额度/补贴标准

对新创建的新区级高技能人才培训基地给予100万元一次性支持，所需资金从新区财政安排的就业专项资金中列支。

# 注意事项

1. **资金用途限制**：专项资金专项用于技能研修实训设备购置、培训基地基础设施完善、指导教师聘用、师资培训、高技能人才课程开发及相关教学科研等事项开支。禁止用于上述事项之中所发生的差旅费、路桥费、燃油费等应由单位公用经费列支的相关费用。
2. **资金管理**：三县财政、人社部门按照职责全程监督资金运行，确保专款专用。对发生问题的项目单位给予通报，问题严重的，追回财政资金、撤销项目单位建设资格并依法追究当事人责任。
3. **重复建设限制**：三县人社局要加强对候选单位申报项目的审核，防止和杜绝项目重复建设。
4. **考核与退出**：新区公共服务局会同改革发展局负责定期对基地的建设情况、资金使用情况和任务完成情况进行考核和评估，对达不到要求的基地实行退出机制。
5. **材料格式要求**：
   - 《申报书》请使用A4纸，双面印，左侧装订，一式四份连同电子文档一并上报。
   - 《实施方案》请使用A4纸，双面印，左侧装订后一式四份连同电子文档一并上报。
   - 《实施方案》填写内容的字体为仿宋 GB2312，字号为五号，行距为固定值16磅。
6. **后续推荐**：每年省级高技能人才培训基地建设项目，原则上从新区级高技能人才培训基地中择优推荐。

# 申请对象

依托具备高技能人才培训能力的以下单位：
1. 技师学院
2. 高级技工学校
3. 大中型企业（集团）职工教育培训中心

# 申报条件

**(一) 规范管理**
1. 具有较强的管理能力和高效的组织管理体系，建立了规范的培训管理、财务管理、资产管理、风险管理等制度。
2. 遵守国家有关法律法规，未发生过违规违纪事件。

**(二) 培训能力**
1. 培训场所和设施设备符合国家建设安全标准，具备年培训300名以上高技能人才的条件。
2. 培训目标明确，严格按照国家职业标准开展职业技能培训，具有与2-3个经济发展急需、短缺高技能人才培训的特色专业（职业）相匹配的培训装备。
3. 面向企业和社会开展职业技能培训，年培训规模不少于1000人。

**(三) 师资队伍**
1. 有符合培训要求的稳定的专兼职教师队伍，师生比为1:16到1:20。
2. 高级实习指导老师和具有高级技师职业资格的教师占培训教师总数的30%以上。

**(四) 校企合作**
院校类基地至少与3家以上规模以上企业建立了稳定的合作关系，与合作企业共同研究确定专业建设、课程设置、培养计划、师资建设、研发课题和学生实习方案，并与合作企业共同建立培训实习基地，聘请技师、高级技师和优秀专业技术人才担任指导教师。

# 申报材料

项目候选单位需提交如下材料：
1. 由推荐单位出具的推荐申报高技能人才培训基地项目的报告。
2. 《河北雄安新区高技能人才培训基地建设项目申报书》（见附件1），申报书应涵盖申报条件要求的具体内容。
3. 《河北雄安新区高技能人才培训基地建设项目实施方案》（见附件2），明确基地建设项目的总体目标、分阶段目标及工作任务等。
4. 项目候选单位制订的《项目实施管理办法》。
5. 项目候选单位制订的《项目经费管理办法》。

# 办理流程

1. **推荐**：三县人社部门负责辖区内的推荐申报工作，根据申报条件，按照公开、公平、公正的原则，严格把握，确保推荐的项目单位符合要求。
2. **评审**：新区公共服务局和改革发展局组织专家，对各单位推荐的项目候选单位进行评审，择优确定高技能人才培训基地项目单位。
3. **公示**：将评审结果及时向社会公示。
4. **验收与管理**：高技能人才培训基地建设完成后，新区公共服务局和改革发展局组织验收。基地管理采取所在县与新区公共服务局共管方式，日常运行、管理及监督由所在县人社局负责。培训基地项目单位每年年底要将当年工作开展情况及第二年工作计划报所在县人社局，并抄报新区公共服务局。
