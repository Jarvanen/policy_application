# 政策简介

雄安新区科普示范基地（以下简称“科普基地”）是指经雄安新区工信科技数据局认定，在雄安新区范围内面向公众开展科普教育活动，致力于普及科学知识、倡导科学方法、弘扬科学家精神和创新文化的机构或组织。该政策旨在加强雄安新区科普能力建设，提升全民科学素质，助力雄安新区建设全国科普示范性高地，推动雄安新区成为新时代的创新高地和创业热土。

# 奖励额度/补贴标准

1. **首次认定奖励**：首次认定为科普基地的，一次性给予 2 万元奖励。
2. **运营奖补**：对认定的科普基地按年度进行考核，考核结果分为优秀、合格、不合格三个等级。优秀比例不超过参评总数的 30%，在每年相关专项资金总预算内对考核优秀的运营单位给予每年不超过 20 万元运营奖补，主要用于科普工作。

# 注意事项

1. **认定有效期**：认定有效期限为 2 年，到期后该认定自行失效。
2. **考核要求**：不参加年度考核视为不合格；不合格的进行整改，对于整改不合格或连续两次考核不合格的取消称号。
3. **变更备案**：科普基地发生迁址、重建、法定代表人变更、承建单位变更或其他重大变化的，应及时向雄安新区工信科技数据局备案，并办理相关变更事宜。
4. **退出机制**：有下列情形之一的予以摘牌：
   - 科普功能已丧失的；
   - 运营单位登记主体资格注销或者吊销的；
   - 在认定评审、年度评价等过程中，存在弄虚作假、伪造成果等不诚信行为的；
   - 有严重违法违纪行为的；
   - 发生重大安全责任事故，受到有关部门处罚的；
   - 宣扬邪教和迷信，或参与反科学和伪科学活动的；
   - 无正当理由不参加年度考核的；
   - 限期整改未通过审核的；
   - 从事损害社会公共利益或其他违法活动的。
5. **加分项**：在科普内容、形式、手段等方面具有创新性；与其他机构建立良好合作关系；在社会上具有较高的知名度和美誉度，曾获得省级以上科普奖项或荣誉称号等，在认定中适当予以加分。

# 申请对象

在雄安新区范围内面向公众开展科普教育活动的机构或组织。

# 申报条件

申请科普基地认定，应当同时具备以下基础条件：

1. **经费保障**：具备一定的经费保障。从事《中华人民共和国科学技术普及法》（2024 年修订）所规定的科普活动，有满足从事科普工作所需要的经费，能保障经常性科普活动的开展。
2. **固定场所**：拥有一定规模的固定场所。专门用于开展科普活动，场地面积应根据不同的科普主题和受众需求合理设置，以满足各类科普展览、演示、讲座等活动的空间需求。
3. **设施器材**：具备开展科普活动所需的设施、器材。如科普展品、演示仪器、多媒体展示设备等，使公众能够接触到最新的科技成果和科学知识。
4. **工作机制与人员**：具备开展科普活动的科普工作机制。专、兼职科普人员不少于 3 人，科普人员能够熟练运用多种方式开展科普活动。具有可供公众参观以及开展科普活动所需要的特色科普资源，举办过系列科普活动，能积极配合新区相关部门参加科技活动周、全国科普日等大型科技科普活动，发挥科普示范基地的作用。

# 申报材料

1. 《雄安新区科普示范基地申报基本情况表》（见附件2）。
   - 需填写基地名称、基地类别（综合场馆类、科研教育类、生产示范类、人文自然类、其他类）、运营单位、运营管理单位性质、地址、联系人、电话、职务。
   - 需填写基地基本情况（主要包括：工作机制、基础设施情况、群众参与情况、科普工作开展情况、亮点工作等）。
   - 需填写下一步工作计划。
   - 需建设单位盖章。

# 办理流程

1. **发布通知**：雄安新区工信科技数据局发布科普基地申报通知及有关要求。
2. **受理与初审**：雄安新区工信科技数据局对申请的材料进行审查，对符合要求的予以受理。
3. **专家评审**：雄安新区工信科技数据局组织专家开展评审工作，形成评审结果。
4. **认定与公告**：雄安新区工信科技数据局对通过科普基地评审的单位予以认定，并进行公示。对公示无异议的，雄安新区工信科技数据局发布科普基地名单。
