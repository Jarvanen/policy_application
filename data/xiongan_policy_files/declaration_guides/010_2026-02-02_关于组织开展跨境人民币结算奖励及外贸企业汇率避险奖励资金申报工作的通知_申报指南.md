# 政策简介

为更好发挥《关于支持雄安新区外向型经济高质量发展的十五条金融开放措施》（雄安自贸发〔2024〕9号）政策激励作用，组织开展跨境人民币结算奖励及外贸企业汇率避险奖励资金申报工作。主要支持方向包括：对在雄安自由贸易试验区内登记注册、具有独立法人的跨境收支企业，办理货物贸易、服务贸易、跨境直接投资（不含资金池业务）中使用跨境人民币结算的，按照结算总量的一定比例给予奖励；对在雄安新区内登记注册、依法纳税、具有独立法人资格的外贸企业，办理汇率避险业务（包括首办户）的，按照履约额的一定比例进行补贴，并对首次办理且签约额超过1万美元的“首办户”企业额外给予资金支持。

# 奖励额度/补贴标准

**1. 跨境人民币结算奖励**
*   对办理货物贸易、服务贸易、跨境直接投资（不含资金池业务）三类业务中使用跨境人民币结算的自贸区内企业，按照结算总量的一定比例给予奖励。
*   每家企业年度奖励金额最高不超过50万元。

**2. 外贸企业汇率避险奖励**
*   对办理汇率避险业务的企业（包括首办户），按照履约额的一定比例进行补贴，每家企业每年不超过15万人民币元。
*   在雄安新区首次办理汇率避险业务，签约额超过1万美元的“首办户”企业，额外给予每家资金支持。

# 注意事项

1.  **申报期限与数据范围**：本次申报日期截止到2026年3月10日，申报所需数据和材料范围自2025年1月1日至2025年12月31日。
2.  **重复享受限制**：企业需承诺未在雄安新区享受其他相关同类型政策支持，并提供《企业是否享受其他相关政策支持情况说明》。
3.  **真实性承诺**：企业需提交承诺书，承诺所提供资料真实、完整，若存在不真实、不完整情况，愿承担相应的法律责任。
4.  **账户要求**：外贸企业汇率避险奖励申请表中，收款账号及开户行需填写奖励资金收款账号及开户行，务必确认账户正常。
5.  **合规要求**：
    *   跨境人民币结算奖励申报企业近两年无实质性违反或故意规避跨境人民币管理规定、外汇管理法规、国家“三反”规定等法律、法规的情形；未被纳入跨境人民币业务重点监管企业名单和货物贸易外汇管理B、C类企业；未涉嫌大规模跨境投机套利活动。
    *   外贸企业汇率避险奖励申报企业资信良好，无不良信用记录；货物贸易项下企业应为外汇管理名录登记的A类企业。
    *   承办银行机构上年度外汇业务合规与审慎经营评估评级为“B”及以上，近两年无违反外汇管理规定记录。

# 申请对象

**1. 跨境人民币结算奖励**
*   在雄安自由贸易试验区内登记注册，具有独立法人的跨境收支企业。

**2. 外贸企业汇率避险奖励**
*   在雄安新区内登记注册、依法纳税，并具有独立法人资格的外贸企业。

# 申报条件

**1. 跨境人民币结算奖励申报条件**
1.  办理货物贸易、服务贸易、跨境直接投资（不含资金池业务）三类业务，且跨境人民币结算业务发生在2025年度；
2.  在雄安自由贸易试验区内登记注册，具有独立法人的跨境收支企业；
3.  企业有运作良好的内控管理机制，能够自行妥善留存跨境人民币业务相关交易单证等真实性证明资料，并愿意配合经办银行、外汇局的事中事后抽查、核实工作；
4.  守法自律，近两年无实质性违反或故意规避跨境人民币管理规定、外汇管理法规、国家“三反”规定等法律、法规的情形；
5.  未被纳入跨境人民币业务重点监管企业名单和货物贸易外汇管理B、C类企业；
6.  未涉嫌大规模跨境投机套利活动。

**2. 外贸企业汇率避险奖励申报条件**
*   **申请企业须具备以下条件：**
    1.  以实际需求为原则，有进出口贸易的真实交易背景；
    2.  资信良好，无不良信用记录；
    3.  货物贸易项下企业应为外汇管理名录登记的A类企业；
    4.  在雄安新区内登记注册、依法纳税。
*   **承办银行机构须具备以下条件：**
    1.  具备人民币与外汇衍生品业务资格；
    2.  具有专门的汇率避险产品和专业服务团队，能够为企业提供优质的汇率避险业务；
    3.  上年度外汇业务合规与审慎经营评估评级为“B”及以上的银行及其分支机构；
    4.  近两年无违反外汇管理规定记录。
