# 政策简介

为深入学习贯彻习近平总书记关于科学普及和科学素质建设的重要论述，贯彻落实新修订的《中华人民共和国科学技术普及法》和《全民科学素质行动规划纲要(2021-2035年)》，根据国家和河北省基层科普行动计划整体工作安排，结合雄安新区实际，实施2026年度基层科普行动计划。坚持“围绕中心、服务大局、注重公益”原则，激发各类科普主体开展科普活动的积极性，提升基层科普服务能力，增加优质科普公共服务供给，促进科普公平普惠。通过“以点带面、榜样示范”方式，加大优质科普公共资源倾斜力度，培育辐射带动能力强的基层科普典型案例，调动全社会参与基层科普工作的积极性，引领广大群众学科学、用科学，推动提升雄安新区公众科学素质。

# 奖励额度/补贴标准

2026年基层科普行动计划专项资金分配如下：
1. **国家级拨款**：共44万元，支持11家科普主体，平均每家4万元。
2. **省级拨款**：共13万元，支持4家科普主体，平均每家3.25万元。

# 注意事项

1. **申报资格限制**：近三年内有违法违规或受到行政处罚的单位和个人，不得申报该项目。
2. **发票要求**：申报单位及科技志愿者、科普信息员必须可以开具正式发票。
3. **公章要求**：申报书须加盖项目申报单位公章，申报单位所盖公章必须为具有独立法人资格、拥有独立账号单位的公章。
4. **材料格式要求**：
   - 申报书为A4纸张正反面打印，报送一式四份。
   - 同时发送申报书电子版到指定邮箱。
   - 相应栏目请填写完整，缺项或格式不符的申报书不予受理。
   - 各栏目如填写不下，可另加附页。
5. **经费使用限制**：经费使用方向仅限于与项目直接相关的费用支出。
6. **绩效目标**：各科普主体要推进预算绩效管理，合理安排资金支出进度，细化工作措施，确保完成任务目标，不断提升财政资金使用效益。

# 申请对象

支持范围包括：
1. 基层农技协、社区、科普教育(示范)基地及科技小院创建单位。
2. 已经落地的流动科技馆（不包括落地到实体科技馆享受科技馆免费开放资金补助的流动科技馆）、科普大篷车、农村中学科技馆运行(委托)单位。
3. 服务于新区基层科普工作的企事业单位、社会组织及科技志愿者、科普信息员等。

# 申报条件

1. **主体资格**：申报单位须具有独立法人资格、拥有独立账号。
2. **合规性**：近三年内无违法违规或受到行政处罚记录。
3. **财务能力**：申报单位及科技志愿者、科普信息员必须可以开具正式发票。
4. **内容要求**：
   - 围绕乡村振兴、食品安全、防灾减灾、健康新区、公共文化服务体系建设等党委政府中心工作。
   - 重点突出全国科普月、科技工作者日、科技周、双创周等品牌科普活动的汇聚集成。
   - 重点突出提升基层科普服务能力建设的具体目标。
   - 重点突出面向公民提供优质科普资源、科技培训、科普讲座、展教展示、发放宣传品等开展科普活动费用。

# 申报材料

2. **申报书电子版**（发送至指定邮箱）。

# 办理流程

1. **申报与推荐**：各片区积极组织符合条件的申报主体进行申报，并做好推荐工作。
2. **评审**：雄安新区工信科技数据局于第一季度末完成项目评审。
3. **中期绩效报送**：6月底前，将中期绩效完成情况报送雄安新区主管部门。
4. **项目实施与总结**：11月底前，完成项目主体实施工作，及时总结经验、确保实施效果，并将项目实施工作总结报送雄安新区相关主管部门。
