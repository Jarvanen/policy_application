# 政策简介

按照省工业和信息化厅工作安排，为推进项目与资本对接，促进新材料项目快速落地，省工业和信息化厅拟于12月中旬在石家庄组织举办河北省新材料产业项目资本对接活动。活动邀请中信银行、平安保险、金石投资等投融资机构与我省电子化学品、医药中间体、储能材料、生物医用材料、生物基材料、高效低毒农药、新型高效催化剂等领域有项目融资需求的优质企业进行现场对接，线上同时向国内投融资机构推介融资项目。

# 申请对象

新区新材料领域有融资需求项目。

# 申报条件

1. 属于新区企业。
2. 属于新材料领域。
3. 有项目融资需求。
4. 属于优质企业（涉及领域包括电子化学品、医药中间体、储能材料、生物医用材料、生物基材料、高效低毒农药、新型高效催化剂等）。

# 申报材料

1. 企业相关资料。
2. 企业项目资本对接基本情况表（附件2）。

# 办理流程

1. 报名提交企业相关资料。
2. 择优推荐参加现场项目资本对接路演。
