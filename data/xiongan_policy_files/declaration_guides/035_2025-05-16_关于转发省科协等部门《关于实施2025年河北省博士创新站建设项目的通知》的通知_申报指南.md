# 政策简介

河北省博士创新站是中国科协打造的重点人才创新服务项目，旨在着力激发科协基层组织活力，推动高校、科研院所博士与中小微企业开展建站合作和技术攻关，搭建产学研用合作平台，解决中小微企业缺技术、缺人才，博士缺项目、缺实践难题，有效打通高校、科研院所与企业的人才流动通道，服务科技工作者成长成才和区域中小微企业创新发展。

# 申请对象

1. **申报主体**：以中小微企业为建站主体。
2. **产业方向**：企业所属产业以河北省八大重点产业（现代化钢铁、绿色化工、生物医药、电子信息、新能源和智能网联汽车、机器人、空天信息和卫星互联网、数字产业）及107个县域重点特色产业集群范畴内的为主。
3. **特殊情形**：各地已具备博士创新站项目实施条件的河北省科技专家企业工作站，可直接申报认定为博士创新站，不受产业条件限制、不占分配名额。

# 申报条件

**（一）申报企业条件**
1. 申报企业须在河北省内依法注册，具备独立法人资格，且生产经营状况良好。
2. 建站企业须以科技创新和人才引育需求为导向，以产学研用合作为纽带，有明确的技术创新需求和合作项目。
3. 建站企业须能为博士及其创新团队进站工作提供必要的科研、生活条件及后勤保障。
4. 建站企业须配备研发及工作人员不少于5名。

**（二）进站博士条件**
1. 进站博士须为省内外高等学校、科研院所等单位在职职工。
2. 进站博士须与建站企业不存在人事隶属关系。
3. 进站博士年龄不超过50周岁（1975年1月1日及以后出生）。

**（三）博士团队构成**
1. 博士人才应在相关领域具有较高的学术水平或技术研发能力，能够为建站企业提供强有力的技术支持与创新指导。
2. 博士团队结构须科学合理，成员不少于5名，其中博士不少于3名（含领衔博士1名）。
3. 领衔博士服务的建站企业不超过1家。

**（四）合作协议要求**
1. 建站企业与进站博士需签订博士创新站共建协议和科技项目合作协议（合同）。
2. 有明确的研究方向和科技创新与成果转化合作任务，合作方向符合国家法律法规和国家、省、市、县产业政策导向。
3. 合作协议中应明确双方的权利、义务、目标、科研成果归属和收益分配及违约责任等。
4. 协议（合同）合作期限一般不少于3年。

# 申报材料

1. 《河北省博士创新站建设项目申报书（2025年）》。
2. 博士创新站共建协议。
3. 科技项目合作协议（合同）。
4. 领衔博士及团队其他成员毕业证、学位证复印件（海外学历的需提供教育部留学服务中心认证书）。
5. 博士人事关系所在单位同意建站证明。
6. 企业法人证书等相关建站资料。
7. 上述材料纸质版1份，经推荐单位审核签字盖章后报送。
8. 与纸质版相同的电子版（PDF格式）。

# 办理流程

1. **自愿申报**：符合条件的企业自愿申报并填写《河北省博士创新站建设项目申报书（2025年）》。
2. **遴选推荐**：各市科协联合教育局、工商联，依据项目实施条件和要求、最低推荐数量，共同遴选推荐本地区建站企业，相关推荐材料由市科协出具推荐意见后统一提交至省科协。省科协所属各省级学会（协会、研究会）和团体会员不限推荐名额。
3. **形式审查**：省科协对申报材料进行形式审查，视情赴现场调研核实。
4. **专家评审**：省科协组织相关领域专家对通过形式审查的申报材料进行评审。
5. **公示认定**：评审结果经审定后公示，对公示无异议的项目实施站点予以认定，有效期3年（2025年至2028年）。
6. **牌匾授予**：主办单位对项目实施站点统一命名，向建站单位及进站博士颁发牌匾和证书。
7. **评估总结**：项目实施站点需按要求开展工作，接受年度评估总结。

2. **全省统一报送**：纸质版及电子版材料请于2025年6月6日前报送至指定地址/邮箱，逾期不予受理。

# 注意事项

1. **材料格式**：申报材料用A4纸双面打印，左侧装订。
2. **取消资格情形**：有下列情形之一的，取消认定资格：
   - 申报过程中隐瞒真实情况、提供虚假材料的；
   - 侵犯他人知识产权，被依法确认为侵权行为的；
   - 建站主体发生重大安全、质量、严重环境污染等事故受到行政、刑事处罚的；
   - 其他不符合省级博士创新站认定要求的。
3. **动态管理**：博士创新站实行动态管理和退出机制，每年对博士创新站建设项目实施情况和绩效进行评估总结。对绩效不达标的，提出整改建议或撤销建站资格。
4. **建站期间义务**：经认定的河北省博士创新站，建站期间须参与河北省科协组织的工作调研、技术指导、人才培养、交流展示、案例征集等工作；主动参与全国科普月相关活动，面向社会公众开放企业实验室、生产线或展厅展馆，积极履行社会责任；收集并提供不少于3名理工类博士信息、5项科技成果，梳理不少于5条建站企业技术创新需求。
