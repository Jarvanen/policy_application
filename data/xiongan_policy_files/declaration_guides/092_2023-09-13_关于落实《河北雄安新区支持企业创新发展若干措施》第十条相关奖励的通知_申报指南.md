# 政策简介

为鼓励支持科技企业孵化器及众创空间建设发展，贯彻落实《河北雄安新区支持企业创新发展若干措施》（雄安政字〔2022〕29号），河北雄安新区改革发展局对符合该文件第十条规定的科技企业孵化器、众创空间进行奖励。

# 奖励额度/补贴标准

自2021年2月10日后首次认定为国家级、省级科技企业孵化器、众创空间的，一次性给予50万元奖励。

# 注意事项

1. 认定依据：以科技部、河北省科技厅发布的认定文件为准。
2. 办理方式：采用“免申即享”的方式进行奖励。
3. 备案要求：申领单位需认真做好预算编制、指标安排等相关工作，按照下达资金用途，合理安排资金，加强规范管理，提高资金使用效益。
4. 材料提交：填写收款人列表和绩效目标表并加盖公章后报雄安新区改革发展局备案。
5. 提交期限：于2023年9月18日前报送。

# 申请对象

在新区辖域范围内工商注册、税务登记并开展实际经营活动的科技企业孵化器、众创空间。

# 申报条件

1. 在新区辖域范围内工商注册、税务登记并开展实际经营活动。
2. 自2021年2月10日后首次认定为国家级、省级科技企业孵化器、众创空间。

# 申报材料

1. 收款人列表（附件1）
2. 绩效目标表（附件2）

# 办理流程

1. 申领单位填写收款人列表和绩效目标表。
2. 加盖公章。
3. 报送至雄安新区改革发展局备案。

# 申报时间

2023年9月18日前。
