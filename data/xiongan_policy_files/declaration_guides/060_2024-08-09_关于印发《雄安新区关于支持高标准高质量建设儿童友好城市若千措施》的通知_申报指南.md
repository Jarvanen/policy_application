# 政策简介

本措施旨在广泛动员社会力量参与雄安新区儿童友好城市建设，支持在雄安新区范围内注册的社会主体（包括企业、民办非企业单位、基层群众性自治组织、其他社会团体）及雄安新区企事业单位，通过建设儿童友好实践基地、科普基地、儿童之家、举办儿童友好活动、开展公益健康科普服务、进行适儿化空间改造或新建、提供婴幼儿托育服务等方式，高标准高质量推进儿童友好城市建设。

# 奖励额度/补贴标准

1. **儿童友好实践基地**：对认定为雄安新区儿童友好实践基地且复核评分达到前30%的基地运营主体，一次性给予2万元奖补资金。
2. **儿童友好科普基地**：
   - 认定为五星级省级科普示范基地的，一次性给予20万元运营补助；
   - 认定为四星级省级科普示范基地的，一次性给予15万元运营补助；
   - 认定为三星级省级科普示范基地的，一次性给予10万元运营补助；
   - 认定为国家科普教育基地、国家自然资源科普基地等各类国家级科普基地的，一次性给予50万元运营补助。
   - *注*：科普基地实现晋级的，以“晋级累差”的方式给予补助；同时认定为五星级省级科普示范基地和国家级科普基地的，按照国家级科普基地的标准给予运营补助。
3. **儿童之家/妇女儿童之家**：
   - 被评为省级示范妇女儿童之家的，在获得省级专项建设资金的基础上，雄安新区给予1:1配套资金；
   - 被评为省级儿童之家示范点的，在获得省级奖励的基础上，雄安新区给予2万元奖补资金；
   - 被评为新区示范性儿童之家的，对基地运营主体给予1万元奖补资金。
4. **儿童友好活动**：授予“雄安新区儿童友好城市建设先进组织”荣誉称号的社会主体，于当年底给予3万元奖补资金。
5. **公益健康科普服务活动**：符合当年度“国家和河北省基层科普行动计划”要求的相关科普活动，按照当年省级下达资金数额给予1:1比例的配套资金。
6. **适儿化改造或新建项目**：采取后补助的方式给予项目实际总投资最高20%的比例资金补助，补助资金总额原则上不超过100万元。
7. **婴幼儿托育服务**：
   - 符合《托育机构质量评估标准》中A档要求的，按照1600元/床的标准，一次性给予总额不超过160000元的奖补资金；
   - 符合《托育机构质量评估标准》中B档要求的，按照1100元/床的标准，一次性给予总额不超过110000元的奖补资金。
8. **特殊项目**：对于特殊或特别重大的儿童友好城市建设项目、儿童友好主题活动，采取一事一议的方式予以支持。

# 注意事项

1. **有效期**：本措施自印发之日起施行，有效期2年。
2. **投资门槛**：适儿化改造或新建项目，实际总投资额原则上不得低于20万元。
3. **补助上限**：适儿化改造或新建项目，补助资金总额原则上不超过100万元。
4. **托育奖补上限**：托育场所运营主体获得的奖补资金总额分别不超过160000元（A档）和110000元（B档）。
5. **认定标准**：
   - 科普基地需认定为省级或国家级相应等级；
   - 儿童之家需被评为省级或新区示范性；
   - 托育场所需符合《托育机构质量评估标准》（国卫通〔2023〕13号）中A、B两档要求；
   - 适儿化改造或新建项目需符合《城市儿童友好空间建设导则（试行）》要求及雄安新区地方相关标准。
6. **评估机制**：儿童友好活动由第三方根据活动备案情况、现场效果、参与嘉宾情况、媒体宣传报道情况、参与人数、持续时间等因素进行综合评估打分。

# 申请对象

1. 在雄安新区范围内注册的企业、民办非企业单位（组织）、基层群众性自治组织、其他社会团体（统称“社会主体”）。
2. 雄安新区企（事）业单位（适用于科普基地建设、儿童友好活动举办等）。

# 申报条件

1. **主体资格**：基地运营主体或活动举办主体需符合第一条规定的支持范围（在雄安新区注册的社会主体或新区企事业单位）。
2. **实践基地条件**：被认定为雄安新区儿童友好实践基地，且复核评分达到前30%。
3. **科普基地条件**：
   - 被认定为三星级、四星级或五星级省级科普示范基地；
   - 或被认定为国家科普教育基地、国家自然资源科普基地等各类国家级科普基地。
4. **儿童之家条件**：
   - 被评为省级示范妇女儿童之家；
   - 或被评为省级儿童之家示范点；
   - 或被评为新区示范性儿童之家。
5. **活动组织条件**：举办各类儿童友好活动，经第三方综合评估打分达到规定分值，获得“雄安新区儿童友好城市建设先进组织”荣誉称号。
6. **科普活动条件**：组织开展儿童视力保健、牙齿保健等公益健康科普服务活动，符合当年度“国家和河北省基层科普行动计划”要求。
7. **空间改造/新建条件**：
   - 采取市场化方式对街区、道路、社区（村）、公园、绿地等城市、农村公共空间和公共服务设施进行适儿化改造，或新建儿童友好空间或公共服务设施；
   - 项目符合《城市儿童友好空间建设导则（试行）》要求及雄安新区地方相关标准；
   - 实际总投资额原则上不低于20万元。
8. **托育服务条件**：
   - 为0-3岁婴幼儿提供托育服务，市场化运营；
   - 符合《托育机构质量评估标准》（国卫通〔2023〕13号）中A、B两档要求；
   - 经行业主管部门评估认定通过。

# 办理流程

1. **认定/评估**：
   - 实践基地需经过复核评分；
   - 科普基地需获得省级或国家级认定；
   - 儿童之家需获得省级或新区示范评定；
   - 活动需经第三方综合评估打分；
   - 托育场所需经行业主管部门评估认定通过。
2. **补助方式**：
   - 大部分项目采取一次性给予奖补资金或运营补助；
   - 适儿化改造或新建项目采取后补助方式。
3. **资金拨付**：
   - 活动奖补资金于当年底给予；
   - 其他奖补资金在认定或评估通过后按规定程序拨付。

# 申报时间

本措施自印发之日（2024年8月9日）起施行，有效期2年。具体申报批次和时间需关注相关部门通知。
