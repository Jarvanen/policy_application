# 政策简介

为落实《河北·京南国家科技成果转移转化示范区建设方案（2022-2025年）》，加快建立以企业为主体、市场为导向、产学研深度融合的技术创新体系，实现技术成果与科技企业的精准对接，促进更多优质技术成果在京南示范区落地转化，助力企业技术创新和产业升级，促进全省经济高质量发展，河北省科学技术厅启动开展“2023河北·京南示范区科技成果直通车”活动。活动采取“直通车+对接会”的形式，在石家庄、保定、廊坊、沧州、衡水、雄安新区各组织1场市（区）科技成果直通车活动，在部分县（市、区）开展直通车供需对接会。

# 申请对象

京南示范区（五市一区）各选择1-2个主导、特色产业的企业。

# 申报条件

1. 产业领域：涉及但不限于生物医药、健康医疗、新能源汽车、智能电网、临空产业、现代农业等领域。
2. 需求明确：企业需求明确，作为成果直通车活动的主题。

# 申报材料

1. 《技术需求信息登记表》（附件1/附件2）：包含企业名称、地域、通讯地址、联系人、联系电话、技术领域、技术需求描述（300字以内）、意向合作专家、单位公章及签字日期。
2. 《技术需求汇总表》（附件3）：由市（区）科技部门整理汇总，包含序号、企业名称、地域、需求名称、技术领域、联系电话。
3. 《京南示范区成果直通车活动申报表》（附件1，针对共同承办单位）：包含市(区)名称、产业领域、共同承办单位、联系人、联系电话、拟邀请协办单位、小型供需对接会信息（县(市、区)、技术领域及需求概述）。

# 办理流程

1. 企业填写《技术需求信息登记表》。
2. 各市（区）科技部门负责组织辖区内企业填写登记表，并将征集到的企业技术需求进行整理汇总，填写《技术需求汇总表》。
3. 各市（区）科技部门将整理汇总的所有文件以压缩包的形式发送至指定邮箱。
