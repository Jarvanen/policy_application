# 政策简介

本政策旨在贯彻落实《关于促进“专精特新”企业和数字经济核心产业高质量发展的若干措施》，明确雄安新区“专精特新”企业培育库的具体管理措施。通过建立培育库，引导和培育新区企业高质量发展，对入库企业及后续获得省级、国家级认定的企业给予资金奖补。资金从新区改革发展局产业发展资金列支。

# 奖励额度/补贴标准

1. **入库补助**：首次完成入库的企业，一次性给予 **20万元** 补助。
2. **省级认定奖励**：企业入库后，首次被评定为河北省省级“专精特新”中小企业，给予 **30万元** 奖励性补贴。
3. **国家级认定奖励**：省级“专精特新”中小企业首次被评定为国家级专精特新“小巨人”企业，再次给予 **50万元** 奖励性补贴。

# 注意事项

1. **免申即享**：已被认定为河北省省级“专精特新”中小企业的新区企业，可“免申即享”直接纳入新区“专精特新”企业培育库；库内企业在申报省级/国家级认定补贴时也可“免申即享”。
2. **申报频率**：奖补政策每半年组织集中申报一次。
3. **资金发放形式**：公示无异议后，通过 **数字人民币** 形式向企业发放。
4. **动态管理与退出**：
   - 入库企业需在每年4月之前更新企业信息。
   - 对于考核评价不再达到入库标准，或发生重大安全（含网络安全、数据安全）、质量、环境污染等事故，或严重失信、偷漏税等违法违规行为的企业，将被移出培育库。
5. **违规处罚**：
   - 申报过程中出现材料弄虚作假、骗取奖补资金等情况，经查实后将追回资金，列入平台不诚信企业名单，取消入库资格，不再享受相关政策支持，且 **至少三年内不得再次申报**。
   - 涉及违法、违纪问题将移送相关部门严肃处理。
6. **有效期**：本办法自发布之日起实施，有效期 **2年**。

# 申请对象

1. 纳入雄安新区“科技企业库”的企业。
2. 符合《中小企业划型标准规定》（工信部联企业〔2011〕300号）的 **中小微型企业**。
3. 已通过河北省“创新型中小企业”评定的企业。

# 申报条件

纳入“专精特新”企业培育库，应同时达到以下条件：

1. **基础资质**：纳入雄安新区“科技企业库”，符合中小微型企业标准，且通过河北省“创新型中小企业”评定。
2. **经营年限**：从事特定细分市场时间达到 **2年以上**。
3. **研发指标**：上年度研发费用总额不低于 **100万元**，且占营业收入总额比重不低于 **3%**。
4. **营收或融资指标**（满足其一即可）：
   - 上年度营业收入总额在 **1000万元以上**；
   - 或上年度营业收入在1000万元以下，但近2年新增股权融资总额（合格机构投资者的实缴额）达 **2000万元以上**。
5. **评分或直通条件**（满足其一即可）：
   - 按照评价指标体系达到 **45分以上**；
   - 或满足下列条件之一：
     (1) 近三年获得过省级科技奖励（排名前三），或获得过国家级科技奖励（排名前五）；
     (2) 近两年研发费用总额均值在 **1000万元以上**；
     (3) 近两年新增股权融资总额（合格机构投资者的实缴额） **6000万元以上**；
     (4) 近三年进入“创客中国”中小企业创新创业大赛全国500强企业组名单。
6. **负面清单**：
   - 近三年未发生过重大安全、重大质量事故和严重环境违法；
   - 提供的产品和服务不属于国家禁止、限制和淘汰类；
   - 未被列入经营异常名录和严重失信主体名单。

# 申报材料

1. 雄安新区“专精特新”企业培育库入库申报表（附件2）；
2. 近三年财务报表（资产负债表、利润表、现金流量表），加盖企业公章（企业成立不满三年，可提供近两年）；
3. 企业近两个年度审计报告（需体现研发费用、研发投入占销售收入比例、主营业务收入总额占营业收入总额比重，也可单独证明）；
4. 企业申请入库/补贴承诺书（盖企业公章）（附件3）；
5. 申报表填报内容相关佐证材料；
6. 审核部门要求的其他相关证明材料；
7. 平台所需其他基础信息。

*注：申报省级/国家级认定补贴时，还需上传评定证书及公示性文件等材料。*

# 办理流程

**一、入库流程**

1. **申报**：
   - 一般企业：通过雄安新区产业互联网平台“专精特新企业培育库入库申请”模块申报，系统校验通过后导出“入库申报表”，填报完善并加盖公章，报三县审核部门审核。
   - 省级专精特新企业：通过平台“免申即享”模块进行一键核验。
2. **审核**：三县审核部门在收到申请后 **7个工作日内** 完成审核，不符合条件的予以退回。
3. **汇总、评审及公示**：审核部门形成审批汇总表，报县政府同意后报送新区改革发展局。新区改革发展局委托第三方组织专家评审，通过后在平台公示（不少于5日）。
4. **入库**：公示无异议后，新区改革发展局在平台确认入库。

**二、奖补申报流程**

1. **申报**：企业在平台上传评定证书、公示文件及承诺书，发起申请（库内企业可“免申即享”）。
2. **初审**：三县审核部门对材料完整性、真实性、有效性及企业运营状态、信用安全等进行审核、勘察，导出汇总表加盖县政府公章后报新区改革发展局。
3. **审核**：新区改革发展局审核，并要求通过审核企业报送纸质版材料备案。
4. **汇总、备案及公示**：新区改革发展局组织局务会集体研究，在平台确认并公示（不少于5日）。
5. **资金拨付**：公示无异议后，通过数字人民币发放。

# 申报时间

1. **入库申报**：一般每年 **4月、10月** 为申报窗口期，具体时间根据当年省“专精特新”中小企业认定时间予以调整。
2. **奖补申报**：每半年组织集中申报一次。
