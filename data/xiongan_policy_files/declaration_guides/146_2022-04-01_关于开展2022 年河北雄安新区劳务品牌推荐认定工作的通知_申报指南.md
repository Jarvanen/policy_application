# 政策简介

为深入推进省委、省政府“三统筹三扩大四创建”活动，切实加强雄安新区劳务品牌建设，开展2022年河北雄安新区劳务品牌推荐认定工作。围绕“五新”目标，聚焦雄安新区“5+6”产业体系，选树3-5个具有雄安特色、行业特征、技能特点、带动就业能力强的雄安新区劳务品牌。

# 奖励额度/补贴标准

对认定的新区级劳务品牌给予5万元资金补助，支持品牌发展建设，进一步增强就业带动能力。

# 注意事项

1. **申报平台**：此次认定工作全流程通过雄安新区产业互联网平台进行。
2. **动态监管与整改**：加强品牌监督管理和绩效考核，每年按照县级自评、新区核查的监管程序，对新区级劳务品牌实行年度动态监管。对带动就业人数比上年度增加不足5%或就业人数增加少于50人、市场竞争力和社会认可度下降明显以及造成其他不良影响的劳务品牌，责令建设单位限期整改。
3. **诚信要求**：对建设单位存在违法违规、经营失信、材料失真等行为，一经发现，取消申报资格。
4. **审核责任**：按照“谁签字、谁负责”的原则，层层做好审核把关，通过核验材料、实地核查、随机走访等方式，确保申报资料真实有效。

# 申请对象

劳务品牌建设主体为依法成立3年以上（截至2022年3月31日）的独立法人单位。

# 申报条件

1. **主体资质**：
   - 依法成立3年以上（截至2022年3月31日）的独立法人单位。
   - 无违法违规失信行为。
   - 有固定经营或办公场地，场地产权清晰。
2. **管理规范**：
   - 各项制度健全，管理规范。
   - 有相应专业知识和技能的服务管理团队，能够提供基本办公服务和后勤保障。
3. **带动就业指标**（截至2022年3月31日）：
   - **中高端技能型、民生保障型劳务品牌**：3年内累计从业人员不低于3000人，1年内从业人员不低于1000人。
   - **高品质服务型、文化和旅游类及其他劳务品牌**：3年内累计从业人员不低于2000人，1年内从业人员不低于800人。
   - 从业人员接受与劳务品牌相关的各类培训比例在30%以上。
   - 品牌创造的经济效益较好。
4. **权益维护**：
   - 注重维权服务，与从业人员依法签订劳动合同（就业协议），或组织推动从业人员与实际用人主体签订劳动合同（就业协议）。
   - 实现促进有组织就业，从业人员合法权益得到有效维护。
5. **品牌形象**：
   - 有独立品牌名称，有形象代言人。
   - 品牌命名突出地区、产业、行业、职业特点。
   - 具有一定知名度、信誉度、美誉度。
   - 近3年内在市级以上主流媒体有过新闻报道或推广宣传。
6. **持续发展**：
   - 具备持续推动劳务品牌培育、发展、壮大的能力。
   - 制定相应工作计划，并得到有效实施。

# 申报材料

1. 《河北雄安新区劳务品牌推荐申报表》。
2. 劳务品牌建设成果简介（2000字以内）。
3. 劳务品牌建设工作计划、管理制度和管理人员花名册。
4. 市级以上主流媒体新闻报道等宣传资料。
5. 劳务品牌从业人员花名册（随机抽取500人）。

# 办理流程

1. **组卷材料**：劳务品牌建设单位认真组织撰写申报材料。
2. **县级推荐**：按照属地管理原则，由建设单位向所在县人社部门提出申请，县人社部门初审。三县严格把关申报材料，认真组织开展审核工作，择优确定雄安新区劳务品牌推荐对象。（4月25日前完成）
3. **新区评审**：雄安新区公共服务局会同有关部门，组成专家评审组，按照公开、公平、公正的原则，对各地申报劳务品牌进行资料审核和综合评审，评选出雄安新区劳务品牌候选名单。（4月30日前完成）
4. **结果公示**：对候选名单进行5天的公示，公示期满无异议的，认定为雄安新区劳务品牌。（5月10日前完成）

# 申报时间

- 县级推荐完成时间：4月25日前
- 新区评审完成时间：4月30日前
- 结果公示完成时间：5月10日前
