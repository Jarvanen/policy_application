# 政策简介

第十四届中国创新创业大赛河北雄安新区赛区暨雄安新区第六届“智绘未来”创新创业大赛由雄安新区工信科技数据局、改革发展局、高新区管委会、科学园管委会联合主办。大赛以“科技引领·向新同行”为主题，秉承“政府引导、公益支持、市场机制”的办赛理念，旨在激发市场主体活力，推进大中小企业融通发展，挖掘整合京津冀地区创新资源，选拔符合新区定位的优秀创业创新项目、团队和企业，培育创新驱动发展新引擎，加快实现雄安新区“新功能、新形象、新产业、新人才、新机制”目标。

# 奖励额度/补贴标准

**企业组奖金（需在雄安新区落地）：**
*   一等奖：60万元
*   二等奖：30万元
*   三等奖：20万元
*   优秀奖：4万元

**团队组奖金（需在雄安新区落地）：**
*   一等奖：30万元
*   二等奖：20万元
*   三等奖：10万元
*   优秀奖：0.3万元

**其他奖励：**
*   各组别获奖项目颁发奖杯、证书。
*   组织贡献奖：对大赛组织协办有突出贡献的社会机构和企事业单位授予组织贡献奖并颁发奖牌。

# 注意事项

1.  **报名限制：** 参赛对象只能选择企业组、团队组其中之一报名，不得重复报名。
2.  **地域限制：** 在雄安新区和其他地方都注册的企业，同一个项目只能选择一个地方参赛；同一项目不得跨地区重复报名。
3.  **往届获奖限制：**
    *   2024年获得第十三届中国创新创业大赛河北雄安新区赛区暨雄安新区第五届“智绘未来”创新创业大赛一、二等奖的企业或团队，不参加本届大赛。
    *   2025年获得“雄才杯”创新创业赛事一等奖及以上的企业或团队，不参加本届大赛。
4.  **信息授权：** 参赛对象及参赛项目同意在媒体公开项目、实体及团队有关信息。
5.  **项目要求：** 项目内容不涉密。
6.  **唯一渠道：** 大赛官网是报名参赛的唯一渠道，其他报名渠道均无效。

# 申请对象

**企业组：**
*   面向雄安新区本级及三县注册的企业。

**团队组：**
*   面向雄安新区及国内外在雄安创新创业的各类团体（如海外留学回国创业人员、进入创业实施阶段的优秀科技团队、大学生创业团队、创客团队等）。
*   报名时参赛项目所在地需选择雄安新区。

# 申报条件

**（一）企业组参赛条件**
1.  企业须在雄安新区注册满半年以上，具有独立法人资格，具有创新能力和高成长潜力，拥有知识产权且无产权纠纷，主要从事高新技术产品研发、制造、服务等业务。
2.  企业经营规范、社会信誉良好、无不良记录，且为非上市企业。
3.  企业2024年营业收入不超过2亿元人民币。
4.  **分组要求：**
    *   初创企业组：工商注册时间在2024年1月1日（含）之后的企业（疏解企业不受限制）。
    *   成长企业组：其他企业。
5.  符合上述“注意事项”中关于往届获奖的限制条件。

**（二）团队组参赛条件**
1.  在大赛报名截止时间之前，尚未在国内注册成立企业的创业团队。
2.  参赛人应具有在雄安新区创业意愿，计划赛后在雄安新区注册成立企业或与雄安新区相关产业达成对接合作。
3.  核心团队成员不少于3人（含）、不超过7人（含），具有良好的身体素质，无科研失信记录。
4.  参赛项目的产品、技术及相关专利归属参赛团队，具有一定的创新性和商业化、市场化潜力，与其他任何企业、团体、个人无产权纠纷。
5.  **分组要求：**
    *   国际组：团队负责人为外籍人员。
    *   国内组：其他团队。
6.  符合上述“注意事项”中关于往届获奖的限制条件。

**（三）通用条件**
*   报名参赛项目应符合国家法律法规和新区产业政策，经营规范，社会信誉良好，无不良记录，不侵犯任何第三方知识产权。

# 申报材料

*   登录大赛官方网站进行统一注册报名。
*   在系统统一身份认证后，提交完整报名材料。
*   对所填信息的准确性和真实性负责。

# 办理流程

1.  **注册报名：** 自评符合参赛条件的企业和团队自愿登录大赛官方网站进行统一注册报名。
    *   企业报名网址：www.cxcyds.com
    *   团队报名网址：www.hbscxcyds.com
2.  **提交材料：** 在系统统一身份认证后，提交完整报名材料。
3.  **初赛评审：** 7月上旬，以封闭评审的方式进行，由大赛组委会聘请专家对参赛作品进行评审。
4.  **复赛：** 时间待定，采取“项目路演、评委提问、现场打分”的模式，由大赛组委会聘请专家对参赛作品进行评审。
5.  **决赛及颁奖典礼：**
    *   企业组和团队组分别有16支队伍进入决赛。
    *   决赛采取“项目路演、评委提问、现场打分”的模式，由大赛组委会聘请专家对参赛作品进行评审。
    *   决赛每个组别分别评出一等奖1名、二等奖2名、三等奖3名、优秀奖10名。
    *   决赛结果产生后，组委会举办颁奖典礼。
