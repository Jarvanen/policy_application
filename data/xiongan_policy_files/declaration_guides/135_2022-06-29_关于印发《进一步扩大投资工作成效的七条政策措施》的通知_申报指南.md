# 政策简介

本政策旨在进一步扩大投资工作成效，通过加大政府投资项目项目建设管理费奖励、加强中央预算内投资争取、扎实做好政府专项债券申报和使用、加强基础设施项目梳理谋划和推动实施、加快推进三县项目建设、加大项目建设服务保障力度以及加快谋划研究2023年建设项目等七条措施，推动项目进度快、成效好的集中建设管理单位及符合条件的项目获得支持。

# 奖励额度/补贴标准

**一、政府投资项目项目建设管理费奖励**

1.  **2022年新开工政府投资项目：**
    *   经认定为2022年6月30日前开工建设的项目：额外奖励30%项目建设管理费。
    *   经认定为2022年9月30日前工程开工建设的项目：额外奖励20%项目建设管理费。
    *   经认定为2022年12月30日前开工建设的项目：额外奖励10%项目建设管理费。
2.  **2023年新开工政府投资项目：**
    *   经认定为2022年12月30日前开工建设的项目：额外奖励30%项目建设管理费。
3.  **计划完工的新区本级政府投资项目（2022年、2023年）：**
    *   按《雄安新区项目建设提质增效行动工作方案》确定时间完工的项目：额外奖励30%项目建设管理费。
    *   提前完工项目：额外奖励50%项目建设管理费。
4.  **其他说明：**
    *   以上各项奖励的项目建设管理费属于单独核算金额，不占用项目竣工验收后考核优秀的奖励额度。
    *   奖励的项目建设管理费可从初步设计概算批复的预备费中列支，但不得突破已批复的概算。

**二、中央预算内投资争取工作奖励**

1.  **新区本级政府投资项目：**
    *   获得中央预算内支持，且在项目完工验收后评定为合格及以上等次的，可直接认定为优秀等次，给予集中建设管理单位30%项目建设管理费奖励。
2.  **新区财政资金以资本金注入形式投资建设的政府投资项目（包括但不限于供热、燃气等领域）：**
    *   获得中央预算内资金的50%计入企业投资部分。
    *   除去中央预算内投资补助部分，其他资金按照批复明确比例进行落实。
3.  **三县政府投资项目：**
    *   获得中央预算内投资的，新区财政资金可按扣除中央预算内投资后缺额部分给予不少于30%的补助资金。
    *   补助资金按照中央预算内投资计划下达时的投资进行测算。
    *   新区财政资金已足额保障的三县政府投资项目不再进行投资补助。

**三、三县项目建设投资补助**

1.  对列入新区年度建设项目清单（三县建设项目专项）的三县政府投资项目，按年度核定投资规模予以投资补助。
2.  对于三县本级2022年完成固定资产投资考核排名第一的县：
    *   对列入新区2023年建设项目清单且在2022年12月底前能开工建设的政府投资项目，优先给予投资补助。

# 注意事项

1.  **认定标准：**
    *   **项目开工认定：** 完成施工、监理招标，并提供监理开工令、进场文字和图片证明。
    *   **项目完工认定：** 完成全部在建工程竣工验收。
2.  **资金列支与拨付：**
    *   奖励的项目建设管理费可从初步设计概算批复的预备费中列支，但不得突破已批复的概算。
    *   资金申请和拨付按新区财政资金管理有关要求执行。
3.  **限制条件：**
    *   以上各项奖励的项目建设管理费在项目达到相应条件后即可申请。
    *   新区财政资金已足额保障的三县政府投资项目不再进行投资补助。
    *   三县列入清单的项目原则上必需满足社会投资项目不少于政府投资项目的比例要求。
4.  **有效期：**
    *   加强中央预算内投资争取措施（第二条）有效时间为自发布之日起执行，有效期一年。

# 申请对象

1.  **集中建设管理单位：** 推动项目进度快、成效好的集中建设管理单位。
2.  **政府投资项目：**
    *   列入2022年、2023年新开工的政府投资项目。
    *   列入2022年、2023年计划完工的新区本级政府投资项目。
    *   获得中央预算内支持的新区本级政府投资项目。
    *   新区财政资金以资本金注入形式投资建设的政府投资项目（包括但不限于供热、燃气等领域）。
    *   获得中央预算内投资的三县政府投资项目。
3.  **三县项目：**
    *   列入新区年度建设项目清单（三县建设项目专项）的三县政府投资项目。
    *   三县本级2022年完成固定资产投资考核排名第一的县的相关项目。
4.  **社会资本：** 参与基础设施项目建设、运营管理等环节的社会资本。

# 申报条件

1.  **开工时间条件：**
    *   2022年新开工项目需分别满足6月30日、9月30日、12月30日前开工。
    *   2023年新开工项目需满足2022年12月30日前开工。
2.  **完工时间条件：**
    *   新区本级政府投资项目需按《雄安新区项目建设提质增效行动工作方案》确定时间完工或提前完工。
3.  **验收等次条件：**
    *   获得中央预算内支持的新区本级政府投资项目，完工验收后需评定为合格及以上等次。
4.  **考核排名条件：**
    *   三县本级2022年完成固定资产投资考核排名第一。
5.  **清单纳入条件：**
    *   项目需列入新区年度建设项目清单（三县建设项目专项）。
    *   三县项目原则上需满足社会投资项目不少于政府投资项目的比例要求。
6.  **资金保障条件：**
    *   申请三县政府投资项目补助的，新区财政资金不得已足额保障。

# 办理流程

1.  **申请：** 项目达到相应条件后即可申请奖励的项目建设管理费。
2.  **申报：**
    *   积极把握国家扩大政府专项债券支持范围机遇，确保国家、省通知下达后能马上组织申报。
    *   加快研究将三县符合条件的项目纳入新区政府专项债券申报范围，予以统筹申报。
    *   将符合条件的项目优先申报政府专项债券支持。
3.  **审批与服务：**
    *   全面推行“不见面”审批，实行项目在线申报、在线审批、在线政策解答、在线评审等。
    *   积极采用“容缺后补”审批方式。
    *   建立健全市场化开发项目“精准对接”服务机制，由新区投资（产业）主管部门、行业主管部门、审批管理部门的主管领导组成工作服务小组，实行主动靠前服务和提供业务指导。
4.  **拨付：** 资金申请和拨付按新区财政资金管理有关要求执行。

# 申报时间

1.  **中央预算内投资争取措施有效期：** 自发布之日起执行，有效期一年。
2.  **2023年建设项目谋划：** 力争在6月底前完成整体研究。
3.  **专项债券开工要求：** 债券资金落实后2个月内能开工建设。
4.  **特定项目开工节点：**
    *   2022年6月30日前、9月30日前、12月30日前。
    *   2023年建设项目清单中在2022年12月底前能开工建设的项目。
