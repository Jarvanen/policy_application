# 政策简介

为加快新区科技型企业创新发展，有序落实《河北雄安新区关于支持企业创新发展的若干措施》(雄安政字〔2021〕5号)，组织开展《若干措施》第十二条中关于在孵企业办公用房租金补贴（以下简称“房租补贴”）的第二批线上申报。

# 申请对象

申报企业须是国家级、省级或新区认定的科技企业孵化器内的在孵企业。

# 申报条件

1. 申报周期：第二批申报周期为2021年10月1日至2022年3月31日。
2. 在孵状态：企业须为在孵企业。
3. 材料合规性：
   - 申请日期须在补贴周期内；
   - 注册地址须在孵化园区内；
   - 合同须加盖公章、有有效签字、有有效日期。
4. 真实性要求：不得弄虚作假、伪造材料，否则不予奖补并取消相关条款的申报资格。

# 申报材料

1. 企业基本信息；
2. 企业租房信息，需上传入孵协议扫描件；
3. 企业财务信息，需上传企业审计报告或财务报表、纳税证明，如无需上传说明；
4. 企业员工信息，需上传员工合同、社保证明，如无需上传说明；
5. 项目申报承诺书。

# 办理流程

1. **线上申报**：房租补贴通过雄安新区产业互联网平台 ([http://www.xaiip.org.cn](http://www.xaiip.org.cn/)) 进行线上申报。路径为“首页—企业公共服务-惠企政策一雄安新区第二批在孵企业办公用房租金补贴”，各企业使用科创服务平台原账号密码登录即可。
2. **审核推荐**：各孵化器对在孵企业的申报材料及承诺书进行审核，信息核实无误后在平台上传核查说明推送至雄安新区改发局账号。

# 申报时间

- 在孵企业申报开始时间：2022年5月16日
- 所在孵化器审核推荐截止时间：2022年6月10日

# 注意事项

1. 在孵企业须如实填报申请材料。
2. 所在孵化器通过管理员账号登录平台，对申报材料严格把关，填写补贴核查表，对不符合申报条件的予以退回。
3. 雄安新区改革发展局对以下情况不予奖补：
   - 申请日期不在补贴周期内；
   - 注册地址不在孵化园区内；
   - 合同未加盖公章、无有效签字、无有效日期。
4. 对弄虚作假、伪造材料等情况不予奖补，并取消其相关条款的申报资格。
