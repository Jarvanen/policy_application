# 政策简介

为创新技术转移人才发展机制，以培育实战型、专业化、应用型技术经理人为核心目标，精准提升技术经理人职业素养与服务能力。同时为了营造科技成果转移转化良好生态，提升科技成果转移转化质量和效益，助力新区经济高质量发展，拟于2026年5月12-13日举办2026年雄安新区中级技术经理人能力提升研修活动。

# 注意事项

1. **名额限制**：本次培训人数限定70人，报满为止。
2. **报名限制**：参训人员一经报名，无特殊事由不得更换人员。
3. **考核要求**：
   - 线上结业考试满分为100分，60分及以上为合格。
   - 案例分析报告满分为100分，60分及以上为合格。
   - 案例分析报告和结业考试均合格，方可取得中级技术经理人结业证书。
   - 如检测到案例分析报告为人工智能生产，记为不合格。
4. **证书效力**：证书可作为技术任职资格和工程系列技术经纪专业职称评审的依据。

# 申请对象

参加过国家技术转移人才培养基地组织的相关学习，并取得初级技术经理人结业证书的技术转移从业者。具体包括：
1. 高校、科研院所研发人员与成果转化人员；
2. 科技型企业负责人、市场经理、产品经理、技术转移部门负责人；
3. 科技园区、孵化器、众创空间管理高层和技术经纪管理人员；
4. 政府科技管理和技术转移职能部门负责人；
5. 技术转移服务机构、投资机构从业人员；
6. 为技术转移提供法律、商务、知识产权、需求分析、成果评价等服务的人员；
7. 河北省内各地市有志于推动京津科技成果在河北转化落地、增强技术转移服务能力的人员。

# 申报条件

1. 必须参加过国家技术转移人才培养基地组织的相关学习。
2. 必须取得初级技术经理人结业证书。

# 申报材料

1. **线上结业考试**：培训结束后参加闭卷考试（考试时间90分钟）。
2. **案例分析报告**：学员需提交一份案例分析报告（可以是学员自己的实际案例，也可以是研修活动授课老师课件中的案例），需对案例进行深入分析，总结案例的优缺点，提出改进措施和建议。

# 办理流程

1. **报名**：通过“河北省技术经纪人培训学习平台”微信小程序或APP进行报名。
2. **研修**：参加现场研修和线上学习。
3. **考核**：
   - 参加线上结业考试。
   - 提交案例分析报告至主办方邮箱：hbsjsrcpyjd1@163.com。
4. **发证**：案例分析报告和结业考试均合格者，取得基地依托单位河北省科技成果转化促进中心（科技大厦）盖章的中级技术经理人结业证书，电子版证书生成后自行下载。

2. **培训时间**：2026年5月12日-2026年5月13日。
