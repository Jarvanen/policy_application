# 政策简介

本通知旨在组织开展2023年度高新技术企业申报认定工作。依据河北省高新技术企业认定管理工作领导小组办公室要求，由河北雄安新区改革发展局组织辖域内企业开展申报，并对申报企业的资质、证明材料、告知承诺书等条件进行严格审查，核实纸质材料与国家高新技术企业认定管理工作网中信息一致后，将符合条件的企业推荐至河北省认定管理机构。

# 申请对象

1. 辖域内企业。
2. 2023年需复审的高企（名单见附件）。

# 申报条件

1. 申报企业需具备河北省科技型中小企业资质，或已在平台申报相关资质。

# 申报材料

1. 电子版申请：需在国家“高新技术企业认定管理工作网”（http://www.innocom.gov.cn/）提交。
2. 纸质版材料：需逐项打印、装订成册，交三县科技管理部门。
3. 一致性要求：电子版材料需与纸质版一致。

# 办理流程

1. 企业申报：企业在国家“高新技术企业认定管理工作网”提交电子版申请，并将纸质版材料交至三县科技管理部门。
2. 部门审查：三县科技管理部门对需复审高企通知到位；雄安新区改革发展局组织专家对申报企业的资质、证明材料、告知承诺书等条件进行严格审查，并核实纸质材料与国家高新技术企业认定管理工作网中信息一致。
3. 推荐上报：将符合条件的企业推荐至河北省认定管理机构。

# 申报时间

1. 第一批高企申报时间截止至2023年5月15日。
2. 全年申报时间截止至2023年9月15日。
3. 其余批次根据科技厅部署另行通知。

# 注意事项

1. 审查严格：雄安新区改革发展局将组织专家对申报企业的资质、证明材料、告知承诺书等条件进行严格审查。
2. 材料一致：需核实纸质材料与国家高新技术企业认定管理工作网中信息一致。
