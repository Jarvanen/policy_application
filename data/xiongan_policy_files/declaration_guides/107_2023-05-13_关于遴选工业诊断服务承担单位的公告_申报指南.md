# 政策简介

为贯彻落实《雄安新区工业诊断力量培育实施方案（2023-2025年）》文件精神，做好雄安新区企业工业诊断服务工作，以政府购买服务的方式，组建雄安新区工业巡诊队伍，引进培育一批具备为工业企业开展科技创新、绿色低碳、制造提升、品牌设计、质量管理等咨询服务，提供解决方案的机构，开展工业诊断服务。

# 奖励额度/补贴标准

本次拟遴选5家服务机构（制造提升1家、科技创新1家、质量管理1家、绿色低碳1家、品牌设计1家），每家机构工作经费不超过5万元人民币。

# 注意事项

1. **材料格式要求**：
   - 书面材料一式五份，含电子光盘一份。
   - 材料需密封。
   - 申报书需使用A4纸标准大小，正文字体大小小四号，页码字体大小小四号。
   - 封面需盖章、骑缝章，申报书脊背标注申报单位名称。
   - 场所照片角度合理、清晰。
   - 真实性声明需签字盖章。
   - 以上证明材料提供复印件并加盖单位公章。
2. **禁止参与评审情形**：规范性自查表不符合要求的，将禁止参与评审工作。
3. **负面清单**：违反国家相关法律和政策规定，在经营管理、企业诚信等方面有不良记录的和发生安全生产责任事故的单位不得申报。
4. **服务要求**：工业诊断服务机构应聚焦2023年新区传统产业转型升级重点工作，主动自行对接有需求的企业开展诊断咨询服务，并向我局进行备案，及时更新服务动态进展；在实施咨询服务中要遵守规范流程，投入足够工作时间，确保诊断服务全程高质高效；在提供服务过程中应接受新区及三县工信部门的监督和管理。
5. **交付成果**：不少于30份工业企业诊断、咨询服务报告及相关佐证材料。
6. **服务周期**：自合同签订之日起一年。

# 申请对象

具有独立法人资格，且具备提供科技创新、绿色低碳、制造提升、品牌设计、质量管理等咨询服务，提供解决方案的机构。

# 申报条件

1. **运营情况要求**：工业诊断服务机构运营和财务状况良好，有专业的业务和管理团队，有明确的服务宗旨、管理制度、激励机制等。
2. **业务能力及业绩要求**：
   - 具备提供制造提升、科技创新、质量管理、绿色低碳、品牌设计等方面的专业能力，拥有可行的诊断咨询体系和方法。
   - 成长性良好，在相关领域行业具有一定的影响力。
   - 至少具有2个及以上成功项目服务和实施案例。
3. **人员配备要求**：
   - 配备稳定的诊断咨询业务技术和管理团队，服务专业人员不少于3人，技术服务能力强，沟通能力好。
   - 长期合作的副高级以上或同等水平的专家应在10人以上。
   - 稳定合作的相关专家30人以上。
   - 签有合作协议的专业化、外联型合作机构10个（含）以上。
4. **评分加分项（参考评审表）**：
   - 提供银行开具资信证明或财务审计报告。
   - 提供信用中国信用报告。
   - 提供近三年与国企、央企、政府机关等同类（需与服务方向一致）交易合同。
   - 具备本地化服务能力（包括服务新区工业企业经验、在新区设有或承诺设立分公司、子公司、机构等）。

# 申报材料

1. 单位基本情况（原件盖章）；
2. 营业执照副本或登记证书复印件；
3. 能证明本单位实力或能胜任本项工作内容的相关证明材料；
4. 法定代表人证明书或法人授权委托书（原件）；
5. 工作方案，包括总体思路、工作开展方式、时间进度、管理规章制度及考核办法、经费使用方案、服务质量承诺等；
6. 雄安新区工业诊断服务机构申报书（见附件1，需包含基本情况、主要产品或服务情况、企业技术实力、典型案例情况、工作方案、相关证明材料、申报材料真实性声明、自查表等内容）。

# 办理流程

1. **提交材料**：各相关机构和企业于规定时间内将书面材料密封后交至指定地址。
2. **评审遴选**：我局将组成内部评标工作组择优遴选本项目承担单位。
3. **签订合同**：与遴选出的承担单位协商签订委托合同。

# 申报时间

2023年5月15日至2023年5月19日（5个工作日）。
请各相关机构和企业于5月19日下午17:00前提交材料。
