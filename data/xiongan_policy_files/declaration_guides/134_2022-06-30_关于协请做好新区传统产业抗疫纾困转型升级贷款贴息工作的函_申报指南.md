# 政策简介

本政策旨在落实新区党政办《关于支持传统产业抗疫纾困转型升级的若干措施》(雄安办字〔2022〕13号)要求，通过实施《关于支持雄安新区传统产业抗疫纾困转型升级贷款贴息实施细则》，支持新区传统产业抗疫纾困转型升级，保障政策落到实处，并与后续一系列支持新区经济稳运行工作做好衔接。

# 注意事项

1. **资金发放形式**：本次贴息资金拟以数字人民币形式发放至企业对公数字钱包。
2. **账户准备**：三县政府需协调县财政部门做好财政对公数字钱包开立准备工作，并在贴息资金发放过程中，根据实际情况，与新区财政部门做好沟通对接工作。
3. **申报平台**：需申请新区产业互联网平台账号，通过该平台开展企业贴息申请组织宣传、审核、审计及汇总报送等工作。

# 申请对象

新区传统产业企业（需符合《关于支持雄安新区传统产业抗疫纾困转型升级贷款贴息实施细则》要求）。

# 申报材料

由新区各银行金融机构按规定向贷款贴息申请企业提供以下材料：
1. 贷款贴息明细表
2. 资金到账证明（银行进账单）
3. 还款记录
4. 银行贷款付息证明（利息单）

# 办理流程

1. **组织宣传与申请**：三县政府指定牵头部门开展企业贴息申请组织宣传工作；各银行机构积极向贷款客户宣讲政策，并引导符合条件的企业客户按规定申请。
2. **材料提供**：各银行机构按规定向企业提供相关申请材料。
3. **审核与汇总**：三县政府指定牵头部门按照实施细则要求，开展企业贴息申请的审核、审计及汇总报送工作。
4. **系统支持**：雄安新区智能创新联合会在新区产业互联网平台开发贴息申请接口，保障申报系统流程合理有序。
5. **资金拨付**：通过数字人民币形式发放至企业对公数字钱包。
