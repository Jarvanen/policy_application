# 政策简介

为推进国家级、省级专业技术人员继续教育基地建设，构建分层分类的继续教育公共服务体系和教育体系，河北雄安新区开展国家级、省级专业技术人员继续教育基地申报工作。国家级基地主要从承担河北省专业技术人才知识更新工程重要任务、相关重点领域急需紧缺人才培养培训能力业绩突出的高等院校、科研院所，地方、部门或行业教育培训机构，专业协会（学会）及大型企业培训机构中遴选。省级基地主要从河北省具有实践基础的高等院校、科研院所，各市、部门或行业教育培训机构，专业协会（学会）及大中型企业培训机构中遴选。

# 奖励额度/补贴标准

经省人力资源社会保障厅命名并授牌的省级专业技术人员继续教育基地，每个基地由省财政资助经费50万元，主要用于开展专家授课、师资培训、教材课件开发、数据库开发、课题研究等工作。

# 注意事项

1. 坚持公平、公正、公开原则，优化布局、突出特色，坚持资源共享、注重实效。
2. 国家级、省级专业技术人员继续教育基地将优先从区域性、行业性专业技术人员继续教育基地中推荐遴选，并重点考察备选单位承担专业技术人员培养培训任务的情况。
3. 申报材料需同时报送纸质版和电子版。

# 申请对象

**国家级专业技术人员继续教育基地申报对象：**
在河北省境内依法登记的具有培训或继续教育相关资质的独立法人机构，主要包括：
1. 承担我省专业技术人才知识更新工程重要任务、相关重点领域急需紧缺人才培养培训能力业绩突出的高等院校、科研院所；
2. 地方、部门或行业教育培训机构；
3. 专业协会（学会）；
4. 大型企业培训机构。

**省级专业技术人员继续教育基地申报对象：**
1. 我省具有实践基础的高等院校、科研院所；
2. 各市、部门或行业教育培训机构；
3. 专业协会（学会）；
4. 大中型企业培训机构。

# 申报条件

**一、国家级专业技术人员继续教育基地申报条件**

1. 在河北省境内依法登记的具有培训或继续教育相关资质的独立法人机构。
2. 具有能够满足培训需要的相对稳定的高素质专兼职教师队伍，有健全的继续教育管理机构及从事继续教育管理的专兼职人员。
3. 具有健全的教学组织管理、学员考核管理、培训登记管理、培训效果评估、后勤保障及财务管理等基本管理制度。
4. 具备能够满足线下、线上培训所需的教学设备和基础设施，有固定的教学培训场所、设备等硬件设施和现代化远程教育条件，建设有网络培训和管理信息化平台。
5. 具有丰富的培养培训专业技术人员经验，围绕经济社会发展重点领域，每年培训专业技术人员总量不少于2000人次。

**二、省级专业技术人员继续教育基地申报条件**

1. 有依法经核准登记或批准成立的具有培训资质的施教机构，有健全的继续教育培训管理机构、完善的管理制度和专职管理人员。
2. 有与所承担的培训任务相适应的相对稳定的专（兼）职教师队伍。专（兼）职师资应当具有较高理论水平和实践经验，在本专业领域具有较高影响力和公认度。
3. 有承担中高级专业技术人才培训任务的场所、教学设备、专业图书资料，具备实施现代化远程教育的基本条件及相应的硬件设施，能开展在线培训和互动交流。
4. 每年至少能承担1000名以上高层次、急需紧缺和骨干专业技术人才培养培训任务。
5. 能为基地基本建设和日常工作提供配套经费保障。

# 申报材料

**一、国家级专业技术人员继续教育基地申报材料**

1. 《国家级专业技术人员继续教育基地申报表》一式三份；
2. 县级人社部门或行业主管部门出具的推荐函；

*注：属于地区或部门、行业设立的区域性、行业性专业技术人员继续教育基地的，应一并报送有关证明材料。*

**二、省级专业技术人员继续教育基地申报材料**

1. 《河北省专业技术人员继续教育基地申请表》一式三份；
2. 其他证明材料或补充材料一式三份。

# 办理流程

**一、国家级专业技术人员继续教育基地认定程序**

1. **预审：** 申报机构自愿填写《国家级专业技术人员继续教育基地申报表》，提供相关材料，报市级人社部门或行业主管部门预审。
2. **审核推荐：** 省人力资源社会保障厅根据知识更新工程的需要，组织专家对申报机构材料进行审核，择优推荐至人力资源社会保障部。
3. **认定公布：** 人力资源社会保障部组织继续教育和相关行业领域专家组成专家评审委员会，对申报材料进行复核、论证、考察、评审，经审核同意后予以认定并向社会公布。

**二、省级专业技术人员继续教育基地认定程序**

1. **提交材料：** 填写《河北省专业技术人员继续教育基地申请表》，报送党群工作部。
2. **评选认定：** 省人力资源社会保障厅组织继续教育和相关领域专家组成专家评审委员会，对申报材料进行审核、论证、考察和评审。
3. **网站公示：** 经省人力资源社会保障厅党组会议研究同意后，在省人力资源社会保障厅网站公示省级专业技术人员继续教育基地名单。
4. **命名授牌：** 经公示无异议，由省人力资源社会保障厅命名并授牌。

# 申报时间

各单位请于2022年5月16日前将纸质申报材料报送至党群工作部，同时须将电子版《国家级专业技术人员继续教育基地申报表》或《河北省专业技术人员继续教育基地申请表》发送至电子邮箱。
