# 政策简介

为贯彻落实《雄安新区工业诊断力量培育实施方案（2023-2025年）》《关于支持工业诊断力量培育的五条措施》，规范雄安新区“工业诊所”创建工作，加快提升工业诊断服务能力和质效，助推新区传统产业高质量发展。

“工业诊所”旨在通过“问诊、巡诊、接诊”业务，出具“体检报告”“诊疗方案”，为工业企业提供科技创新、绿色低碳、制造提升、品牌设计、质量管理、投资营销、政策宣贯等服务，解决企业转型提升和产业链条提升中的问题。

# 申请对象

1. 具有独立法人资格且在雄安新区注册或建有分支机构的单位。
2. 包括有关协会、大专院校、研究机构、产业（技术）研究院、设计创新中心、第三方服务机构或其他适合的单位。
3. 在雄安新区注册且已被认定为省级工业诊所的机构（实行“免申即享”，自动认定为雄安新区工业诊所）。

# 申报条件

1. 具备相关专业咨询服务能力，自有副高级以上或同等水平的常驻专家5人以上，稳定合作的相关专家20人以上，签有合作协议的专业化、外联行合作机构5个（含）以上。
2. 具备办公用房等服务场所和办公条件，能够组建专门运营团队，有明确的服务宗旨、管理制度、激励机制等。
3. 申报单位近一年来累计为20家工业企业开具诊断报告。

# 申报材料

根据《雄安新区“工业诊所”申报书（大纲）》，需准备以下材料（以上证明材料提供复印件并加盖单位公章）：

1. **基本情况**：现有基础（公司简介、行业地位、人员状况、专家水平、重大成绩和荣誉等）；经营情况（年产值、销售收入、利税、员工基本情况、主要业务开展情况、服务企业能力、自身特色优势、业务成绩等）。
2. **主要产品或服务情况**：提供的主要产品或服务基本情况，产品或技术主要核心优势，主要技术指标，服务范围、行业、场景，服务工业企业的主要成效（服务企业数量、典型行业服务效果等），获得相关试点示范和资金奖励情况。
3. **企业技术实力**：
    - 研发能力：企业自主研发能力介绍，取得的相关自主知识产权情况（如专利、软件著作权等）和取得的相关认证情况。
    - 队伍能力：企业技术人员和技术服务人员情况，负责人及核心团队主要成员履历，企业技术或服务团队情况（包括团队人数、学历情况等）。
    - 盈利能力：商业模式和盈利能力介绍，企业目前经营状况，未来3-5年发展规划，对行业发展的推动作用或社会效益分析。
4. **典型案例情况**：提供10个服务工业企业的成功案例，每个案例不超过2000字，重点说明帮助企业解决发展难题和服务成效。
5. **下一步工作安排**：创建目标及任务（所要达到的服务水平和能力，本年度及未来几年计划服务企业情况等）；服务内容（明确主要服务方向）。
6. **相关证明材料**：
    - 企业法人营业执照/组织机构代码证；
    - 自主知识产权证明材料，相关认证证书；
    - 主要专家/技术团队成员履历、资质相关证明材料，可证明主要专家/技术团队成员与单位雇佣或合作关系的材料；
    - 服务客户证明材料（如合同、用户报告或反馈意见等，不少于20个）；
    - 主要合作伙伴合作关系证明材料；
    - 获得荣誉证书等证明材料；
    - 其他服务能力证明材料。

# 办理流程

1. **公开征集**：雄安新区改革发展局发布“工业诊所”申报通知，公开征集有意愿参与“工业诊所”建设的机构单位，指导编制创建实施方案，明确创建思路、目标、重点及保障措施。
2. **申报评审**：三县工信部门组织初审筛选、实地考察提出推荐意见，并上报创建实施方案。雄安新区改革发展局组织专家评审创建实施方案，视情进行实地考察。
3. **公告认定**：雄安新区改革发展局将评审通过的创建单位，在网站上予以公示，公示期为5个工作日。公示无异议后进行批复，向社会公告“工业诊所”名单。
4. **评价考核**：按照“谁申报谁负责”原则，三县工信部门负责本地“工业诊所”自评价和年度考核工作，并于每年1月底前上报上年度绩效考核结果和推进实施情况。工业诊所每年须完成至少20项“体检（诊断）报告”，包含企业基本情况、体检存在的问题、诊断建议及解决方案、企业意愿及采纳情况、跟踪服务情况等内容，要求有出诊照片、专家签字、企业盖章认可。
5. **动态管理**：实行“择优入选、定期评估、不合格淘汰”的动态管理机制，雄安新区改革发展局根据考核结果和抽查情况，组织专家评级评定，对每年发布的名单进行动态调整。

# 注意事项

1. **免申即享**：在雄安新区注册且已被认定为省级工业诊所的机构自动认定为雄安新区工业诊所。
2. **考核要求**：工业诊所每年须完成至少20项“体检（诊断）报告”，报告需包含企业基本情况、体检存在的问题、诊断建议及解决方案、企业意愿及采纳情况、跟踪服务情况等内容，并要求有出诊照片、专家签字、企业盖章认可。
3. **动态调整**：实行“择优入选、定期评估、不合格淘汰”的动态管理机制，对每年发布的名单进行动态调整。
