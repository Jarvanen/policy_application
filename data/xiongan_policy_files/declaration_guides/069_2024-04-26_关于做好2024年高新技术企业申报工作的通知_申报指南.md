# 政策简介

本通知旨在组织开展2024年度高新技术企业申报认定工作。依据河北省高新技术企业认定管理工作领导小组办公室要求，由河北雄安新区改革发展局部署，积极组织辖域内企业开展高企申报认定。

# 注意事项

1. **严格审查与一致性要求**：雄安新区改革发展局将组织专家对申报企业的资质、证明材料、告知承诺书等条件进行严格审查，并核实纸质材料与国家高新技术企业认定管理工作网中信息一致，将符合条件的企业推荐至河北省认定管理机构。
2. **审计报告赋码要求**：
   - 自本年度起，申报材料提供的由会计师事务所出具的专项审计报告或鉴证报告，应当按照有关规定上传注册会计师行业统一监管平台（网址http://acc.mof.gov.cn）并赋码。
   - 经行政登记设立并加入中国注册税务师协会的税务师事务所出具的专项审计报告或鉴证报告，报告首页须带有经中税协信息服务平台（http://www.ecctaa.com）备案的二维码。
3. **复审通知**：三县科技管理部门需对2024年需复审高企通知到位，名单见附件。

# 申请对象

1. 雄安新区辖域内企业。
2. 2024年需重新认定的高新技术企业（具体名单见附件）。

# 申报条件

1. 申报企业需具备国家级、河北省科技型中小企业资质，或已在平台申报相关资质。

# 申报材料

1. **电子版材料**：在国家“高新技术企业认定管理工作网”（http://www.innocom.gov.cn/）提交。
2. **纸质版材料**：按照申报材料内容要求逐项打印、装订成册，交三县科技管理部门。
3. **一致性要求**：电子版材料需与纸质版一致。

# 办理流程

1. **网上申报**：在国家“高新技术企业认定管理工作网”提交电子版申请。
2. **纸质提交**：将装订成册的纸质版材料交至三县科技管理部门。
3. **审查推荐**：雄安新区改革发展局组织专家审查资质、证明材料及告知承诺书，核实纸质与网络信息一致性。
4. **上报**：将符合条件的企业推荐至河北省认定管理机构。

# 申报时间

1. **第一批截止**：2024年6月3日。
2. **全年截止**：2024年9月20日。
3. **逾期处理**：逾期不再受理。
4. **其余批次**：根据雄安新区改革发展局部署另行通知。
