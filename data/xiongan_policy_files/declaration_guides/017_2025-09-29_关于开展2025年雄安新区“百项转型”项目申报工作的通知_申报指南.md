# 政策简介

为落实《雄安新区支持高端高新制造业发展的若干措施》，切实发挥资金引领作用，支持一批重点转型升级项目建设，提升产业链现代化水平，引领新区制造业向高端化、智能化、绿色化发展。本政策旨在大力推进新型工业化，扎实推进制造业技术改造升级工作，支持领域包括技术改造项目、数字化转型项目、公共服务平台项目。

# 奖励额度/补贴标准

1. **技术改造项目**：对拟支持技术改造项目按已完成的固定资产投资额（包括但不限于购置生产设备、检测仪器、研发工器具、配套软硬件系统等，不含土地、建筑物投资），采取后补助方式给予不超过20%资金支持，单个项目最高不超过300万元。
2. **数字化转型项目**：对拟支持数字化转型项目按照信息化软硬件购置、系统开发与服务等费用，采取后补助方式给予不超过30%资金支持，单个项目最高不超过300万元。
3. **公共服务平台项目**：对拟支持的公共服务平台项目按照信息化软硬件购置、系统开发与服务等费用，采取后补助方式给予20%资金支持，每个平台奖补金额最高不超过300万元。

# 注意事项

1. **重复支持限制**：
   - 曾获得国家、省级财政支持的项目，可继续申报“百项转型”项目，按“百项转型”项目奖励标准补足差额。
   - 已经获得雄安新区财政支持的项目，原则上不得再申报“百项转型”项目。
   - 承诺此次项目申报过程中提供的所有财务票据均为首次作为申报新区级资金凭证依据，且所有票据未获得过国家、省、新区各类资金补助或奖励支持。
2. **材料格式要求**：
   - 申报材料用 A4 纸双面打印，按顺序装订，左侧胶装成册，书脊处注明2025年“百项转型”项目、企业名称、项目名称。
   - 全套申报材料提供电子文档（包括Word和PDF格式）。
3. **责任与监管**：
   - 申报单位对申报材料的真实性、合规性负责。
   - 以虚报、冒领、伪造、关联交易等手段套取财政资金的行为，一经查实，将收回已拨付资金，取消该单位后续申报资格，并依法依规将相关责任单位和责任人纳入失信名单，严肃追究责任。
4. **其他**：
   - 进口设备需提供海关税务发票及相关资料。

# 申请对象

在新区注册登记、具有独立法人资格的企业或其他机构。

# 申报条件

1. **基本要求**：具备较强的技术开发、资金筹措、项目实施能力，在经营管理、社会信用、安全生产、节能环保等方面无不良记录。
2. **建设要求**：项目实施地在雄安新区，应为在建项目，已办理完成备案（核准）及土地、环评等手续，相关审批程序完备，并已开工建设。
3. **投资要求**：项目计划总投资500万元以上，并纳入统计部门工业技改项目管理。
4. **技术要求**：项目改造升级后不属于《产业结构调整指导目录（2024年本）》中限制类、淘汰类目录。优先支持符合《工业重点行业领域设备更新和技术改造指南》（工信厅规〔2024〕33号）要求的技术改造项目。

# 申报材料

1. “百项转型”项目申请书；
2. “百项转型”项目申请报告（根据项目类型选择：技术改造项目申请报告、数字化转型项目申请报告、公共服务平台项目申请报告）；
3. 项目核准或备案证明（复印件）；
4. 项目实施单位同意税务机关提供设备、仪表仪器、软件产品、专利技术及设备改造等固定资产投资发票等相关资料的授权书；
5. 项目专项审计报告（复印件，加盖申请单位公章）；
6. 项目淘汰或计划淘汰设备清单；
7. 项目已购置或计划购置设备清单；
8. 项目固定资产（设备）投资完成明细表；
9. 项目设备购置合同及技术参数说明，设备支付款的银行付款凭证及发票（复印件）；
10. “百项转型”项目汇总表；
11. 其他需要提供的相关材料。

# 办理流程

1. **项目申报**：新区工信科技数据局在平台发布通知，各申报单位登录“雄安新区产业互联网平台”（www.xaip.org.cn）选择“百项转型”栏目填报申报材料。
2. **县级评审**：三县工信科技数据局组织开展项目完整性和真实性审查，实地考察项目进展，核查审批手续原件及财务佐证，征求主管部门意见。委托第三方机构进行专家评审和财务现场审核，提出评审意见、财务审核意见及资金支持建议，经县局党组会审议后报送。
3. **复核及公示**：新区工信科技数据局委托第三方机构对申报材料进行复核，并将复核意见反馈至三县工信科技数据局。三县工信科技数据局对复核通过的项目，在平台进行公示。
4. **资金拨付**：经新区财政部门审核后，新区工信科技数据局按程序将资金拨付至三县财政部门，由三县财政部门会同工信部门将资金拨付至申报单位。

# 申报时间

1. 三县工信科技数据局于 **10月13日前** 完成项目评审，并将申报材料和审议意见报新区工信科技数据局。
2. 三县工信科技数据局于 **10月23日前** 将资金申请报告、公示情况、资金汇总表报新区工信科技数据局。
