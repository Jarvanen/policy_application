# 政策简介

本政策旨在支持海内外高层次人才来雄安新区创新创业。主要支持方向包括启动支持、融资支持、场地支持、人才支持、创新支持及奖励支持。核心认定类别为“雄创计划”入选企业，以及参加“雄才杯”创业大赛或其他省级以上创业大赛获奖并落地雄安的创业项目。政策涵盖启动资金、股权投资、租金补贴、生活津贴、研发配套、首台（套）奖励、应用示范补贴以及上市奖励等多个方面。

# 奖励额度/补贴标准

1. **启动资金与奖励**：纳入“雄创计划”的企业，根据评审结果给予一定金额的启动资金；参加“雄才杯”创业大赛获奖一年内或参加其他省级以上创业大赛获奖两年内，或其他优秀创业项目，经评审通过并落地雄安后，根据评审结果给予奖励。
2. **场地支持**：
   - 雄创企业入驻指定园区或政府性空间，给予租金支持；入驻无装修环境办公空间的，给予装修补贴。
   - 经评审落地雄安但还未入选“雄创计划”的企业，入驻指定园区或政府性空间，给予租金支持。
3. **人才支持**：
   - 雄创企业引进的人才给予生活津贴。
   - 雄创企业高级管理人员和业务骨干，按其上一年度对新区综合贡献的予以奖励。
4. **创新支持**：
   - 成功申报国家级重大研发项目或省级以上创新平台后，新区按照实际获得专项经费比例配套资金支持。
   - 对通过省级以上认定的首台（套）重大技术设备项目，给予奖励；对获得工业和信息化部或河北省补贴的项目，按照实际投保年度保费比例给予补贴。
   - 对于购买示范应用产品、技术、服务的新区企业，根据实际合同额给予补贴。
5. **上市奖励**：
   - 申请在境内A股上市的企业，进入辅导期并在河北证监局完成备案后，在省级政策补助基础上，新区额外给予补助。
   - 向中国证监会或沪、深、京交易所提交上市申请并获受理后，在省级政策补助基础上，新区额外给予补助。
   - 申请在境外主要证券交易所上市的企业，经具有批准权限的部门或机构正式受理后，在省级政策补助基础上，新区额外给予补助；通过审核后，在省级政策补助基础上，新区额外给予补助。

# 注意事项

1. **资金拨付方式**：启动资金及创业大赛获奖奖励均分期拨付。
2. **优先推荐**：对落地转化效果好的创业项目可优先推荐入选“雄创计划”。
3. **人员资质要求**：经评审落地雄安但还未入选“雄创计划”的企业，其核心团队成员应具有本科及以上学历，且需达到一定人数方可获得生活保障。
4. **平台认定范围**：省级以上创新平台包括省级学科重点实验室、省级企业重点实验室、省级技术创新中心、省级产业技术研究院、省级企业技术中心、省级工程研究中心、省级工业设计中心、省级中小企业公共服务平台。

# 申请对象

1. 在新区注册的人才团队企业。
2. 参加“雄才杯”创业大赛获奖一年内或参加其他省级以上创业大赛获奖两年内的项目团队。
3. 其他优秀创业项目团队。
4. 入选“雄创计划”的企业（简称“雄创企业”）。
5. 经评审落地雄安但还未入选“雄创计划”的企业。
6. 新区企业（针对创新支持中的应用示范补贴）。

# 申报条件

1. **启动支持条件**：
   - 人才团队在新区注册企业，一定期限内参加“雄创计划”项目评审并通过。
   - 参加“雄才杯”创业大赛获奖一年内，或参加其他省级以上创业大赛获奖两年内，经评审通过并落地雄安。
   - 其他优秀创业项目，经评审通过并落地雄安。
2. **场地支持条件**：
   - 雄创企业入驻科学园管委会所管理园区、科创中心中试基地、雄安人力资源服务产业园或其他政府性空间。
   - 经评审落地雄安但还未入选“雄创计划”的企业，入驻科学园管委会所管理园区、雄安人力资源服务产业园或其他政府性空间。
3. **创新支持条件**：
   - 成功申报国家级重大研发项目或省级以上创新平台。
   - 开展首台（套）重大技术设备的推广和使用，通过省级以上认定；或获得工业和信息化部或河北省补贴。
   - 购买经定期征集、组织行业专家结合新区智能城市建设研究开发的应用示范新产品、新技术、新服务。
4. **上市奖励条件**：
   - 申请在境内A股上市，进入辅导期并在河北证监局完成备案。
   - 向中国证监会或沪、深、京交易所提交上市申请并获受理。
   - 申请在境外主要证券交易所上市，经具有批准权限的部门或机构正式受理；或通过审核。

# 办理流程

1. **项目评审**：人才团队参加“雄创计划”项目评审，或创业项目经评审通过。
2. **落地注册**：评审通过的项目落地雄安，人才团队在新区注册企业。
3. **入驻与认定**：企业入驻指定园区或空间；成功申报国家级/省级项目或平台；通过首台（套）认定或获得上级补贴；完成上市辅导备案或提交上市申请。
4. **资金拨付**：根据评审结果或认定情况，分期拨付启动资金、奖励或补助。

# 申报时间

1. 人才团队在新区注册企业后，在一定期限内可参加“雄创计划”项目评审。
2. 参加“雄才杯”创业大赛获奖一年内，或参加其他省级以上创业大赛获奖两年内。
3. 定期征集新产品、新技术、新服务。
