# 政策简介

科创主体数量是评估地区科技创新能力的重要指标，为壮大雄安新区创新主体，集聚培育一批高新技术企业、龙头骨干企业和高成长性科技企业，提升企业的科研创新能力，引导雄安新区科技型企业持续加大研发投入，进一步推动科技创新支撑引领高质量发展，河北雄安新区管委会改革发展局拟采取比选方式委托机构在科创主体孵化育成、项目申报、平台建设、科技奖励等方面提供指导培训和咨询服务。

# 奖励额度/补贴标准

本项目比选最高限价为：350000元。预付款100000元，服务结束后，按照服务企业数量、研发费用加计扣除金额、企业满意度等方面进行绩效评估，按照评估结果支付尾款。

# 注意事项

1. 本次比选不接受联合体申请。
2. 申请人的项目报价不得超出比选最高限价，且不得以低于成本的报价竞标，最低报价不是被授予合同的唯一保证。
3. 报价含税，中选人需提供增值税普通发票。
4. 申请人应在递交截止时间前完成提交，逾期未提交的，视为放弃参与。
5. 比选响应文件需一式五份（统一密封加盖公章），电子扫描件（加盖单位公章）上传至指定邮箱。
6. 采购人仅通知选中的服务商，其他不再单独通知。

# 申请对象

具有有效的营业执照，且符合以下专业资格要求的单位：
1. 具有高新技术企业、科技型中小企业申报、培训、咨询等相关业务的行业资质和项目经验；
2. 具有从事科创企业服务的各领域技术专家和财务专家，预审核专家组组长需具有高级及以上职称。

# 申报条件

申请人需具有有效的营业执照，且不得存在下列任何情形：
1. 与采购人有控制或持股关联；
2. 申请人单位负责人与本比选项目的其他申请人的单位负责人为同一人；
3. 与本比选项目的其他申请人存在控股、管理关系；
4. 被依法暂停或者取消在雄安新区的投标资格；
5. 被责令停业，暂扣或者吊销执照或许可证，或吊销资质证书；
6. 进入清算程序，或被宣告破产，或其他丧失履约能力的情形；
7. 被工商行政管理机关在全国企业信用信息公示系统中列入严重违法失信企业名单；
8. 被最高人民法院在“信用中国”网站（www.creditchina.gov.cn）中列入失信被执行人名单；
9. 在近三年（自比选公告发布之日起倒算3年）内申请人或其法定代表人或其负责人、拟委任的项目负责人有行贿犯罪行为的；
10. 法律法规规定的其他情形。

# 申报材料

1. 比选响应文件（一式五份，统一密封加盖公章）；
2. 电子扫描件（加盖单位公章）；
3. 申请文件内容需包含：
   - 服务机构简介（注册时间、地点、规模、主营业务等）；
   - 基础条件（专兼职人员情况，与各领域专家、财会机构签订协议和开展工作情况，对河北省、雄安新区科创主体培育相关政策文件掌握情况等）；
   - 工作成效（近三年科创主体培育指导服务成效，对当地企业发展的推动作用等，提供典型案例）；
   - 服务内容（针对科创主体培育可提供的服务内容、开展的其他各类服务内容等，服务企业相关工作目标及方案）；
   - 佐证材料（营业执照、专兼职人员从事企业服务的证明如学历证明、授课、主持培训等）；
   - 服务报价表。

# 办理流程

1. 准备比选响应文件及申请文件，加盖公章；
2. 将比选响应文件一式五份（统一密封加盖公章）报送至指定地点；
3. 将电子扫描件（加盖单位公章）上传至指定邮箱；
4. 采购人进行初步评审，对满足比选文件实质性要求的代理申请书进行打分；
5. 按照得分由高到低进行排序，确定1名中选候选人；
6. 采购人通知选中的服务商。
