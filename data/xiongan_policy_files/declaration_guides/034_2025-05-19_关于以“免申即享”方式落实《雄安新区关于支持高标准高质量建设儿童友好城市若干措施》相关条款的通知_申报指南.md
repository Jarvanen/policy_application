# 政策简介

为广泛动员社会力量参与雄安新区儿童友好城市建设，贯彻落实《雄安新区关于支持高标准高质量建设儿童友好城市若干措施》（雄安儿童办发〔2024〕3 号），河北雄安新区改革发展局拟以“免申即享”方式落实相关条款奖励政策。支持方向包括：对运营单位是社会主体的雄安新区儿童友好实践基地给予奖补；支持社会主体及雄安新区企（事）业单位建设儿童友好科普基地；支持社会主体建设儿童之家或妇女儿童之家；支持社会主体组织开展儿童视力保障、牙齿保健等公益健康科普服务活动；为0-3岁婴幼儿提供托育服务的市场化运营托育场所给予奖补。

# 奖励额度/补贴标准

1. **儿童友好实践基地**：对运营单位是社会主体的雄安新区儿童友好实践基地，复核评分达到前30%的，对基地运营主体一次性给予2万元奖补资金。
2. **儿童友好科普基地**：
   - 认定为五星级省级科普示范基地的，一次性给予20万元运营补助；
   - 认定为四星级省级科普示范基地的，一次性给予15万元运营补助；
   - 认定为三星级省级科普示范基地的，一次性给予10万元运营补助；
   - 认定为国家科普教育基地、国家自然资源科普基地等各类国家级科普基地的，一次性给予50万元运营补助。
   - 科普基地实现晋级的，以“晋级补差”的方式给予补助；同时认定为五星级省级科普示范基地和国家级科普基地的，按照国家级科普基地的标准给予运营补助。
3. **儿童之家/妇女儿童之家**：
   - 被评为省级示范妇女儿童之家的，在获得省级专项建设资金的基础上，雄安新区给予1:1配套资金；
   - 被评为省级儿童之家示范点的，在获得省级奖励的基础上，雄安新区给予2万元奖补资金；
   - 被评为新区示范性儿童之家的，对基地运营主体给予1万元奖补资金。
4. **公益健康科普服务活动**：符合当年度“国家和河北省基层科普行动计划”要求的儿童视力保障、牙齿保健等公益健康科普服务活动，按照当年省级下达资金数额给予1:1比例的配套资金。
5. **托育场所**：为0-3岁婴幼儿提供托育服务，市场化运营，且符合《托育机构质量评估标准》（国卫通〔2023〕13号）中A、B两档要求的托育场所，经新区卫健局评估认定通过后：
   - A档：按照1600元/床的标准，一次性给予总额不超过160000元的奖补资金；
   - B档：按照1100元/床的标准，一次性给予总额不超过110000元的奖补资金。

# 注意事项

1. **适用主体限制**：托育场所奖补对象须属于《雄安新区关于支持高标准高质量建设儿童友好城市若干措施》中支持的雄安新区范围内注册的企业、民办非企业单位（组织）、基层群众性自治组织、其他社会团体。
2. **认定依据**：
   - 儿童友好实践基地以复核通知为准；
   - 省级科普示范基地以省科技厅发布认定通知为准；国家级科普基地以科技部发布认定通知为准；
   - 省级示范妇女儿童之家、省级儿童之家示范点以省妇联或新区发布认定通知为准；雄安新区示范性儿童之家以雄安新区发布认定通知为准；
   - 公益健康科普服务活动以雄安新区工信科技数据局发布的认定结果为准；
   - 托育场所以雄安新区卫生健康局发布的评估通知为准。
3. **兑现方式**：符合条件的单位需注册或登录雄安新区产业互联网平台，填写收款信息，由雄安新区改革发展局定期对符合条件的运营主体按照程序拨付相应奖励。

# 申请对象

1. 运营单位是社会主体的雄安新区儿童友好实践基地运营主体。
2. 社会主体及雄安新区企（事）业单位（儿童友好科普基地建设）。
3. 社会主体（儿童之家或妇女儿童之家建设）。
4. 组织开展儿童视力保障、牙齿保健等公益健康科普服务活动的社会主体。
5. 为0-3岁婴幼儿提供托育服务，市场化运营，且符合《托育机构质量评估标准》（国卫通〔2023〕13号）中A、B两档要求的托育场所运营主体（须为雄安新区范围内注册的企业、民办非企业单位（组织）、基层群众性自治组织、其他社会团体）。

# 申报条件

1. **儿童友好实践基地**：运营单位是社会主体，且复核评分达到前30%。
2. **儿童友好科普基地**：
   - 被认定为省级科普示范基地（三星级、四星级、五星级）；
   - 或被认定为国家科普教育基地、国家自然资源科普基地等各类国家级科普基地。
3. **儿童之家/妇女儿童之家**：
   - 被评为省级示范妇女儿童之家；
   - 或被评为省级儿童之家示范点；
   - 或被评为新区示范性儿童之家。
4. **公益健康科普服务活动**：符合当年度“国家和河北省基层科普行动计划”要求。
5. **托育场所**：
   - 为0-3岁婴幼儿提供托育服务；
   - 市场化运营；
   - 符合《托育机构质量评估标准》（国卫通〔2023〕13号）中A、B两档要求；
   - 经新区卫健局评估认定通过。

# 办理流程

1. **平台注册/登录**：符合条件的单位注册或登录雄安新区产业互联网平台（https://www.xaiip.org.cn）。
2. **信息填写**：在平台上填写收款信息。
3. **资金拨付**：雄安新区改革发展局定期对符合条件的运营主体按照程序拨付相应奖励。
