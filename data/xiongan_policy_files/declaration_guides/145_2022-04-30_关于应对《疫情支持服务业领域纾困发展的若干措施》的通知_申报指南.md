# 政策简介

为积极应对新冠肺炎疫情影响，落实“六稳六保”要求，特别是稳定市场主体和稳定就业，保障服务业健康有序发展，结合新区实际，提出增加市场消费活力、降低企业运行成本、加大助企支持力度、加强稳工就业保障等措施。

# 奖励额度/补贴标准

**一、消费补贴**
1. **家电类**：面向新区居民分批发放数字人民币京东线上优惠券和线下满减红包，分为满5000减500、满4000减400、满3000减300、满2000减200、满1000减100等5种。
2. **家具类**：面向新区居民分批发放数字人民币满减红包，分为满5000减500、满4000减400、满3000减300、满2000减200、满1000减100等5种。
3. **餐饮类**：面向新区分批发放数字人民币满减红包，分为满200减40、满100减20、满50减10等3种。
4. **商超类**：每周发放一批满50减15的数字人民币满减红包。

**二、租金与运营补贴**
1. **租金减免**：
   - 2022年被列为疫情中高风险地区所在地的新区服务业小微企业和个体工商户承租国有资产类经营用房，减免6个月租金。
   - 其他所在地的减免3个月租金。
2. **非国有资产租金补贴**：对于在新区注册且受疫情影响较为严重的旅游、文化、体育、交通运输、快递物流类小微企业，承租非国有资产类经营用房的，按照1元/平方米·天的标准给予6个月补贴，单家企业最高10万。
3. **运营补贴**：对于在新区注册的旅游、文化、体育、交通运输、快递物流类小微企业，可在用水、用气、用电价格方面给予一定比例的运营补贴，单家企业最高5万。

**三、金融与就业支持**
1. **贷款贴息**：对于服务业中小微企业，获得银行等金融机构新增贷款，按照不高于贷款合同签订当日执行的中国人民银行公布的一年期贷款市场报价利率（LPR）予以贴息，单家企业最高30万。
2. **新增就业补贴**：
   - 新区注册服务型企业在2022年5月1日至12月31日期间新招用雄安新区三县户籍人员，补贴标准1000元/人。
   - 新区注册的其他类型企业和参建企业（含劳务分包企业），补贴标准500元/人。
3. **零工驿站建设补贴**：经新区人社部门认定的建设方，可通过产业互联网平台申请不高于20万元补贴。
4. **劳务品牌培育补贴**：经新区人社部门认定的建设方，可通过产业互联网平台申请不高于5万元资金补贴。

# 注意事项

1. **有效期**：除已明确限定期限外，有效期至2022年12月31日。
2. **活动期限**：家电、家具、商超类消费补贴活动时间为3个月；运营补贴时间为2022年1月1日至6月30日；新增就业补贴申请期间为2022年5月1日至12月31日。
3. **合规要求**：对于在工作中弄虚作假、骗取财政资金等违法行为，将按照有关规定严肃处理。
4. **认定要求**：新增就业补贴需在产业互联网平台进行注册并经新区人社部门认定；零工驿站和劳务品牌需经新区人社部门认定。

# 申请对象

1. **居民**：新区居民（家电、家具、餐饮、商超类消费补贴）。
2. **企业及个体户**：
   - 新区服务业小微企业和个体工商户（租金减免）。
   - 在新区注册且受疫情影响较为严重的旅游、文化、体育、交通运输、快递物流类小微企业（非国有资产租金补贴、运营补贴）。
   - 服务业中小微企业（贷款贴息）。
   - 缴纳房产税和土地增值税确有困难的新区服务企业（税费减免）。
   - 新区注册服务型企业、其他类型企业和参建企业（含劳务分包企业）（新增就业补贴）。
3. **行业人群**：交通运输、快递物流、外卖配送、餐饮住宿、商超零售、文体娱乐等行业中符合新冠肺炎疫情防控重点人群（免费核酸检测）。
4. **建设方**：零工驿站建设方、劳务品牌建设方（需经新区人社部门认定）。

# 申报条件

1. **消费补贴**：
   - 家电/家具：新区居民，购买绿色智能家电或绿色环保家具。
   - 餐饮：在参加活动的餐饮商家消费。
   - 商超：在参加活动的商超使用。
2. **租金减免**：
   - 承租国有资产类经营用房的新区服务业小微企业和个体工商户。
   - 2022年被列为疫情中高风险地区所在地的，或其他所在地。
3. **非国有资产租金补贴**：
   - 在新区注册。
   - 受疫情影响较为严重。
   - 属于旅游、文化、体育、交通运输、快递物流类小微企业。
   - 承租非国有资产类经营用房。
4. **运营补贴**：
   - 在新区注册。
   - 属于旅游、文化、体育、交通运输、快递物流类小微企业。
5. **贷款贴息**：
   - 服务业中小微企业。
   - 获得银行等金融机构新增贷款。
6. **新增就业补贴**：
   - 新区注册企业。
   - 在2022年5月1日至12月31日期间新招用雄安新区三县户籍人员（以签订劳动合同时间为准）。
   - 在产业互联网平台进行注册并经新区人社部门认定。
7. **零工驿站/劳务品牌补贴**：
   - 经新区人社部门认定的建设方。

# 办理流程

1. **消费补贴**：分批发放数字人民币优惠券或满减红包，居民在指定场景使用。
2. **租金减免**：针对承租国有资产类经营用房的企业和个体户，直接减免租金。
3. **非国有资产租金补贴/运营补贴/贷款贴息/就业补贴/驿站及品牌补贴**：
   - 部分补贴需通过“产业互联网平台”进行注册、申领或申请。
   - 部分项目需经新区人社部门认定。
   - 税费减免需向主管税务机关申请。

1. **消费补贴**：活动时间为3个月（具体起止日期原文未详述，仅提及“首批”、“分批发放”）。

3. **新增就业补贴**：2022年5月1日至12月31日（以签订劳动合同时间为准）。
4. **政策有效期**：除已明确限定期限外，有效期至2022年12月31日。
