# 政策简介

为做好2024年省级科技型中小企业评价备案工作，根据《河北省科技型中小企业评价服务办法》（冀科创服规〔2023〕1号）和省科技厅有关规定，雄安新区组织开展2024年省级科技型中小企业评价工作。旨在积极发动辖区内符合条件的中小企业参与评价，培育更多高质量企业纳入全省科技型中小企业后备库，使企业更好发挥创新主体作用。强化创新政策落实，加大政策宣传和对企业的培育指导，特别是落实好研发费用税前加计扣除等优惠政策，激励企业加大研发投入，提高企业科技创新能力。

# 申请对象

符合条件的中小微企业。

# 申报条件

1. 企业需持续开展科技创新活动，具有自主知识产权、专有技术或创新集成能力。
2. 企业对所填报信息和上传资料的准确、真实、合法、有效性承担有关法律责任。
3. 对确认核实不符合科技型中小企业条件的，将取消证书编号。

# 申报材料

1. 《河北省科技型中小企业评价申请表》。
2. 相关佐证资料。

# 办理流程

1. **注册填报**：符合条件的中小微企业在“河北省科技型中小企业综合管理平台”（网址：https://zxqy.hebkjt.cn/）以企业统一社会信用代码为用户名进行注册，在线填报《河北省科技型中小企业评价申请表》、上传相关佐证资料。
2. **受理审查和评审认定**：区县科技管理部门负责对《评价申请表》及相关佐证资料进行评价审核。市级科技管理部门按季度复核，重点对企业持续开展科技创新活动能力，具有自主知识产权、专有技术或创新集成能力等内容进行研判，确定通过评价的科技型中小企业名单于3、6、9、11月20日前批次推荐至省厅。
3. **备案登记**：省厅按批次对通过评价的企业与省市场监督局进行核实，符合要求的进行备案登记。企业名称及证书编号可在“河北省科技型中小企业综合管理平台”查询。
4. **复审申请**：“河北省科技型中小企业证书”有效期为3年，企业应在证书到期的上个季度在综合管理平台上进行复审填报，复审流程与申报一致。

# 申报时间

1. 采用常年申报、分季度受理的形式。
2. 雄安新区改革发展局申报受理截至日期为3、6、9、11月15日。
3. 《评价申请表》受理截止日期为11月15日。
4. 逾期将延至下一批审核。

# 注意事项

1. 一站式管理：省级科技型中小企业在“河北省科技型中小企业综合管理平台”进行常态化申报、评价和备案，坚持科技型中小企业评价工作全流程网上办理。
2. 如遇问题可及时与属地科技管理部门咨询联系。
