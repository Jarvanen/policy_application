# 政策简介

为贯彻落实河北雄安新区党工委管委会党政办公室印发《关于应对疫情支持服务业领域纾困发展的若干措施》的通知要求，针对受疫情影响较为严重的旅游、文化、体育、交通运输、快递物流类小微企业，支持企业渡过难关，制定本细则。本政策旨在切实减轻疫情期间企业房租负担，所需资金从传统产业转型升级专项资金列支。

# 申请对象

在新区行政区域内依法注册，具有独立法人资格，主营业务为旅游、文化、体育、交通运输、快递物流类小微企业，且符合行政主管部门要求的企业。

# 申报条件

1. 在新区行政区域内依法注册，具有独立法人资格，主营业务为旅游、文化、体育、交通运输、快递物流类小微企业，且符合行政主管部门要求；
2. 租赁房屋为新区范围内符合相关规定非国有资产的经营性用房；
3. 房屋租赁合同需签订满一个月以上，同时在申报截止日仍在有效期内；
4. 申报企业需提供2022年6月1日前纳税申报记录。

# 申报材料

1. 《疫情租金补助申请表》(见附件2)；
2. 企业营业执照及法定代表人身份证；
3. 企业在有效期内的租房合同;
4. 需提供出租方(企业/个人)信息：营业执照、房产证及法定代表人身份证(房产证及房主身份证)扫描件；
5. 带企业名称的门头照片；
6. 2022年6月1日前纳税申报记录；
7. 与申报有关的其他证明。

# 办理流程

1. **申报**：申报主体需通过新区产业互联网平台进行申报(产业互联网平台网址http://120.53.124.140:10001)，填写《申请表》并附相关认定材料。
2. **受理**：产业互联网平台对申报材料与内容完备性进行初筛，材料齐全的予以提交受理；不齐全的，系统一次性告知原因并无法进行提交。申请单位在2个工作日内补充申报材料与内容完备的，予以提交受理；逾期未补充申报或补充申报材料与内容仍不完备的，不予受理。
3. **审核**：核查企业提供相关材料信息，必要时可通过电话、上门、委托第三方机构等方式予以核实认定，企业申报提交受理后，要求7个工作日内完成审核。(三县文化、旅游、体育、交通、发改等有关部门按照行业管理职责分工负责)
4. **公示**：对审核通过的，在产业互联网平台进行公示。公示结束，三县相关部门将《申请表》《汇总表》(见附件3)盖章后报新区改革发展局。(三县文化、旅游、体育、交通、发改等有关部门按照行业管理职责分工负责)
5. **汇总**：新区改革发展局对三县资金需求进行审核汇总，并组织相关部门抽查企业运营情况。(新区改革发展局牵头，宣传网信局、规划建设局分工负责)

# 注意事项

1. **主体责任与承诺**：坚持企业自愿申报，企业承担主体责任，企业须出具真实性承诺(附件1)，企业法定代表人对申报材料和相关数据的真实性负责。
2. **违规处罚**：如有伪造证件材料、虚报冒领等欺瞒行为，自愿接受下列处罚：
    - 按照《财政违法行为处罚处分条例》(国务院令第427号)第十四条之规定，追回违反规定使用、骗取的补贴资金，给予警告，没收违法所得，并处被骗取补贴资金10％以上50％以下的罚款或者被违规使用补贴资金10％以上30％以下的罚款；对直接负责的主管人员和其他直接责任人员处3000元以上5万元以下的罚款；
    - 在新闻媒体上曝光；
    - 情节严重，触犯法律的，移交司法机关处理。
3. **资金列支**：本实施办法所需资金，从传统产业转型升级专项资金列支。
