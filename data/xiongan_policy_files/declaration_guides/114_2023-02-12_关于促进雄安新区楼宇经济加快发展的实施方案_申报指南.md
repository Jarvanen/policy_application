# 政策简介

本政策旨在有效繁荣雄安新区存量及增量楼宇，有序承接北京非首都功能疏解，加快以楼宇为载体吸纳各类创新要素、集聚高端高新产业，加速提升区域经济发展水平。政策围绕“五新”目标，坚持“重点突破、集中繁荣、特色鲜明、联动发展”的总体思路，聚焦楼宇主题、服务能力、入驻率、经济贡献等核心指标。通过制定楼宇经济“普惠+专项”奖励政策，支持运营主体和入驻企事业单位在注册开办、物业租购、人才服务、金融服务等方面获得支持，并针对金融科技、数字经济、生物技术、新能源、新材料等重点产业提供专项支持。同时，鼓励打造主题楼宇、特色楼宇，建立“亿元楼、五千万楼”考评机制，对区域经济贡献突出的楼宇给予“一事一议”奖励。

# 奖励额度/补贴标准

1. **楼宇运营主体奖励**：
   - 对楼宇运营主体根据发展绩效分档给予资金奖励和荣誉授牌。
   - 对楼宇运营主体根据总量和增量贡献，每年认定一批营收规模、综合贡献突出的主题楼宇，给予一定奖励。
   - 对引入支撑性、带动性、示范性企业的楼宇运营主体，视情况给予一次性补贴奖励。
   - 对区域经济贡献突出的楼宇给予“一事一议”奖励。

2. **入驻企事业单位支持**：
   - 对符合条件的疏解单位、疏解服务保障单位及符合雄安新区产业准入的企事业单位，给予落户补助、房租补贴、综合贡献奖励、人才激励等支持。
   - 对入驻的服务外包企业和培训机构根据贡献给予一定补贴奖励。
   - 对符合条件的楼宇引进的高端服务业人才，按照相关人才等政策给予奖励。

# 注意事项

1. **有效期**：附件2中的七条措施自发文之日起施行，有效期至2025年。
2. **不重复享受原则**：对措施所列各项支持政策与新区其他政策有重复交叉的，或同一事项涉及多项补贴的，按照从高从优不重复的原则执行。
3. **一事一议**：对具有引领性、示范性、带动性的市场主体可另行“一事一议”。
4. **考核指标**：将企业入驻、经济贡献、服务创新等纳入楼宇运营支持政策兑现指标体系。

# 申请对象

1. **楼宇运营主体**：包括新区楼宇运营主体、全国知名商业地产开发企业、央企、国内外知名总部企业、上市企业、龙头骨干企业等。
2. **入驻企事业单位**：
   - 疏解单位、疏解服务保障单位。
   - 符合雄安新区产业准入的企事业单位。
   - 六类疏解单位、国央企及民企分支机构和区域总部。
   - 高新技术企业和“专精特新”企业。
   - 国内外具有一定影响力的大型零售（餐饮）企业、连锁超市、星级酒店、会计师事务所、律师事务所、税务师事务所、金融科技、融资租赁、科技中介等机构。
   - 发展ITO（信息技术外包）、BPO（业务流程外包）、KPO（知识流程外包）等服务外包和新区产业发展需要的各类职业认证培训等产业的企业和培训机构。

# 申报条件

1. **楼宇运营主体条件**：
   - 参与开发建设高品质楼宇并主导后续楼宇运营。
   - 独立或联合建设办公楼宇，符合条件的享受相关用地支持。
   - 培育高端品牌楼宇，营收规模、综合贡献突出。
   - 面向特定对象开展“二次招商”，构建特色楼宇产业生态，引入支撑性、带动性、示范性企业。
   - 完善文印、餐饮、零售、酒店、文娱、停车、外卖、快递、幼托等各类配套设施及服务。
   - 打造楼宇党建共同体，引进高端服务业人才。

2. **入驻企事业单位条件**：
   - 优先入驻新开发功能片区楼宇。
   - 符合雄安新区产业准入。
   - 属于六类疏解单位、国央企及民企分支机构和区域总部、高新技术企业和“专精特新”企业等。
   - 属于国内外具有一定影响力的大型零售（餐饮）企业、连锁超市、星级酒店、会计师事务所、律师事务所、税务师事务所、金融科技、融资租赁、科技中介等机构。
   - 属于发展ITO、BPO、KPO等服务外包和新区产业发展需要的各类职业认证培训等产业的企业和培训机构。

# 办理流程

1. **政策制定与宣传**：新区相关部门制定楼宇经济“普惠+专项”奖励政策，并通过多种渠道进行宣传推广。
2. **申报与审核**：
   - 楼宇运营主体和入驻企事业单位根据政策要求，向新区投资促进部门或相关专班办公室提交申请。
   - 新区相关部门对申请材料进行审核，结合楼宇信息管理平台的数据进行核实。
3. **评审与认定**：
   - 对楼宇运营主体进行发展绩效评估，分档给予资金奖励和荣誉授牌。
   - 每年认定一批营收规模、综合贡献突出的主题楼宇。
   - 对引入支撑性、带动性、示范性企业的情况进行评估，给予一次性补贴奖励。
4. **公示与兑现**：对审核通过的对象进行公示，无异议后兑现相关奖励和补贴。
5. **跟踪问效**：建立新区楼宇经济发展简报制度，对重大政策落实、重大项目推进、重大事项研究全程介入、跟踪问效。
