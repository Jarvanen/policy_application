# 政策简介

为有效承接北京非首都功能疏解，加快实现“五新”目标，加速科技创新资源集聚，积极构建雄安新区现代产业体系，做强做优数字经济，促进符合产业定位的企业高质量发展，河北雄安新区管理委员会设立科技企业库、“专精特新”企业培育库、上市企业后备库、孵化载体库四个企业库，并依据企业所属库别分类施策。在新区辖区范围工商注册、纳税人身份信息确认，符合相应条件的企业即纳入对应类型企业库进行服务和支持。

# 奖励额度/补贴标准

**一、科技企业库入库企业支持政策**
1. **贷款贴息**：对在库企业销售订单、应收账款等相关流动资金贷款进行贴息，按照贷款合同签订实际贷款利率的50%予以贴息，期限两年，每家企业的贴息总额不超过100万元。
2. **开办补助**：首次入库，一次性给予2万元补助。
3. **租金补贴**：对在新区孵化载体内租赁且自用办公用房的企业，给予1.5元/平方米/天租金补贴，每年不超过50万元。
4. **科技创新券**：面向在库企业，以5万元/年为上限发放科技创新券，最高可支付企业实际支出费用的50%。

**二、“专精特新”企业培育库入库企业支持政策**
1. **入库及认定奖励**：
   - 首次纳入新区“专精特新”企业培育库的企业，一次性给予20万元奖励；
   - 首次评定为省级“专精特新”企业，再次给予30万元奖励；
   - 首次评定为国家级专精特新“小巨人”企业再次给予50万元奖励。
   - 对疏解到新区的各类“专精特新”企业给予相应奖励。
2. **科技创新券额度提升**：在库企业科技创新券使用额度上限提高至15万元/年，使用范围和支付比例不变。
3. **智能设备投入补助**：对投入强度大、提升效益明显的，按软件投入给予30%补助，最高不超过100万元。
4. **地方贡献奖励**：对新区产业贡献度较大的在库企业，按照贡献程度分别给予不超过100万元、300万元、600万元的奖励资金。

**三、孵化载体库入库企业支持政策**
1. **专业化奖励**：对首次入库的专业孵化载体，按其认定(备案)为省、国家级专业孵化载体的等级，依次给予60万元、80万元一次性专业化奖励。
2. **引进培育奖励**：
   - 引进和培育的省级以上“专精特新”企业：不低于15万元/家；
   - 引进和培育的高新技术企业：10万元/家；
   - 引进和培育的国家级科技型中小企业：2万元/家；
   - 成功挂牌上市（含主板、创业板、科创板及海外主板）：给予孵化载体一次性奖励60万元。

# 注意事项

1. **申报平台**：本措施依托雄安新区科创服务区块链平台进行管理，企业入库申请在该平台填报。
2. **政策叠加原则**：“专精特新”企业培育库和上市企业后备库是科技企业库的子集，其在库企业可享受科技企业库政策。企业同时满足各库相同类型的支持政策条件的或同时满足本措施与新区其他同类型支持政策条件的，按照从高不重复的原则予以支持。
3. **重大事项支持**：对重点企业、重要项目等重大事项，采取“一事一议”方式，进行专项政策支持。

# 申请对象

1. **科技企业库**：在新区辖区范围工商注册、纳税人身份信息确认，产业方向符合雄安新区高端高新产业、数字经济核心产业定位的疏解企业或新区企业。
2. **“专精特新”企业培育库**：科技企业库内，年营业收入达标、发展方向符合“专业化、精细化、特色化、新颖化”或数字经济核心产业要求的疏解企业或新区企业。
3. **上市企业后备库**：疏解至新区或新设立的，主营业务突出、竞争能力较强、盈利水平较好、发展潜力较大，有明确的上市意愿，并成立专门的上市筹备工作团队的科技企业库内企业。
4. **孵化载体库**：在新区注册并经新区认定(备案)的孵化载体（包括众创空间、科技企业孵化器、科技企业加速器等）。

# 申报条件

**一、科技企业库认定标准**
同时达到以下条件：
1. 具有独立法人资格；
2. 产业方向符合雄安新区高端高新产业、数字经济核心产业定位；
3. 在职员工大学专科及以上学历占比达20%以上；
4. 上年销售收入超过500万元，R&D总额占同期销售收入总额的比例原则上超过3%；
5. 企业入库当年内未发生重大安全、重大质量事故或严重环境违法等行为。
*注：新区范围内的高新技术企业、省级或国家级科技型中小企业，以及当年申报高新技术企业通过新区初审，可直接纳入本库。*

**二、“专精特新”企业培育库认定标准**
对同时达到以下条件的科技企业库内企业：
1. 上年度营业收入超过1000万元；
2. 具备持续研发能力，符合以下“专精特新”发展方向之一，具有培育发展潜力：
   - **专业化**：某一产品在细分市场处于领先地位，主导产品销售收入占本企业销售收入的60%以上；拥有与主业相关的自主知识产权1项以上。
   - **精细化**：企业取得质量管理体系认证；拥有自主品牌或注册商标一年以上。
   - **特色化**：企业拥有专利权、软件著作权，或近2年经过专门机构认定的专有技术1项以上；企业研发投入占销售收入的比例超过3%。
   - **新颖化**：在研发设计、生产制造、供应链管理等环节，至少有1项核心业务采用信息化系统支撑。
*注：已被认定为河北省省级“专精特新”的新区企业和符合数字经济核心产业定位的企业，可直接纳入本库予以支持。*

**三、上市企业后备库认定标准**
对主营业务突出、竞争能力较强、盈利水平较好、发展潜力较大，有明确的上市意愿，并成立专门的上市筹备工作团队的科技企业库内企业，经新区认定，纳入“挂牌上市后备企业资源库”。

**四、孵化载体库认定标准**
对围绕新区产业发展重点，由产业空间开发主体、龙头骨干企业、高等院校、科研院所、新型研发机构、投资机构、专业孵化机构等设立，运营主体在新区辖区范围内注册，经新区认定(备案)的众创空间、科技企业孵化器、科技企业加速器等孵化载体。

# 办理流程

1. **申报**：企业依托雄安新区科创服务区块链平台填报入库申请。
2. **认定/入库**：经新区认定后，纳入对应类型企业库（科技企业库、“专精特新”企业培育库、上市企业后备库、孵化载体库）。
3. **支持落实**：依据入库类别享受相应的信贷、贴息、补助、奖励等服务和支持。
