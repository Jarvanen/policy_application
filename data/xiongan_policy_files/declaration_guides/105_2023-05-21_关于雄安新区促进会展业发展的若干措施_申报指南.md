# 政策简介

为贯彻落实《国务院关于进一步促进展览业改革发展的若干意见》和《河北省关于推动会展经济高质量发展的实施意见》文件精神，推动新区会展业高质量发展，结合新区实际，制定本措施。主要支持方向包括：支持举办展览项目（鼓励规模化、连续、重点品牌办展）；支持举办高端会议项目（鼓励引进、规模化、高端办会）；培育本地会展业发展（支持新成立会展企业、龙头企业做大做强、国际化发展）；强化会展要素保障（提高场馆利用率、人才支持、专属参观通道、智能线上会展政务平台）。

# 奖励额度/补贴标准

**一、支持举办展览项目**
1. **鼓励规模化办展**：
   - 实际会期2天以上，标准展位数100（含）以上：奖励主办方10万元，每增加一个标准展位多奖励300元。
   - 符合新区5大产业发展方向：奖励主办方20万元，每增加一个标准展位多奖励600元。
   - 单个展览项目每届规模奖励上限50万元。
2. **鼓励连续办展**：
   - 连续多届举办，每届间隔超6个月，每届标准展位数大于前一届，且符合“鼓励规模化办展”条件：每届奖励主办方实际增加支出的20%。
   - 同一个展览项目奖励不超过5届，每届上限20万元。
3. **鼓励重点品牌展览办展**：
   - 符合新区“创新发展示范区”定位且达到“鼓励规模化办展”标准，或入选省级重点产业品牌会展名单且符合新区主导产业重点方向：奖励主办方20万元。
   - 连续三年在雄安新区举办的，自第三年起奖励提高至50万元。

**二、支持举办高端会议项目**
1. **鼓励引进高端会议项目**：
   - 引进符合新区5大产业发展方向会议项目：给予会议项目场地租赁费5%的一次性奖励，上限20万元。
   - 引进国家级、国际性、世界500强企业或中国百强企业主导的会议项目：额外奖励20万元。
2. **鼓励规模化办会**：
   - 实际会期2天及以上，参会人数200（含）～400（不含）人：奖励主办方6万元。
   - 参会人数400（含）～600（不含）人：奖励主办方12万元。
   - 参会人数600（含）人以上：奖励主办方15万元。
3. **鼓励举办高端会议**：
   - 纳入UFI、ICCA等国际权威机构统计范围：在给予“鼓励规模化办会”奖励的同时，额外给予10万元奖励；连续举办3届后，奖励提升至20万元。
   - 符合“鼓励规模化办会”条件，参会人员不少于三个国家或地区且不少于20%总参会人数的国际会议：给予主办方租用会议场地实际支出40%奖励，最高不超过40万元。

**三、培育本地会展业发展**
1. **支持新成立会展企业**：
   - 本年度在新区设立从事会展业务且依法纳税的独立法人企业：从产生地方贡献的下一年度起三年内给予经营补贴，补贴额度不高于上一年度企业地方贡献的30%，每年补贴金额上限50万元。
   - 在新区新注册经营或设立分支机构的重点会展企业（机构、协会等），经认定租赁办公用房：自起租之日起，按照不超过500平方米每年的标准，给予不高于房租50%的补贴，连续补贴3年，累计总额不超过50万元。
2. **支持龙头企业做大做强**：
   - 在新区注册从事会展业务且依法纳税的独立法人企业，年主营业务收入超过1000万元：给予20万元一次性奖励。
   - 年主营业务收入超过3000万元：给予40万元一次性奖励。
3. **支持会展企业国际化发展**：
   - 加入UFI、ICCA等国际会展机构或者取得UFI、ICCA等国际会展认证的会展企业：给予20万元一次性奖励。

**四、强化会展要素保障**
1. **提高展览场馆利用率**：
   - 场馆全年承办标准展位数100（含）以上的展览达10场（含）以上：给予场馆单位5万元奖励。
   - 全年承办标准展位数100（含）以上的展览达20场（含）以上：给予场馆单位10万元奖励。

**五、其他限制**
- 单个展览项目每届奖励最高不超过110万元。
- 单个会议项目每届奖励最高不超过50万元。

# 注意事项

1. **资金渠道**：所需资金从产业发展专项资金里细化。
2. **报备要求**：会展项目主办方需在活动举办前向新区有关部门报备，经审核符合条件的方可按程序申请奖励资金。
3. **凭证要求**：项目主办方资质认定需提供在新区实际发生的消费的发票、合同等作为凭证。
4. **认证要求**：会议引进单位需由会议主办方通过提供承诺书、第三方服务合同等方式进行认证。

# 申请对象

1. **展览/会议主办方**：在雄安新区举办展览项目或会议项目的单位。
2. **会议引进单位**：引进符合新区5大产业发展方向会议项目的单位。
3. **会展企业**：
   - 本年度在新区设立从事会展业务且依法纳税的独立法人企业。
   - 在新区新注册经营或设立分支机构的重点会展企业（机构、协会等）。
   - 在新区注册从事会展业务且依法纳税的独立法人企业（针对龙头企业奖励）。
   - 加入国际会展机构或取得国际认证的新区会展企业。
4. **场馆单位**：在新区承办展览的场馆。
5. **会展相关人才**：符合条件的会展相关人才（申领“雄才卡”）。

# 申报条件

**一、展览项目条件**
1. **规模化办展**：实际会期达到2天以上，标准展位数达到100（含）以上。
2. **连续办展**：连续多届在雄安新区举办，每届间隔超过6个月，每届标准展位数大于前一届，且符合“鼓励规模化办展”奖励条件。
3. **重点品牌展览**：符合新区“创新发展示范区”定位，促进新区相关产业发展有明显作用，且达到“鼓励规模化办展”奖励标准；或入选省级重点产业品牌会展名单且符合新区主导产业重点方向。

**二、会议项目条件**
1. **引进高端会议**：符合新区5大产业发展方向；或为国家级、国际性、世界500强企业或中国百强企业主导。
2. **规模化办会**：实际会期达到2天及以上，参会人数200（含）人以上。
3. **高端会议**：纳入国际展览业协会（UFI）、国际会议协会（ICCA）等国际权威机构统计范围；或参会人员不少于三个国家或地区且不少于20%总参会人数。

**三、会展企业条件**
1. **新成立企业**：本年度在新区设立，从事会展业务，依法纳税的独立法人企业。
2. **重点会展企业**：在新区新注册经营或设立分支机构，经过认定，在新区租赁办公用房。
3. **龙头企业**：在新区注册，从事会展业务，依法纳税的独立法人企业，年主营业务收入超过1000万元或3000万元。
4. **国际化企业**：加入UFI、ICCA等国际会展机构或者取得UFI、ICCA等国际会展认证。

**四、场馆条件**
- 全年承办标准展位数100（含）以上的展览达10场（含）以上或20场（含）以上。

# 办理流程

1. **事前报备**：会展项目主办方需在活动举办前向新区有关部门报备。
2. **资格审核**：经审核符合条件的方可按程序申请奖励资金。
3. **资质认定**：项目主办方资质认定需提供在新区实际发生的消费的发票、合同等作为凭证；会议引进单位需由会议主办方通过提供承诺书、第三方服务合同等方式进行认证。
4. **资金申请**：按程序申请奖励资金。

# 申报时间

- 会展项目主办方需在活动举办前向新区有关部门报备。
- 新成立会展企业经营补贴从产生地方贡献的下一年度起三年内给予。
- 重点会展企业房租补贴自起租之日起计算。
