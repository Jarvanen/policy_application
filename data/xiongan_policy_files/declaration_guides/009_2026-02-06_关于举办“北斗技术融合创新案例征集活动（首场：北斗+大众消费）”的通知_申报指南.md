# 政策简介

为深入贯彻国家北斗卫星导航系统建设与应用推广的战略部署，充分释放北斗技术创新潜能与产业价值，推动北斗与5G、人工智能、物联网等前沿技术深度融合，发挥中国时空龙头引领带动作用，构建连接技术、人才与产业的开放性交流合作平台，河北雄安新区工信科技数据局与中国时空信息集团拟围绕“北斗+大众消费”“北斗+应急”“北斗+交通”“北斗+农业”等重点领域联合主办“北斗技术融合创新案例征集系列活动”。本次首场活动聚焦“北斗+大众消费”领域，面向企事业单位、高等院校、科研院所等征集具有前瞻性与实用价值的北斗应用解决方案，重点聚焦北斗高精度定位、短报文通信、通导遥融合等核心技术领域的创新突破与场景落地。

# 申请对象

1.  **企事业单位**：应具有独立法人资格，无重大违法违规行为及不良社会信用记录，并具备较强的科技人才队伍与突出的科技创新能力。
2.  **自然人**：须为国内外高等院校、科研院所等单位中具备空天信息相关创新研发能力的团队或个人。
3.  **联合申报**：允许联合申报，联合申报的牵头主体须满足相应组别上述申报条件，联合申报主体总数不超过3家（含牵头单位）。

# 申报条件

1.  **方向符合性**：申报案例须符合“北斗+大众消费”赛题方向，包括但不限于以下类别：
    *   智能出行与共享交通（如共享单车/电单车精准停放与管理、车道级导航、驾驶行为分析等）；
    *   智慧生活与安全监护（如民生穿戴设备、宠物智能定位、具身智能设备空间定位、居家安防定位告警等）；
    *   文旅导览与户外探险（如智能运动装备、景区智能导览、户外探险轨迹记录、研学旅行定位打卡、露营地精准选址、非遗文化场景AR定位导览等）；
    *   大众消费与融合创新（如北斗短报文在消费场景中的应急通信应用、通导遥融合消费级监测产品、跨场景北斗定位数据融合服务等）。
2.  **唯一性**：每个申报案例限报一个活动方向；同一申报主体可针对多个征集方向进行申报，但同一案例仅能选择一个方向进行申报，不得重复提交。
3.  **知识产权**：申报案例的创意、产品、技术及相关专利等知识产权应归属申报单位，并未侵犯任何他人的专利权、著作权、商标权及其他知识产权，且不得违反国家相关法律法规。
4.  **真实性**：参赛团队需遵守赛事规则，对所有信息的准确性和真实性负责，一经发现虚假信息将取消参赛资格。
5.  **名称规范**：具体申报案例名称自行拟定，符合活动要求，能体现出活动主题，不得包含不文明字样。

# 申报材料

1.  **活动案例申报书**：模板详见附件，内容可包括但不限于案例简介、需求分析、解决方案、案例价值等。
    *   *作品简介*：研发背景、核心目标、总体技术框架与主要内容、当前应用或验证阶段情况，重点突出与“北斗+大众消费”主题的契合度及北斗技术特色。
    *   *需求分析*：痛点问题分析、应用场景需求、市场定位分析。
    *   *实施方案*：根据产品类、应用类、服务类分别描述（如核心产品设计、技术实现路径、研发与生产计划、场景落地架构、方案架构设计等）。
    *   *技术创新突破及最新情况*：技术创新点、作品开发情况（概念设计、原型开发、小批量试制、已上市产品升级等）。
    *   *作品价值*：服务受众、运营情况、成本分析、经济效益、社会效益、未来应用前景、创新性分析等。
2.  **单位及团队介绍**：包括申报单位相关的基本资质、信用情况等，活动案例相关的知识产权证明等材料，以及申报团队成员介绍、团队分工等。
3.  **其他相关材料**：例如案例评审时需要的介绍PPT，可直观展示参赛作品效果的视频、作品的模型 Demo 和说明文档等。
4.  **格式要求**：参赛者需提交易于电子设备阅读的可视化文件（包括但不限于jpg/pdf/ppt/docx），要求统一以 zip 形式提交，文件大小原则上不超过 500MB。

# 办理流程

1.  **报名及参赛作品提交**：各参与活动队伍统一通过报名网址进行在线报名及材料上传。
2.  **初选**：由组委会对申报主体资格、材料完整性和内容符合性进行审核，确定进入路演的案例名单。
3.  **路演**：最终评审采用线下路演与专家互动方式开展，内容包括10分钟PPT演示和5分钟现场答辩。评审专家按照活动评分规则对作品独立评审打分，根据最终得分现场评选出优秀案例。

*   **初选时间**：2026年3月20日前

# 注意事项

1.  **费用说明**：大赛为公益性赛事，全程不收取参赛队伍任何费用。
2.  **材料退还**：所有已提交的申报案例和相关材料原则上不予退还。
3.  **材料补充**：评审期间，申报团队须按照组委会的要求补充提交申报案例有关材料。
4.  **案例迭代**：活动期间，申报团队可在不改变案例名称和主要功能的基础上，持续推进申报案例迭代升级，对应活动最后一次提交的作品为申报案例。
5.  **解释权**：大赛最终解释权归大赛组委会所有。
