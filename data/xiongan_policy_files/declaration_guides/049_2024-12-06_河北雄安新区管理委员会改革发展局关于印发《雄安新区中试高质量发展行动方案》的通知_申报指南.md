# 政策简介

本政策为《雄安新区中试高质量发展行动方案》，旨在深入贯彻落实国家及河北省关于中试发展的工作部署，抢抓新一轮科技革命和产业变革机遇，建设一流中试服务能力，打造原始创新和自主创新重要策源地。方案聚焦顶层设计、能力提升、开放合作等方面，通过完善中试能力体系、创新发展中试产业、构建新型体制机制，促进中试与创新链产业链同步发展，加快发展新质生产力。重点支持商业航天、时空信息、医工交叉转化、前沿材料等重点产业领域，推动京津冀中试协同，构建优质高效的中试创新及服务生态体系。

# 奖励额度/补贴标准

1. **疏解机构支持**：给予疏解机构中试平台落户一定比例启动资金支持；针对未来会落户雄安的中试项目给予一定比例资金支持。
2. **重点机构支持**：重点企业和机构中试扶持政策参考疏解机构政策标准；对于关键人员，给予疏解人员相关待遇。
3. **第三方机构支持**：对重点机构引入的中试项目建立首轮投资机制；利用先投后股等方式开展支持。
4. **大赛成果支持**：对大赛优秀成果给予奖金支持；为选拔出的中试人才提供落户直接资金补贴。
5. **中试示范平台奖补**：开展中试示范平台综合评价，加大成果转化落地评价权重，对表现突出的予以奖补。
6. **人才奖励**：开展年度优秀中试人才评定，对于促进项目落地雄安的重要人才给予资金奖励；对于特别重要中试人才给予疏解人员待遇。
7. **基金收益让渡**：中试基金产生的增值收益让渡比例最高可达100%。
8. **服务支撑机构支持**：根据项目实效给予中试服务支撑机构资金支持。
9. **联盟项目支持**：对于联盟引进重点中试转化项目给予资金支持。

# 注意事项

1. **用地政策**：支持使用产业综合用地，打造研发、办公、中试、生产等功能融合发展的雨林生态。
2. **税收优惠**：落实研发费用加计扣除、增值税留抵扣税等政策。
3. **采购机制**：鼓励国有企业采购重大装备首台（套）、新材料首批次、软件首版次等创新产品，探索采购创新产品“合格免责”机制。
4. **容错机制**：进一步落实尽职免责原则，健全容错纠错机制；完善中试基金配套尽职免责机制。
5. **成果处置**：财政资金支持的项目，后期融资时财政资金可按相关规定转化为对应的投资权益；财政资金支持科技成果在一定时期内没有转化且无正当理由的，将成果纳入“先用后转”清单，鼓励通过第三方实施转化。
6. **负面清单**：试点建设应用场景示范区，制定应用场景开放负面清单。

# 申请对象

1. **疏解机构**：包括北京交通大学、北京科技大学、北京林业大学、中国地质大学(北京)以及北大人民医院、北京协和医院(国家医学中心)等高校科研机构；中国星网、中国华能、中国中化等央企。
2. **重点机构**：世界500强等重点企业，985高校、中科院院所等重点机构。
3. **第三方机构**：具备中试功能的第三方新型研发机构，具有较强专业能力和资源整合能力的技术经纪人和技术经理人创办的“小而美”第三方专业中试机构。
4. **中试平台与项目**：中试示范平台、重大中试项目、中试服务支撑机构。
5. **人才**：工程技术人才、工程管理人才、技术经理(经纪)人、“候鸟式”中试人才、中试领军人才、老科学家、老专家、老工程师等。
6. **其他主体**：专精特新中小企业、创新型企业、国有企业、科研平台、孵化载体、行业组织、龙头企业、金融机构等。

# 申报条件

1. **疏解机构**：需在雄安设立概念验证中心、中试平台等科技创新转移转化机构，或建设专业中试机构形成重要的中试转化业务单元。
2. **重点机构**：需围绕世界500强等重点企业，985高校、中科院院所等重点机构，吸引中试功能落户雄安。
3. **第三方机构**：需具备中试功能，具有较强专业能力和资源整合能力，或按照“飞地合作、离岸中试、回迁发展”原则与重点园区对接。
4. **中试示范平台**：需在商业航天和时空信息、医工交叉转化、前沿材料等重点领域打造，提供全生命周期服务，积极吸引中试创新成果在雄安落地转化。
5. **中试项目**：需解决核心难题，鼓励在中试示范平台开展合同科研；或为大赛落地雄安成果；或为联盟引进重点中试转化项目。
6. **人才**：需具备丰富中试经验，或为“候鸟式”中试人才，或为特别重要中试人才。

# 办理流程

1. **联络与摸排**：打造疏解机构中试平台服务专班，与疏解机构建立常态化联络机制；开展中试特派员工作，深入疏解机构各单位摸排科技成果和技术需求。
2. **标准制定与认定**：制定重点企业和重点高校科研机构的具体标准；制定重点第三方专业中试机构的具体标准，协助开展相关资质认定。
3. **项目申报与评审**：
    - 通过“揭榜挂帅”擂台、“以赛代评”等方式支持项目落地转化。
    - 对优秀成果实现支持保底，投资保障，竞价拍卖上不封顶等方式扩大参与度。
    - 开展中试示范平台综合评价，加大成果转化落地评价权重。
    - 开展年度优秀中试人才评定。
4. **资金与资源支持**：
    - 给予疏解机构中试平台落户一定比例启动资金支持。
    - 对重点机构引入的中试项目建立首轮投资机制，利用先投后股等方式开展支持。
    - 对大赛优秀成果给予奖金支持，提供全方位落地服务。
    - 对表现突出的中试示范平台予以奖补。
    - 对促进项目落地雄安的重要人才给予资金奖励。
    - 根据项目实效给予中试服务支撑机构资金支持。
    - 对于联盟引进重点中试转化项目给予资金支持。
5. **场景与对接服务**：为中试机构开展中试活动提供必要场地支持；为中试成果应用提供场景及下游对接服务；搭建场景-资本对接平台；定期召开专题场景对接会。
6. **金融支持**：开展中试成果转化路演，搭建项目-基金对接平台；鼓励健全适应中试项目融资需求特点的信贷产品服务；探索建立雄安中试项目融资担保基金。
