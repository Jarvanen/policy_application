# 政策简介

本措施旨在贯彻落实《国务院办公厅关于以高水平开放推动服务贸易高质量发展的意见》要求，加快雄安新区服务外包产业高端化、高质量发展。服务外包是指党政机关、企事业单位（服务发包商）将内部的某些流程或职能，转移给外部服务提供商进行交付的经济活动；服务外包企业是指根据其与服务发包商签订的中长期服务合同，向客户提供约定服务的服务提供商；服务外包业务是指服务外包企业向客户提供的信息技术外包（ITO）、业务流程外包（BPO）和知识流程外包（KPO）。

# 奖励额度/补贴标准

1. **疏解支持**：服务外包企业整体疏解到新区或在新区设立控股公司，根据员工数量给予三年合计每人10000元的支持，最高不超过100万元。
2. **生态打造支持**：对新区服务发包商（党政机关、事业单位除外）通过业务关联吸引服务外包企业依法依规疏解到新区的，每疏解1家企业给予10万元的资金支持。
3. **企业认定奖励**：
   - **成长型服务外包企业**：首次认定一次性给予100万元资金支持。
   - **骨干型服务外包企业**：首次认定一次性给予200万元资金支持。
   - **龙头型服务外包企业**：首次认定一次性给予300万元资金支持。
4. **人才生活补贴**：鼓励高校优秀应届毕业生到新区服务外包企业工作，入职后5年内每月按博士生3000元、硕士生2000元、本科生1000元发放生活补贴。
5. **技能培训支持**：对未纳入《河北省职业技能培训职业（工种）目录及补贴标准（2022版）》且与生产有直接关联的培训，经认定给予每人培训费用80%最高不超过8000元的培训费用支持。
6. **展会活动支持**：对企业参加商务主管部门指定的境内外服务外包（服务贸易）专业展会、投资促进、经贸交流等活动，给予参加活动实际产生的场地费、布展费的50%最高不超过20万元的费用支持。
7. **公共服务平台支持**：支持建设服务外包产业公共服务平台，依法依规给予按照服务合同额50%的资金支持，每个平台企业奖励最高不超过100万元。

# 注意事项

1. **不重复享受**：企业同时满足同类支持政策的，按照从高不重复的原则执行。
2. **有效期**：本措施自发布之日起实施，有效期2年。施行期间如遇相关政策调整，本措施同步调整。
3. **解释权**：本措施由新区投资促进主管部门负责解释。

# 申请对象

1. **服务外包企业**：具有独立法人资格，且市场主体登记地和税收缴纳关系在新区范围内。
2. **服务发包商**：新区范围内的党政机关、企事业单位（党政机关、事业单位除外的情形见第五条）。
3. **高校优秀应届毕业生**：到新区服务外包企业工作的人员。
4. **企业或培训机构**：开展技能提升培训的相关主体。
5. **三县和新建片区楼宇运营主体**：建设专业园区和楼宇的主体。

# 申报条件

**一、服务外包企业基本条件**
1. 具有独立法人资格，且市场主体登记地和税收缴纳关系在新区范围内；
2. 在新区注册满1年以上，工作人员不少于50人；
3. 企业服务外包业务年营业收入1000万元以上，或国际服务外包业务年营业收入150万美元以上；
4. 主要从事服务外包业务，且当年服务外包业务收入不低于企业总收入的70％；
5. 企业无违规违法经营行为。
*注：经新区投资促进主管部门认定的其它服务外包企业，参照本措施执行。*

**二、特定事项条件**
1. **疏解支持（第四条）**：基本条件参照第三条，且不受注册年限限制。
2. **生态打造支持（第五条）**：基本条件参照第三条，且不受注册年限限制。
3. **成长型服务外包企业（第六条）**：
   - 企业员工达到100人以上；
   - 且服务外包业务年营业收入2000万元以上，或国际服务外包业务年营业收入300万美元以上。
4. **骨干型服务外包企业（第六条）**：
   - 企业员工达到300人以上；
   - 且服务外包业务年营业收入5000万元以上，或国际服务外包业务年营业收入700万美元以上。
5. **龙头型服务外包企业（第六条）**：
   - 企业员工达到500人以上；
   - 且服务外包业务年营业收入1亿元以上，或国际服务外包业务年营业收入1500万美元以上。
6. **人才补贴（第七条）**：高校优秀应届毕业生，入职新区服务外包企业。
7. **技能培训（第八条）**：
   - 纳入目录的：按照《河北省职业技能培训职业（工种）目录及补贴标准（2022版）》规定执行。
   - 未纳入目录的：与生产有直接关联，培训对象与新区服务外包企业签订1年以上劳动合同。
8. **展会活动（第十条）**：参加商务主管部门指定的境内外服务外包（服务贸易）专业展会、投资促进、经贸交流等活动。
9. **公共服务平台（第十一条）**：提供数据储存、信息传输、测试演示、专业认证、孵化培训、管理咨询、金融、法律等服务。

# 办理流程

1. **认定与管理**：对新区范围内服务外包企业，根据吸纳就业、业务实绩、成长潜力等综合情况，每年确定一批成长型、骨干型、龙头型服务外包企业，根据疏解规定，择优纳入疏解体系认定和管理。
2. **协商确定**：对疏解到新区的重点企业、重要项目等重大事项，按照“早来早支持、大干大支持、贡献大支持大”的原则，另行协商确定。
3. **培训认定**：对未纳入目录且与生产有直接关联的培训，经认定给予支持。

# 申报时间

本措施自发布之日起实施，有效期2年。
