# 政策简介

为全面加强雄安新区知识产权保护，凝聚知识产权工作合力，根据《知识产权综合保护联系点工作办法》，开展2021年知识产权综合保护联系点申报工作。通过认定联系点，由新区综合执法局、雄安检察分院定期走访，引导和支持联系点完善知识产权管理体系，提升知识产权保护能力，并提供知识产权合规性审查服务等支持措施。

# 申请对象

在雄安新区范围内登记注册的企事业单位、行业组织、个体工商户及其他社会组织。

# 申报条件

申报对象应同时满足以下条件：

1. **创造优势明显**：申报对象拥有发明专利3件及以上；或拥有海外同族发明专利2件以上；或拥有驰名商标；或属于省级以上非物质文化遗产名录。
2. **工作体系完善**：有较为健全的知识产权管理制度且执行情况良好；有知识产权管理机构；有知识产权工作人员；有知识产权工作经费。
3. **保护能力较强**：近三年无恶意侵犯他人知识产权行为；有知识产权风险预警机制。
4. **信用记录良好**：近三年无严重失信行为。

# 申报材料

1. 通过“雄安新区科创服务区块链平台”在线填报的相关信息。
2. 证明申报条件真实性的证明材料（申报对象应当对填报信息和证明材料的真实性、合法性负责）。

# 办理流程

1. **填报**：申报对象在“雄安新区科创服务区块链平台”(https://innofund.info/)完成填报相关信息，在线提交至所在县市场监督管理局、县人民检察院。
2. **推荐**：县市场监督管理局、县人民检察院联合对申报材料进行初审并择优推荐至新区综合执法局、省人民检察院雄安新区分院。
3. **认定**：新区综合执法局、省人民检察院雄安新区分院根据推荐情况，联合认定2021年知识产权综合保护联系点单位。

# 申报时间

1. **申报对象填报截止时间**：10月9日前。
2. **各县推荐完成截止时间**：10月12日前。
3. **联系人信息报送截止时间**：9月30日前（针对各县市场监督管理局和人民检察院指定工作联系人）。

# 注意事项

1. 申报对象应当对填报信息和证明材料的真实性、合法性负责。
2. 各县市场监督管理局、人民检察院需加强对申报材料的审查把关。
3. 申报名额分配见附件1。
4. 请三县市场监督管理局和人民检察院各指定1名工作联系人，并将联系人信息表发至新区综合执法局。
