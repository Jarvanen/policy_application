# 政策简介

本通知旨在组织开展2026年专精特新“小巨人”企业认定和复核推荐工作。支持方向包括新申请第八批专精特新“小巨人”企业，以及对2023年认定的第五批和复核通过的第二批专精特新“小巨人”企业进行复核。认定标准依据《优质中小企业梯度培育管理办法》（工信部企业〔2026〕2号），复核标准依据2022年印发的《优质中小企业梯度培育管理暂行办法》（工信部企业〔2022〕63号）。

# 申请对象

1. **新申请对象**：有效期内河北省专精特新中小企业。
2. **复核对象**：2023年认定的第五批和复核通过的第二批专精特新“小巨人”企业。

# 申报条件

1. **基础资格**：
   - 新申请企业需符合《优质中小企业梯度培育管理办法》中专精特新“小巨人”企业有关认定标准。
   - 申请复核的“小巨人”企业，需符合2022年印发的《优质中小企业梯度培育管理暂行办法》相关标准和要求。
   - 相关指标需按《办法》附件4中“部分指标和要求说明”严格把握。

2. **排除情形（不予推荐）**：
   - 已成为工业和信息化部制造业单项冠军示范企业或单项冠军产品的企业，不再推荐新申请第八批专精特新“小巨人”企业。
   - 与工业和信息化部已认定的专精特新“小巨人”企业存在控股关系的企业。
   - 同一集团内生产相似主导产品的企业。

3. **合规性要求**：
   - 申报期间未被列入经营异常名录或严重失信主体名单。
   - 提供的产品（服务）不属于国家禁止、限制或淘汰类。
   - 近三年未发生较大生产安全事故、重大网络和数据安全事件、重大环境违法行为、严重质量问题、严重违反相关行业管理规定等情况。

# 申报材料

1. **申请书**：
   - 新申请企业提供《第八批专精特新“小巨人”企业申请书》。
   - 复核企业提供《专精特新“小巨人”企业复核申请书》。
   - 纸质申请书需从工信部官网自行下载填报使用。

2. **营业执照复印件**。

3. **审计报告**：
   - 近三年（2023年-2025年）经会计师事务所审计，且在财政部注册会计师行业统一监管平台完成报备后的财务审计报告。
   - 报告需体现与认定条件相关年度的各项数据，必须体现主营业务收入、主营业务成本两项指标。
   - 如审计报告中不能体现研发经费指标数据，需提供近两年研发经费的专项审计报告。
   - 需将完成报备后的已赋码电子原件上传至优质中小企业梯度培育平台，如不一致将影响申报结果。

4. **产业领域和产业链配套情况说明**：
   - 说明企业主导产品属于制造业核心基础零部件、核心基础元器件、关键软件、先进基础工艺、关键基础材料、产业技术基础，或属于改造提升传统产业、培育壮大新兴产业、布局建设未来产业，位于产业链关键环节，对提升产业链供应链韧性和安全水平发挥重要作用。

5. **河北省经营主体专项信用报告**：
   - 可在河北政务服务网“高效办成一件事”专区“信用代证一件事”办事系统申请，须覆盖全部领域。

6. **其他佐证材料**：
   - 对应认定标准的其他佐证材料。
   - 企业无需提供第三方机构出具的“细分市场占有率证明或说明”，仅需如实说明市场占有率即可。
   - 申报佐证材料各项内容均需加盖企业公章。
   - 企业纸质申请书及佐证材料需装订成册，规范整理并形成目录，列明题目、分项及对应页码。

# 办理流程

1. **线上填报**：
   - 平台：工信部优质中小企业梯度培育平台（https://zjtx.miit.gov.cn）。
   - 时间：2026年4月25日至5月25日。
   - 要求：申请企业应如实、自主填报申请表，并按要求提供有关佐证材料。线下与线上数据应保持一致。

2. **线下报送**：
   - 时间：以企业所在市级工信主管部门要求为准。
   - 雄安新区具体要求：相关材料纸质版请于5月25日下班前报新区工信科技数据局中小企业处。

3. **市级推荐与审核**：
   - 各市、雄安新区工信主管部门组织动员申报，择优推荐新申请企业，确保复核企业应复尽复。
   - 对不推荐复核的企业，需正式出具情况原因说明（盖章，并入推荐文件附件）。
   - 市级主管部门需对企业数据真实性、技术创新性进行审核，确保申请书填报数据与佐证材料一致。

4. **省级报送**：
   - 各市、雄安新区工信主管部门务于2026年5月29日下班前将加盖公章的正式推荐文件及附件（纸质版及可编辑电子版各1份）、《第八批专精特新“小巨人”企业申请书》（含佐证材料，1份）、《专精特新“小巨人”企业复核申请书》（含佐证材料，1份）统一报送省工业和信息化厅。

# 申报时间

- **线上填报时间**：2026年4月25日至5月25日。
- **雄安新区线下报送截止时间**：2026年5月25日下班前。
- **市级主管部门向省厅报送截止时间**：2026年5月29日下班前。

# 注意事项

1. **费用说明**：河北省工业和信息化厅未委托任何机构开展专精特新申请业务，相关申请均不收取任何费用。
2. **数据一致性**：线下报送材料与线上填报数据应保持一致；审计报告电子原件上传平台需与纸质版一致，否则影响申请结果。
3. **造假惩戒**：如发现企业存在申报数据造假，工业和信息化部将取消企业创新型中小企业、专精特新中小企业、专精特新“小巨人”企业等称号，并禁止企业三年内再次申请。涉及骗取财政资金的，将依法依规严肃处理。
4. **复核要求**：对于2023年认定和复核通过的专精特新“小巨人”企业，不推荐复核的，需正式出具情况原因说明（盖章，并入推荐文件附件）。
5. **推荐限额**：对第八批专精特新“小巨人”企业推荐数量较多但通过率较低的市（新区），省工业和信息化厅将按照工业和信息化部有关要求在专精特新相关支持政策中进行减分或限额。
6. **账号准备**：申请企业需提前登录平台确认账号、密码。
