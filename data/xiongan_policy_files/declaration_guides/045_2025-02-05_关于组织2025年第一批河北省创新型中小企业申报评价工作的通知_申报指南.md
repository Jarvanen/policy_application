# 政策简介

为进一步做好新区优质中小企业梯度培育工作，组织开展2025年第一批河北省创新型中小企业申报评价工作。

# 申请对象

参评企业（科技类）。

# 申报条件

1. 企业需登录工业和信息化部优质中小企业梯度培育平台（https://zjtx.miit.gov.cn）进行注册登记。
2. 企业需在平台进行填报自评，且自评通过。

# 申报材料

1. 《河北省创新型中小企业自评表》（自评通过的企业下载打印）。
2. 必要证明资料（通过平台上传）。
3. 申报书。

# 办理流程

1. **平台注册与自评**：参评企业登录工业和信息化部优质中小企业梯度培育平台（https://zjtx.miit.gov.cn）注册登记后进行填报自评。
2. **材料准备与提交**：自评通过的企业下载打印《河北省创新型中小企业自评表》，提供（平台上传）必要证明资料，形成申报书向所在地工业和信息化主管部门申请评价。
3. **县级审核评价**：三县工信主管部门完成本县域第一批创新型中小企业审核评价工作。
4. **材料报送**：将审批备案报告（含县级审核评价工作情况）、建议通过评价企业名单和备案信息汇总表等材料（各一份，加盖县级工信主管部门公章）正式报送新区工信科技数据局。

# 申报时间

2025年2月20日前（三县工信主管部门完成审核评价工作并报送材料）。
