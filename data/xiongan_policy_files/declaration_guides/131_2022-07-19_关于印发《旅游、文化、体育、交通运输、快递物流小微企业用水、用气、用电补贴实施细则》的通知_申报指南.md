# 政策简介

为贯彻落实河北雄安新区党工委管委会党政办公室印发《关于应对疫情支持服务业领域纾困发展的若干措施》的通知要求，针对旅游、文化、体育、交通运输、快递物流类小微企业，支持企业渡过难关，切实降低在新区注册的旅游、文化、体育、交通运输、快递物流类小微企业运营成本，制定本细则。本细则涉及行业参照《国民经济行业分类》(GB/T4754—2017)《国家旅游及相关产业统计分类(2018)》。本细则所涉及小微企业参照《关于印发中小企业划型标准规定的通知》(工信部联企业〔2011〕300号)。

# 奖励额度/补贴标准

1. 补贴2022年1月至6月的水、电、气费用，最高6个月。
2. 对符合条件的小微企业用水、用电、用气运营费用实行全额补贴。
3. 每家企业最高补贴5万元。

# 注意事项

1. **用气范围限制**：本细则所指用气补贴仅针对管道燃气，不包括加气、煤气罐等情况。
2. **真实性承诺**：坚持企业自愿申报，企业承担主体责任，企业须出具真实性承诺，企业法定代表人对申报材料和相关数据的真实性负责。
3. **违规处罚**：如有伪造证件材料、虚报冒领等欺瞒行为，自愿接受追回补贴资金、罚款、媒体曝光及移交司法机关处理等处罚。
4. **数字货币发放**：审核通过的，通过产业互联网平台和财政对公钱包将数字货币发放至企业对公钱包。
5. **政策有效期**：本办法自发布之日起实施，到2022年8月26日失效。

# 申请对象

1. 在新区行政区域内依法注册，具有独立法人资格，主营业务为旅游、文化、体育、交通运输、快递物流类小微企业。
2. 在新区行政区域内依法注册、在新区纳税的快递行业分公司。

# 申报条件

1. 在新区行政区域内依法注册，具有独立法人资格，主营业务为旅游、文化、体育、交通运输、快递物流类小微企业，以及在新区行政区域内依法注册、在新区纳税的快递行业分公司。
2. 申请企业需要在新区2022年1月至6月开展正常经营活动，依法依规缴纳税收，且符合行政主管部门要求。

# 申报材料

1. 《水电气运营补贴申请表》(见附件2)。
2. 企业营业执照及法定代表人身份证。
3. 企业在2022年1月至6月用水、用气、用电支付凭证(收费机构提供的合法发票或加盖公章的缴费清单、物业及村集体等代收单位提供的加盖公章的收费凭证或申请企业缴费记录)。
4. 企业可支持接收数字货币的对公钱包。
5. 2022年6月1日前纳税申报记录。
6. 与申报有关的其他证明材料。

# 办理流程

1. **申报**：申报主体需通过新区产业互联网平台进行申报(产业互联网平台网址http:/120.53.124.140:10001)，填写《申请表》并上传相关认定材料。
2. **受理**：产业互联网平台对申报材料与内容完备性进行初筛，材料齐全的予以提交受理；不齐全的，系统一次性告知原因并无法进行提交。申请单位在2个工作日内补充申报材料与内容完备的，予以提交受理；逾期未补充申报或补充申报材料与内容仍不完备的，不予受理。
3. **审核**：核查企业提供相关材料信息，必要时可通过电话、上门、委托第三方机构等方式予以核实认定。(三县文化、旅游、体育、交通、发改等有关部门按照行业管理职责分工负责)
4. **公示**：对审核通过的，在产业互联网平台进行公示，公示时间不少于5日。公示结束，三县相关部门将《申请表》《汇总表》(见附件3)盖章后报新区改革发展局。(三县文化、旅游、体育、交通、发改等有关部门按照行业管理职责分工负责)
5. **汇总**：新区改革发展局对三县资金需求进行审核汇总，并组织相关部门抽查企业运营情况。(新区改革发展局牵头，宣传网信局、规划建设局分工负责)
6. **拨付**：新区财政部门对纾困补贴报账资料进行审核。审核通过的，通过产业互联网平台和财政对公钱包将数字货币发放至企业对公钱包。审核未通过的，退回修改完善后办理数字货币发放。(新区改革发展局负责)

# 申报时间

1. **申报日期**：2022年7月22日至8月10日。
2. **发放日期**：2022年8月25日前，对申报企业完成组织审核和资金拨付工作。
