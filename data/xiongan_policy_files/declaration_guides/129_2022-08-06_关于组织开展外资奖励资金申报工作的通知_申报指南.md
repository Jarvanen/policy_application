# 政策简介

为认真贯彻落实雄安新区管理委员会印发的《关于印发扎实稳定雄安新区经济运行的一揽子措施及配套政策的通知》要求，组织开展《关于促进外贸外资高质量发展的三十二条政策措施》中有关外资奖励资金申报工作。旨在支持在新区依法注册设立或办理增资手续、具有独立法人资格、已正常开展生产经营活动的外商投资企业，涵盖外商投资企业设立、增资、并购、股权转让增资、境外跨国公司总部及功能性总部、世界500强及龙头企业外资研发机构及重大项目等支持方向。

# 奖励额度/补贴标准

1. **外商投资企业设立奖励**：对当年实缴外方注册资本超过1000万美元(含)的外商投资企业，按其当年实缴外方注册资本总额不超过1%(含)的比例给予奖励，最高奖励500万元，分3年按40%、30%、30%的比例发放。
2. **外商投资企业增资奖励**：对当年实缴新增外方注册资本超过1000万美元(含)的现有外商投资企业，按其当年实缴新增外方注册资本总额不超过1%(含)的比例给予奖励，最高奖励500万元，分3年按40%、30%、30%的比例发放。
3. **外商投资并购奖励**：对在新区以并购方式设立、并购交易资金的收款账户在新区开立且当年实际到位外资(并购交易资金)超过1000万美元(含)的外商投资企业，按其当年实际到位外资金额不超过1%(含)的比例给与奖励，最高奖励500万元，分3年按40%、30%、30%的比例发放。
4. **外资股权转让增资奖励**：对在新区以股权转让方式增资、股权转让交易资金的收款账户在新区开立且当年实际到位外资(股权转让交易资金)超过1000万美元(含)的现有外商投资企业，按其当年实际到位外资金额不超过1%(含)的比例给与奖励，最高奖励500万元，分3年按40%、30%、30%的比例发放。
5. **境外跨国公司总部、地区总部和功能性总部奖励**：对当年实缴外方注册资本超过500万美元(含)经省级认定的境外跨国公司总部、地区总部和功能性总部，按其当年实缴外方注册资本总额不超过1%(含)的比例给与奖励，最高奖励500万元，分3年按40%、30%、30%的比例发放。
6. **世界500强、龙头企业外资研发机构奖励**：对境外世界500强企业(以上年度《财富》排行榜为准)、全球行业龙头企业在新区设立的具有独立法人资格的外资研发机构，享受省级“一项目一议”资金支持的，按照省级奖励资金给予50%的配套奖励，分3年按40%、30%、30%的比例发放。
7. **世界500强、龙头企业重大项目奖励**：对境外世界500强企业、全球行业龙头企业在新区设立，且当年实缴外方注册资本超过3000万美元(含3000万美元)的投资项目，按照重大项目“一事一议”原则给予重点支持。

# 注意事项

1. **申报期限**：每半年开展一次集中申报工作。上半年申报所需数据和材料的截止时间为当年6月30日，申报日期截至当年8月底；下半年申报所需数据和材料的截止时间为当年12月31日，申报日期截至次年2月底。
2. **诚信承诺**：获奖后10年不迁离雄安新区、不改变在新区的纳税义务、不减少注册资本。若违反承诺，将主动退还已经获得的奖励资金。
3. **违规处理**：对以虚报、冒领、伪造、关联交易等手段骗取财政资金的行为，一经查实，将收回已下达的资金，并取消相关企业5年的申报资格。
4. **材料要求**：申报材料应当真实、完整、准确。各申报企业网上注册时需填写准确完整的企业信息，项目申报前需确保申报系统相关信息已更新完善。
5. **公示要求**：申报企业、申报项目、申报金额和审核意见等信息在新区产业互联网平台上公示7个工作日(涉密事项除外)。

# 申请对象

符合《关于促进外贸外资高质量发展的三十二条政策措施》中《外商投资企业奖励细则》规定的在新区依法注册设立或办理增资手续、具有独立法人资格、已正常开展生产经营活动，近五年内无重大违法违规纪录或严重失信行为的外商投资企业。

# 申报条件

1. 在新区依法注册设立或办理增资手续。
2. 具有独立法人资格。
3. 已正常开展生产经营活动。
4. 近五年内无重大违法违规纪录或严重失信行为。
5. 具体金额门槛：
   - 设立/增资/并购/股权转让增资奖励：当年实缴外方注册资本或实际到位外资超过1000万美元(含)。
   - 境外跨国公司总部等：当年实缴外方注册资本超过500万美元(含)且经省级认定。
   - 世界500强/龙头企业重大项目：当年实缴外方注册资本超过3000万美元(含)。

# 申报材料

1. 申报表(附件1：雄安新区外商投资企业奖励资金申报表)。
2. 承诺书(附件2：雄安新区外商投资企业奖励申报承诺书)。
3. 企业营业执照。
4. 外商投资企业批准证书、备案回执或外商投资信息报告回执。
5. 第三方专业机构出具的资金到位情况验资报告。
6. 省级认定为境外跨国公司总部、地区总部和功能性总部的相关证明材料(限于境外跨国公司总部、地区总部和功能性总部奖励项目)。
7. 享受省级“一项目一议”资金支持的相关证明材料(限于世界500强、全球行业龙头企业外资研发机构奖励项目)。

# 办理流程

1. **申报**：
   - 各申报企业在规定的申报期限内登陆“雄安新区产业互联网平台”(http://120.53.124.140:10001/)选择“慧企政策—商贸服务”进行申报(未在该平台注册的先注册再申报)，并按系统要求上传相关材料。
2. **审核**：
   - 申报主体为三县企业的，由三县商务主管部门通过新区产业互联网平台对企业申报材料进行初审，并将初审结果以书面形式报雄安自贸试验区管委会复审。
   - 申报主体为新区本级企业的，由雄安自贸试验区管委会通过新区产业互联网平台对企业申报材料进行审核。
3. **公示**：
   - 申报企业、申报项目、申报金额和审核意见等信息在新区产业互联网平台上公示7个工作日(涉密事项除外)。

# 申报时间

每半年开展一次集中申报工作：
- 上半年：数据材料截止当年6月30日，申报日期截至当年8月底。
- 下半年：数据材料截止当年12月31日，申报日期截至次年2月底。
