# 政策简介

为加快新区科技型企业创新发展，有序落实惠企政策，雄安新区改革发展局组织开展在孵企业办公用房租金补贴（以下简称“房租补贴”）的第三批线上申报。该政策主要支持孵化器内的在孵企业，以当年在新区办公的正式在职员工人数为基数，按照人均10平方米办公场所，给予办公用房租金补贴。

# 奖励额度/补贴标准

以当年在新区办公的正式在职员工人数为基数，按照人均10平方米办公场所，给予办公用房租金1.5元/平方米/天补贴。

# 注意事项

1. 请在孵企业如实填报申请材料，孵化器及三县科技管理部门严格把关。
2. 雄安新区改革发展局对不符合申报条件的、入孵协议不规范等情况不予奖补。
3. 对弄虚作假、伪造材料等情况不予奖补，并取消其相关条款的申报资格。

# 申请对象

申报企业须是国家级、省级或新区认定的科技企业孵化器内的在孵企业。

# 申报条件

1. 主要从事新技术、新产品的研发、生产和服务，并满足科技型中小企业相关要求；
2. 企业注册地和主要研发、办公场所须在本孵化器场地内，入驻时成立时间不超过24个月；
3. 孵化时限原则上不超过48个月，技术领域为生物医药、现代农业、集成电路的企业，孵化时限不超过60个月。

# 申报材料

1. 企业基本信息；
2. 企业租房信息，需上传入孵协议扫描件；
3. 企业财务信息，需上传企业审计报告或财务报表、纳税证明，如无需上传说明；
4. 企业员工信息，需上传员工合同、社保证明，如无需上传说明；
5. 项目申报承诺书。

# 办理流程

1. **线上申报**：房租补贴通过雄安新区产业互联网平台（http://www.xaiip.org.cn）进行线上申报，路径为“首页—企业公共服务—惠企政策—雄安新区第三批在孵企业办公用房租金补贴”。各在孵企业、孵化器、三县科技管理部门使用原账号密码登录即可。
2. **孵化器审核**：各孵化器对在孵企业的申报材料及承诺书进行审核，信息核实无误后在平台审核通过，下载填写补贴核查表和核查说明，6月30日前盖章报送至所在县科技管理部门。
3. **县级审核**：县科技管理部门对照补贴核查表在线审核申报材料，信息核实无误后在平台审核通过，出具补贴核查表和核查说明，7月14日前盖章报送至雄安新区改革发展局。

# 申报时间

1. 第三批申报周期为2022年4月1日至2023年3月31日。
2. 在孵企业自2023年6月5日开始申报。
3. 所在孵化器审核推荐时间截至2023年6月30日。
4. 所在县科技管理部门审核推荐时间截至2023年7月14日。
