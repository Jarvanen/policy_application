# 政策简介

为深入贯彻落实国家创新驱动发展战略及2025年河北省人民政府工作报告精神，聚力发展新质生产力，推进新型工业化，积极借力全国高校、科研院所创新资源，持续加快创新成果在河北转化应用，不断提升企业创新能力和核心竞争力，推动传统产业转型升级，助力制造强省建设。2025年将组织开展创新成果转化“双百工程”系列对接活动，现就企业产学研合作需求征集有关事项通知如下。

# 申请对象

我省现代化钢铁、绿色化工、生物医药、电子信息、新能源和智能网联汽车、机器人、空天信息和卫星互联网、数字经济等 8 大产业及 107 个重点产业集群内（详见附件 1），建立研发机构的工业企业和科技性服务类企业。

# 申报条件

1. 企业需属于现代化钢铁、绿色化工、生物医药、电子信息、新能源和智能网联汽车、机器人、空天信息和卫星互联网、数字经济等 8 大产业及 107 个重点产业集群内。
2. 企业需建立研发机构。
3. 企业类型为工业企业或科技性服务类企业。

# 申报材料

1. 企业产学研合作需求表（盖章扫描件、电子版）。
2. 需求表内容需包含：
   - 需求名称、投资预算。
   - 描述产学研合作需求（包括目前生产状况、遇到的技术难题内容、需要达到的技术指标、技术需求、人才需求等，内容请尽可能详细）。
   - 行业分类、技术领域、所属县区。
   - 拟合作方式、意向合作单位（有/无）、意向合作专家（有/无）。
   - 填报人、手机、电子邮箱。
   - 企业名称及简介。
   - 已开展产学研合作情况（可附页，包括合作单位名称、项目名称、项目起止时间、进展及成效等）。
   - 其他需求（中试平台合作、科技成果评价、科技奖励、院士(中/外籍)工作站、产学研协同创新平台建设、科技支撑项目等）。

# 办理流程

1. **组织填报**：各市（含定州、辛集市）工业和信息化局、雄安新区工信科技数据局组织企业积极填报需求。需求填报要详细、具体、明确，确保后期活动能够精准对接。
2. **汇总报送**：
   - **雄安新区**：雄县、容城县、安新县工信科技数据局于4月28日12:00前，将有关材料报送河北雄安新区工信科技数据局（电子版发送至邮箱）。
   - **各市**：于2025年5月1日前，将企业产学研合作需求表（盖章扫描件、电子版）汇总报送至河北省产学研合作促进会。
3. **择优对接**：省厅将从企业报送的需求中择优选择，分批次组织企业走入京津冀、长三角地区高校、科研机构及工信部部属高校进行对接，力促达成一批合作意向和合作项目。拟定5月份组织第一批对接，具体活动安排另行通知。
4. **活动总结**：本地区组织开展的创新成果转化“双百工程”对接活动，请各市（含定州、辛集市）工业和信息化局、雄安工信科技数据局在活动结束后及时向省厅科技处报送活动总结。

# 申报时间

1. **雄安新区各县**：2025年4月28日12:00前。
2. **各市（含定州、辛集市）**：2025年5月1日前。

# 注意事项

1. **数量要求**：每个县报送企业数量不少于5家；每个集群填报企业数不少于5家。
2. **重点指导**：指导产业集群企业，特别是龙头和领军企业，认真梳理和填报需求。
3. **咨询渠道**：高校和科研机构的科技成果、填报注意事项可咨询河北省产学研合作促进会。
4. **材料格式**：报送材料需包含盖章扫描件和电子版。
