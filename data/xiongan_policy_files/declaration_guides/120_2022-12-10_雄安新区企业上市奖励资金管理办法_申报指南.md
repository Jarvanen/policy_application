# 政策简介

本办法旨在贯彻落实《关于印发扎实稳定雄安新区经济运行的一揽子措施及配套政策的通知》中关于企业上市的有关政策措施，规范新区企业上市奖励资金使用管理，提高财政资金使用效益。奖励资金是指雄安新区本级财政预算安排的用于培育和支持新区企业上市的、符合《配套政策》有关规定的奖励资金。

# 奖励额度/补贴标准

1. **境内A股上市企业**：
   - 进入辅导期并在河北证监局完成备案后，奖励100万元；
   - 向中国证监会或沪、深、京交易所提交上市申请并获受理后，奖励200万元。
2. **境外主要证券交易所上市企业**：
   - 经具有批准权限的部门或机构正式受理，奖励200万元；
   - 通过审核后，奖励100万元。
3. **迁入新区的区外上市企业或通过借壳(买壳)上市并将壳公司注册地迁至新区的企业**：
   - 给予300万元奖励。

# 注意事项

1. **申请时效**：申请主体在符合奖励申请政策当日起一年内申请有效，逾期未申请的不予追溯。2020年1月1日以来符合条件的企业首次申请截止时间为2023年3月31日。
2. **承诺事项**：申请企业需承诺自申请之日起5年内不迁离雄安新区。如在承诺期限内提前迁出，需全额退还已申请上市奖励资金。
3. **重复享受限制**：本次申请奖励资金项目与新区其他财政扶企政策不存在交叉，未重复申请。
4. **材料真实性**：申请材料必须真实、准确、可靠，若存在虚假、伪造等违法违规情况，需交还财政资金并承担相应法律责任。
5. **公示期**：审核意见及奖励资金信息公示期为7个工作日。

# 申请对象

1. 2020年1月1日（含）后在新区拟上市或完成上市企业；
2. 2020年1月1日后迁入新区的区外上市企业；
3. 2020年1月1日后通过借壳(买壳)上市并将壳公司注册地迁至新区的企业。

# 申报条件

1. 须在雄安新区完成工商注册和税务登记，拥有独立法人资格；
2. 依法在雄安新区开展实际经营活动，服务新区建设发展；
3. 申请之日的前三年中，不得存在相关行政主管部门认定的严重违法违规失信行为，且未被相关部门实行惩戒措施。

# 申报材料

1. **通用材料**：
   - 雄安新区企业上市奖励申请表；
   - 承诺书。
2. **境内A股上市企业**：
   - 通过辅导验收的：河北证监局出具的验收工作完成函等相关证明文件；
   - 上市申请获得受理的：在上述文件基础上提供中国证监会或沪、深、京交易所出具的上市申请受理证明文件。
3. **境外主要证券交易所首次公开发行股票并上市企业或境外上市后回归A股的企业**：
   - 中国证监会同意企业赴境外上市相关证明文件；
   - 境外证券交易所同意企业上市的函件；
   - 证券简称和证券代码证明文件；
   - 境外上市后回归A股的：还需提供企业成功回归A股并上市的证明文件。
4. **2020年后迁入的上市企业**：
   - 国家证监部门批准上市的有关文件；
   - 证券简称和证券代码证明文件；
   - 企业税务变更等相关证明文件；
   - 借壳(或买壳)上市的企业：还需提供股权交易证明文件。

# 办理流程

1. **申报**：符合奖励政策的申请主体通过新区产业互联网平台自主填写申报材料，并将有关纸质版申报材料按要求报送新区改发局。
2. **审核**：新区改发局每月月末会同相关部门对提交的申报材料进行审核，必要时可委托中介机构、组织专家或内部评审等方式对申报材料进行评审。
3. **公示**：将审核意见、申请主体奖励资金信息等情况进行公示，公示期为7个工作日。对公示有异议的项目，由新区改发局提出处理意见。
4. **拨付**：审核通过并公示无异议后，办理资金拨付。

# 申报时间

- 申请主体在符合奖励申请政策当日起一年内申请有效。
- 2020年1月1日以来符合条件的企业首次申请截止时间为2023年3月31日。
