# 政策简介

本细则旨在支持雄安新区传统产业抗疫纾困转型升级，对符合条件的制造业企业在新区范围内银行机构申请的销售订单、应收账款等相关流动资金贷款给予贴息支持。传统产业指2017年4月1日前在新区三县范围内注册并依法纳税的制造业法人企业（国家标准《国民经济行业分类》（GB/T4754-2017）中，行业代码前两位为13-43的企业）。贷款贴息按照“先付后贴”、“免申即享”的方式实施，即企业按合同约定支付银行机构利息后，新区政府定期对符合条件的企业贴息。以个人名义贷款（包括企业法定代表人及配偶、企业主要股东、实际控制人等）用于经营的不列入贴息范围。

# 奖励额度/补贴标准

1. **贴息利率**：按照贷款合同签订当日执行的人民银行公布的一年期贷款市场报价利率（LPR）予以贴息；实际贷款利率低于LPR的，按照实际贷款利率予以贴息。
2. **贴息期限**：最多不超过2年。
3. **贴息总额**：每户企业贴息总额不超过100万元。
4. **适用时间段**：2022年2月22日至2023年12月31日新发生的流动资金贷款（以提款日期为准）。

# 注意事项

1. **排他性规定**：各家企业同时只可享受一项贷款贴息政策。
2. **资金拨付限制**：贴息资金在预算安排总额下，视贴息进度和资金使用情况，新区改发局有权顺延贴息资金拨付或暂停、终止贴息。
3. **主体要求**：申报企业应守法经营，履行安全生产和环境保护责任，未被相关行政主管部门列入失信联合惩戒名单，信用记录良好。企业承担主体责任，企业法定代表人对申报材料和相关数据的真实性负责。
4. **公示要求**：拟贴息情况需在新区产业互联网平台上进行公示，不少于5日。
5. **政策有效期**：本细则有效期至2023年12月31日。

# 申请对象

2017年4月1日前在新区三县范围内注册并依法纳税的制造业法人企业（国家标准《国民经济行业分类》（GB/T4754-2017）中，行业代码前两位为13-43的企业）。

# 申报条件

1. 企业需在新区范围内银行机构申请销售订单、应收账款等相关流动资金贷款。
2. 贷款发生时间在2022年2月22日至2023年12月31日之间（以提款日期为准）。
3. 企业已按合同约定支付银行机构利息。
4. 企业未被相关行政主管部门列入失信联合惩戒名单，信用记录良好。
5. 企业需登录雄安新区产业互联网平台完善贷款信息并上传贷款合同等有关材料。

# 申报材料

1. 贷款合同等有关材料（通过雄安新区产业互联网平台上传）。
2. 贷款贴息明细表（附件1，由对应贷款银行出具）。
3. 付息有关证明材料（由对应贷款银行出具）。
4. 《贷款贴息资金申请表》（附件2，由三县审核部门填写）。
5. 综合审计报告（由第三方审计机构出具）。

# 办理流程

1. **完善信息**：企业主体登录雄安新区产业互联网平台（http://www.xaiip.org.cn），通过“用户中心”-“企业数据账户”-“贷款信息”或“惠企政策”-“传统产业贷款贴息”自主完善贷款信息，并将贷款合同等有关材料进行上传。完成该项信息录入后，企业无需再报送其他申报材料。
2. **银行核准**：各银行定期对企业客户填写信息进行核对，核对无误的，提交产业互联网平台推送至三县审核部门进行审核；核准有错误的，请企业客户重新填写后提交。
3. **三县审核**：三县审核部门根据企业贷款信息协请对应贷款银行出具贷款贴息明细表、付息有关证明等材料，并聘请第三方会计师事务所开展审计工作。
4. **汇总及公示**：三县审核部门定期对本县审核通过的企业拟贴息情况进行汇总，填写《贷款贴息资金申请表》，请第三方审计机构出具综合审计报告，报县政府研究同意，同步在新区产业互联网平台上进行公示（不少于5日）。
5. **资金拨付**：三县审核部门根据县政府研究情况，形成明确书面意见，以县政府名义向新区改发局提起贴息申请，后附《贷款贴息资金申请表》、综合审计报告等有关材料，新区改发局按程序提起资金拨付申请，完成资金拨付。

# 申报时间

2022年2月22日至2023年12月31日（新发生流动资金贷款的提款日期范围）。
