# 政策简介

为深入贯彻落实《雄安新区工业诊断力量培育实施方案（2023-2025年）》工作部署，加快培育立足新区、服务河北、辐射京津的工业诊断力量，结合雄安新区实际，特制定《关于支持工业诊断力量培育的五条措施》。政策主要支持方向包括打造雄安新区工业诊断服务平台、培育壮大“工业大夫”队伍、建立完善“工业诊所”体系、支持创建“工业医院”以及支持企业购买工业诊断服务。

# 奖励额度/补贴标准

1. **工业诊断服务平台**：给予经认定的雄安新区工业诊断服务平台一次性30万元专项经费支持。
2. **工业诊所**：
   - 对成功创建新区级工业诊所的机构，给予10万元一次性奖励；
   - 对成功创建省级工业诊所的机构，给予20万元一次性奖励。
3. **工业医院**：
   - 对经认定的新区级“工业医院”的机构，给予10万元一次性奖励；
   - 对经认定的省级“工业医院”的机构，给予20万元一次性奖励。
4. **企业购买工业诊断服务**：
   - 支持标准为投入资金实际发生额的50%，最高不超过10万元。
   - 支持符合条件的工业诊断服务项目申报“百项转型”示范项目，按现行政策予以奖励。

# 注意事项

1. **有效期**：本政策有效期至2025年12月31日。
2. **解释权**：具体解释工作由雄安新区改革发展局承担。
3. **人才政策衔接**：建立“工业大夫”专家资源库，优先推荐符合条件的入库专家申报雄才卡、燕赵英才服务卡等各项人才政策。

# 申请对象

1. 经认定的雄安新区工业诊断服务平台。
2. 成功创建新区级或省级工业诊所的机构。
3. 经认定的新区级或省级“工业医院”的机构。
4. 采用工业诊断报告并产生科技创新、绿色低碳、制造提升、品牌设计、质量管理、投资营销等领域实际投资的工业企业。
5. 符合条件的“工业大夫”入库专家。

# 申报条件

1. **工业诊断服务平台**：需经认定为雄安新区工业诊断服务平台。
2. **工业诊所**：需成功创建新区级或省级工业诊所。
3. **工业医院**：需经认定为新区级或省级“工业医院”。
4. **企业购买服务**：
   - 主体为工业企业；
   - 采用工业诊断报告；
   - 产生科技创新、绿色低碳、制造提升、品牌设计、质量管理、投资营销等领域的实际投资。
5. **工业大夫**：需符合“工业大夫”专家资源库入库条件。
