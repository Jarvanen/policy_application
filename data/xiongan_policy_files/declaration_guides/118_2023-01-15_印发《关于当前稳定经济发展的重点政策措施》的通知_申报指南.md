# 政策简介

为深入贯彻落实党的二十大精神和中央经济工作会议要求，高效统筹疫情防控和经济社会发展，巩固已有稳定经济发展政策成果，持续帮助市场主体纾困发展，加快推动企业复工复产，提振市场信心，推动经济回稳向好，结合新区实际，制定《关于当前稳定经济发展的重点政策措施》。政策涵盖加快推进项目建设、加大支持促消费力度、促进工业企业复工复产、降低企业经营成本、促进文旅行业恢复发展、延长贷款还款期限、加大贷款贴息支持力度、支持企业吸纳就业、支持扩大外贸企业进出口规模、鼓励企业参加境内外重点会展等方面。

# 奖励额度/补贴标准

**一、项目建设管理费奖励**
对列入2023年新区本级政府投资的新开工项目，经认定：
- 2023年3月31日前开工建设的，额外奖励40%的项目建设管理费；
- 6月30日前开工建设的，额外奖励20%的项目建设管理费。

**二、促消费资金安排**
- 统筹5000万资金，组织开展“购享雄安”活动，面向家电、家具、餐饮、商超等重点领域进行消费补贴；
- 安排不低于100万资金，开展汽车促消费活动专场，扩大对新能源汽车购车支持力度；
- 安排不低于100万资金，促进电商经济发展，组织电商促消费活动专场，对销售额排名靠前的电商企业给予奖励；
- 安排100万资金，春节期间开展年货消费补贴。

**三、工业企业复工复产支持**
- 符合“小升规”“专精特新”“百项转型”奖励支持政策企业，免申即享、即享即补。
- 对2023年上半年产值达到1亿元且同比增长10%以上的新区规模以上工业企业，给予半年产值0.2‰的一次性物流补贴。
- 鼓励企业改建N95口罩、四类药品、抗原试剂等重点防疫物资生产线，符合条件的可在“百项转型”工程补助基础上额外提高不超过10%补助比例。

**四、降低企业经营成本**
- 对新区符合条件的服务业小微企业和个体工商户承租国有资产类经营用房，减免全年6个月房租；
- 对于用水、用气、用电价格方面给予一定比例的运营补贴，单家企业最高1万。

**五、文旅行业奖励与补贴**
- 对新获评的4A级以上景区、省级旅游度假区、省级夜间文化和旅游消费集聚区、省级文化产业示范园区、省级文明旅游示范单位、省级文化产业赋能乡村振兴“五个100”分别奖励30万元；
- 对新获评的省级旅游休闲街区、四星级酒店、国家丙级以上旅游民宿等分别奖励10万元（同一单位多个荣誉的仅可获得一次奖励）；
- 对经认定经营时间长、产品丰富的景区(点)给予最高不超过20万元的补贴。

**六、贷款贴息支持**
- 对新区范围内获得银行等金融机构新增贷款的传统产业企业、服务业中小微企业，按照不高于贷款合同签订当日执行的人民银行公布的一年期贷款市场报价利率（LPR）予以贴息，每户传统产业企业、服务业中小微企业贴息总额不超过100万、30万；
- 对新区范围内获得银行等金融机构新增贷款的科技入库企业，按照贷款合同签订实际贷款利率的50%予以贴息，每户企业贴息总额不超过100万元。

**七、吸纳就业补贴**
- 新区注册服务型企业在新招用雄安新区三县户籍人员（以签订劳动合同时间为准），在雄安新区产业互联网平台进行注册并经新区人社部门认定的，补贴标准1000元/人；
- 新区注册的其他类型企业和参建企业（含劳务分包企业），补贴标准500元/人；
- 人力资源服务企业每成功介绍一名新区三县户籍人员成功就业（在新区依法缴纳社会保险费6个月以上），给予500元的职业介绍补贴，每家人力资源服务企业每年最高可申领20万元；
- 对吸纳2年择业期内全日制大专以上未就业高校毕业生参加就业见习并按月发放基本生活费的见习单位，给予最长不超过12个月的就业见习补贴，补贴标准按照新区最低工资标准执行；
- 对吸纳16~24岁失业青年参加见习的，补贴标准按照最低工资标准的60%执行；
- 对见习后留用率超过50%的，见习补贴标准提高10%。

**八、外贸进出口补贴**
- 对年出口额较上一年度增长超过10万美元的企业，增长部分每1美元补贴0.04元，单个企业每年最高补贴金额不超过100万元；
- 对列入国家鼓励进口技术和产品目录中的先进技术、关键设备和零部件年度进口额达到500万美元以上的企业给予补贴10万元，每增加100万美元增加补贴5万元，单个企业每年最高补贴金额不超过50万元。

**九、参加会展补贴**
- 对参加经新区商务部门确定的境内重点线下展会的外贸企业，给予展位费用最高80%补贴，每家企业单个展会最高补贴不超过2万元；
- 对参加河北省商务厅组织的境外重点线下展会（含代参展）的企业，给予展位费用最高40%的补贴；
- 对参加经新区商务部门确定的境外重点线下展会（含代参展）的企业，给予展位费用最高60%、展品运输费用最高50%补贴，每家企业单个展会最高补贴不超过5万元；
- 每家企业参加境内外展会每年最高补贴不超过20万元。

# 注意事项

1. **执行期限**：有明确时限的，按其规定时限执行；无明确时限的，执行至2023年12月31日。
2. **政策衔接**：本政策措施执行中国家、省有相关规定变化调整的，从其规定。
3. **享受原则**：与其他已出台政策按照就高不重复享受原则执行。
4. **文旅奖励限制**：同一单位多个荣誉的仅可获得一次奖励。

# 申请对象

1. **项目建设**：列入2023年新区本级政府投资的新开工项目。
2. **促消费**：家电、家具、餐饮、商超等重点领域消费者；新能源汽车购车者；电商企业；年货消费者。
3. **工业企业**：符合“小升规”“专精特新”“百项转型”奖励支持政策企业；2023年上半年产值达到1亿元且同比增长10%以上的新区规模以上工业企业；改建N95口罩、四类药品、抗原试剂等重点防疫物资生产线的企业。
4. **服务业及个体户**：新区符合条件的服务业小微企业和个体工商户承租国有资产类经营用房。
5. **文旅行业**：新获评的4A级以上景区、省级旅游度假区、省级夜间文化和旅游消费集聚区、省级文化产业示范园区、省级文明旅游示范单位、省级文化产业赋能乡村振兴“五个100”、省级旅游休闲街区、四星级酒店、国家丙级以上旅游民宿等；经认定经营时间长、产品丰富的景区(点)。
6. **贷款延期**：2022年第4季度到期的、因疫情影响暂时遇困的小微企业贷款(含个体工商户和小微企业主经营性贷款)借款人。
7. **贷款贴息**：新区范围内获得银行等金融机构新增贷款的传统产业企业、服务业中小微企业、科技入库企业。
8. **吸纳就业**：新区注册服务型企业、其他类型企业和参建企业（含劳务分包企业）、人力资源服务企业、见习单位。
9. **外贸企业**：年出口额较上一年度增长超过10万美元的企业；年度进口额达到500万美元以上的企业；参加经新区商务部门确定的境内重点线下展会、河北省商务厅组织的境外重点线下展会、经新区商务部门确定的境外重点线下展会的外贸企业。

# 申报条件

1. **项目建设**：列入2023年新区本级政府投资的新开工项目，经认定在2023年3月31日前或6月30日前开工建设。
2. **工业企业**：
   - 符合“小升规”“专精特新”“百项转型”奖励支持政策；
   - 2023年上半年产值达到1亿元且同比增长10%以上；
   - 改建N95口罩、四类药品、抗原试剂等重点防疫物资生产线且符合条件。
3. **服务业及个体户**：新区符合条件的服务业小微企业和个体工商户，承租国有资产类经营用房。
4. **文旅行业**：
   - 新获评相应等级或级别的景区、度假区、集聚区、园区、单位、街区、酒店、民宿等；
   - 经认定经营时间长、产品丰富的景区(点)。
5. **贷款延期**：2022年第4季度到期，因疫情影响暂时遇困的小微企业贷款(含个体工商户和小微企业主经营性贷款)。
6. **贷款贴息**：新区范围内获得银行等金融机构新增贷款。
7. **吸纳就业**：
   - 新区注册服务型企业新招用雄安新区三县户籍人员（以签订劳动合同时间为准），在雄安新区产业互联网平台进行注册并经新区人社部门认定；
   - 人力资源服务企业成功介绍新区三县户籍人员成功就业（在新区依法缴纳社会保险费6个月以上）；
   - 见习单位吸纳2年择业期内全日制大专以上未就业高校毕业生或16~24岁失业青年参加就业见习并按月发放基本生活费。
8. **外贸企业**：
   - 年出口额较上一年度增长超过10万美元；
   - 列入国家鼓励进口技术和产品目录中的先进技术、关键设备和零部件年度进口额达到500万美元以上；
   - 参加经新区商务部门确定的境内重点线下展会、河北省商务厅组织的境外重点线下展会（含代参展）、经新区商务部门确定的境外重点线下展会（含代参展）。
