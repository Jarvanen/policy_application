# 政策简介

为促进新区会展业高质量发展，推进上下游产业链融合发展，结合新区实际，制定本措施。主要支持方向包括支持举办展览项目（鼓励规模化、连续性及重点品牌办展）、支持举办高端会议项目（鼓励引进、规模化及高端办会）以及强化会展要素保障（提高场馆利用率、强化人才支持、开辟专属参观通道、推动建立智能线上会展政务平台）。

# 奖励额度/补贴标准

**一、支持举办展览项目**

1.  **鼓励规模化办展**
    *   在雄安新区举办展览项目，实际会期达到2天以上，标准展位数达到100（含）以上，奖励项目主办方10万元，每增加一个标准展位，多奖励300元。
    *   实际会期达到2天以上，标准展位数达到100（含）以上，且符合新区5大产业发展方向的，奖励项目主办方20万元，每增加一个标准展位，多奖励600元。
    *   单个展览项目每届规模奖励上限50万元。
2.  **鼓励连续办展**
    *   对连续多届在雄安新区举办，每届间隔超过6个月，每届标准展位数大于前一届，且符合“鼓励规模化办展”奖励条件的展览项目，每届奖励主办方实际增加支出的20%。
    *   对同一个展览项目奖励不超过5届，每届上限20万元。
3.  **鼓励重点品牌展览办展**
    *   对在雄安新区举办，符合新区“创新发展示范区”定位，促进新区相关产业发展有明显作用，且达到“鼓励规模化办展”奖励标准的展览项目，奖励主办方20万元。
    *   连续三年在雄安新区举办的自第三年起奖励提高至50万元。

**二、支持举办高端会议项目**

1.  **鼓励引进高端会议项目**
    *   对引进符合新区5大产业发展方向会议项目的单位，给予会议项目场地租赁费5%的一次性奖励，上限20万元。
    *   引进国家级、国际性、世界500强企业或中国百强企业主导的会议项目额外奖励20万元。
2.  **鼓励规模化办会**
    *   在新区举办的会议与论坛项目，实际会期达到2天及以上，参会人数200（含）～400（不含）人，奖励主办方6万元。
    *   参会人数400（含）～600（不含）人，奖励主办方12万元。
    *   参会人数600（含）人以上，奖励主办方15万元。
3.  **鼓励举办高端会议**
    *   对符合“鼓励规模化办会”奖励条件的，参会人员不少于三个国家或地区且不少于20%总参会人数的国际会议，给予主办方租用会议场地实际支出40%奖励，最高不超过40万元。

**三、强化会展要素保障**

1.  **提高展览场馆利用率**
    *   场馆全年承办标准展位数100（含）以上的展览达10场（含）以上，给予场馆单位5万元奖励。
    *   全年承办标准展位数100（含）以上的展览达20场（含）以上，给予场馆单位10万元奖励。
2.  **强化会展人才支持**
    *   按照《河北雄安新区“雄才卡”管理办法（试行）》，支持符合条件的会展相关人才申领“雄才卡”，给予落户、人事档案管理、人才公寓入住、子女入学等方面的绿色通道服务。

# 注意事项

1.  **奖励上限**：单个展览项目每届奖励最高不超过110万元，单个会议项目每届奖励最高不超过50万元。
2.  **资金来源**：所需资金从产业发展专项资金里细化。
3.  **报备要求**：会展项目主办方需在活动举办前向新区有关部门报备，经审核符合条件的方可按程序申请奖励资金。
4.  **凭证要求**：项目主办方资质认定需提供在新区实际发生的消费发票、合同等作为凭证。
5.  **资金发放**：根据实际情况优先将资金发放给项目引进单位，由其负责拨付。
6.  **排除情形**：政府部门全额出资主办的会展项目不享受本支持措施，部分出资主办的会展项目按实际情况进行确定。
7.  **一事一议**：对于在雄安新区举办的国家级、国际性，具有重大国际国内影响力的品牌展览活动，视其规模和影响力采取“一事一议”方式予以支持。
8.  **解释权**：本措施具体解释工作由新区商务主管部门承担。
9.  **施行时间**：本措施自印发之日起施行。《河北雄安新区党工委管委会党政办公室印发〈关于雄安新区促进会展业发展的若干措施〉的通知》（雄安办传〔2023〕7号）同时废止。

# 申请对象

1.  展览项目主办方。
2.  会议项目主办方。
3.  引进符合新区5大产业发展方向会议项目的单位（会议引进单位）。
4.  展览场馆单位。
5.  符合条件的会展相关人才。

# 申报条件

**一、展览项目**

1.  **规模化办展基础条件**：在雄安新区举办，实际会期达到2天以上，标准展位数达到100（含）以上。
2.  **规模化办展加分条件**：符合新区5大产业发展方向。
3.  **连续办展条件**：连续多届在雄安新区举办，每届间隔超过6个月，每届标准展位数大于前一届，且符合“鼓励规模化办展”奖励条件。
4.  **重点品牌展览条件**：在雄安新区举办，符合新区“创新发展示范区”定位，促进新区相关产业发展有明显作用，且达到“鼓励规模化办展”奖励标准。

**二、会议项目**

1.  **引进高端会议条件**：引进符合新区5大产业发展方向的会议项目；或引进国家级、国际性、世界500强企业或中国百强企业主导的会议项目。会议引进单位需由会议主办方通过提供承诺书、第三方服务合同等方式进行认证。
2.  **规模化办会条件**：在新区举办，实际会期达到2天及以上。
    *   参会人数200（含）～400（不含）人；
    *   或参会人数400（含）～600（不含）人；
    *   或参会人数600（含）人以上。
3.  **高端会议条件**：符合“鼓励规模化办会”奖励条件，且参会人员不少于三个国家或地区且不少于20%总参会人数。

**三、场馆单位**

1.  全年承办标准展位数100（含）以上的展览达10场（含）以上；或
2.  全年承办标准展位数100（含）以上的展览达20场（含）以上。

# 申报材料

1.  在新区实际发生的消费发票。
2.  合同。
3.  会议引进单位认证材料：承诺书、第三方服务合同等。

# 办理流程

1.  **事前报备**：会展项目主办方需在活动举办前向新区有关部门报备。
2.  **审核**：经审核符合条件的方可按程序申请奖励资金。
3.  **申请**：按程序申请奖励资金。
4.  **发放**：根据实际情况优先将资金发放给项目引进单位，由其负责拨付。
