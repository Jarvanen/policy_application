# 政策简介

“百项转型”工程聚焦推动雄安新区传统产业提档升级，以数字化转型和绿色化升级为重点，推动传统产业高端化、智能化、绿色化发展。支持符合国家和省现行产业结构调整目录、技术改造投资导向目录及《河北雄安新区传统产业转移升级工作实施方案》的传统产业转型升级项目，优先支持符合《工业重点行业领域设备更新和技术改造指南》要求的技术改造项目。主要支持方向包括企业技术改造项目（落后低效设备替代、试验检测设备升级）、企业数字化转型项目（智能制造装备、工业软件、智能工厂、产业链协同）以及公共服务平台项目（综合产业互联网平台、行业级平台、共享工厂等）。

# 奖励额度/补贴标准

1.  **企业技术改造项目**：按项目已完成的固定资产投资额（不含土地、建筑物投资），采取后补助方式给予不超过 **20%** 资金支持，单个项目最高不超过 **500万元**。
2.  **企业数字化转型项目**：按照信息化软硬件购置、系统开发与服务等费用，采取后补助方式给予不超过 **30%** 资金支持，单个项目最高不超过 **300万元**。
3.  **公共服务平台项目**：按照信息化软硬件购置、系统开发与服务等费用，采取后补贴方式给予 **20%** 资金支持，每个平台奖补金额最高不超过 **500万元**。其中，新区级公共服务平台每年支持不超过 1 个项目，行业级公共服务平台中每个行业每年支持不超过 1 个项目。

# 注意事项

1.  **重复享受限制**：已经获得国家、省、新区财政支持的同一项目，原则上不得再申报新区级专项资金支持。申报时需出具《申报项目未获得国家、省级、新区级专项资金支持情况说明》。
2.  **材料真实性**：项目申报单位对专项资金申报材料内容的真实性负责，并出具真实性声明。对以虚报、冒领、伪造、关联交易等手段骗取财政资金的行为，一经查实，将收回已下达的资金，取消后续申报资格，纳入失信成员名单，并依法依规追究责任。
3.  **材料格式要求**：
    *   资金申请报告书脊标注：“2024 第一批+\*\*县+申报单位名称+项目名称”。
    *   电子版正文为 **word** 格式，附件为 **pdf** 格式，表格为 **excel** 格式。
    *   项目资金申报资料需永久保存备查。
4.  **统计入库要求**：项目计划投资500万元以上的，应纳入统计部门工业技改项目管理。
5.  **优先支持条件**：开展两化融合管理体系贯标并通过评定的，被认定为省级以上“专精特新”中小企业、单项冠军、绿色工厂的，已入统的规模以上工业企业及临规企业培育库在库企业，同等条件下优先纳入项目支持。

# 申请对象

1.  依法在雄安新区注册且在本地缴纳税收、具有独立法人资格的**制造业企业**。
2.  **产业互联网平台服务机构**。

# 申报条件

1.  **主体资格**：项目实施单位应为依法在雄安新区注册且在本地缴纳税收、具有独立法人资格的制造业企业，以及产业互联网平台服务机构。
2.  **能力与信用**：项目实施单位具备较强的技术开发、资金筹措、项目实施能力，成长性较好，具有良好的社会信用。
3.  **项目状态**：
    *   申报项目实施地在雄安新区。
    *   已办理完成备案（核准）及土地、环评等手续。
    *   已开工建设。
4.  **合规性**：项目单位遵守国家法律法规，符合国家产业发展要求，企业经营状况良好、发展规划和目标明确，在经营管理、社会信用、安全生产、节能环保等方面无不良记录。
5.  **资金限制**：未获得过国家、省级、新区级专项资金支持（同一项目）。

# 申报材料

**通用基础材料：**
1.  项目资金申请报告（严格按照编制要点编制）。
2.  项目单位营业执照（组织机构代码证）复印件。
3.  项目核准或备案证、环评批复、土地手续复印件。
4.  项目财务审计报告（复印件，加盖申请单位公章）。
5.  三县工业和信息化主管部门出具的项目在建证明材料。
6.  项目固定资产设备投资完成明细表及对应发票复印件。
7.  项目单位同意税务机关提供设备、仪表仪器、软件产品、专利技术及设备改造等固定资产投资发票等相关资料的授权书。
8.  申报项目未获得国、省、新区级专项资金支持情况说明。
9.  项目单位对项目单行材料及附件真实性负责的声明。

**分类特定材料：**
*   **技术改造类**：需包含企业现有生产工艺及装备技术水平对比、产品竞争力情况、建设规模及内容、工艺技术详情、投资明细（土建、设备、技术等）及已完成投资情况。
*   **数字化转型类**：需包含企业数字化水平自评结果、在优质中小企业梯度培育平台完成数字化水平自测结果截图、项目建设目标及创新方向、技术路线、预期成效（提质增效、转型升级等方面）。
*   **公共服务平台类**：需包含目前已完成建设内容及应用效果、服务用户企业概述、具体应用场景、解决方案及实施内容、预期成效（服务企业数量、连接设备情况、APP数量等）。

# 办理流程

1.  **项目申报**：采用网上填报与纸质报送相结合的方式。各项目申报单位登录“雄安新区产业互联网平台”（www.xaip.org.cn）选择“百项转型”项目申报栏目填报申报材料，并提供申报书及佐证材料等。
2.  **县级评审**：
    *   三县工信主管部门开展项目完整性审查和真实性审查，实地考察项目进展，核查审批手续原件及财务相关佐证。
    *   委托专家或第三方机构，对项目资金申请报告进行专家评审和财务现场审核，提出评审意见、财务审核意见以及资金支持建议。
    *   三县工信主管部门组织县局办公会（党组会）对支持项目进行审议，并将审议意见上报县政府同意并加盖县政府公章，报新区改革发展局。
3.  **新区复核**：新区改革发展局对三县工信主管部门的项目审议意见进行复核，并将复核意见反馈三县工信主管部门。
4.  **公示**：三县工信主管部门对复核通过的项目，在产业互联网平台进行公示。公示结束后，将公示无异议项目的资金申请报告、评审意见、财务审核意见、资金支持建议、公示情况、资金汇总表盖章后报新区改革发展局。
5.  **资金支持**：新区改革发展局按程序将支持资金拨付至相关项目实施单位。

# 申报时间

1.  **县级评审截止**：三县工信部门于 **2024年8月8日** 前完成第一批申报项目评审，并将审议意见加盖县政府公章报新区改革发展局。
2.  **材料上报截止**：**2024年8月18日** 前将通过复核项目的资金申请报告、评审意见、资金支持意见、公示情况、资金汇总表盖章后报新区改革发展局。
