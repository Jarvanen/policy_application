# 政策简介

为支持生命科学产业发展，鼓励高端高新医疗器械企业疏解落户，推进政务服务事项集成化办理，雄安新区将第一类医疗器械生产企业涉及的“第一类医疗器械产品备案”、“第一类医疗器械生产备案”整合成“一件事”办理。该改革旨在创新探索实施第一类医疗器械产品备案、生产备案同步办理，创设更多高效办成一件事场景，促进雄安新区生命科学产业更好发展，打造市场化、法治化、国际化一流营商环境。

# 注意事项

1. **办理时限**：备案时限压减 50%，承诺时限为 1 个工作日。
2. **材料要求**：
   - 电子材料应为清晰 pdf 版，并加盖公章。
   - 证明文件为复印件的，需注明与原件一致。
   - 不再提交产品备案编号、凭证及技术要求复印件等材料。
3. **自主选择**：充分尊重医疗器械备案人自主选择权，可自主选择“第一类医疗器械一件事”同步备案或者先行办理第一类医疗器械产品备案。
4. **风险控制**：严禁出现“非医疗器械按医疗器械备案”、“高类低备”等情况。
5. **不予备案情形**：第一类医疗器械产品备案不予备案的，同步申报的生产备案（或生产备案变更）不予备案。
6. **收费情况**：不收费。

# 申请对象

河北雄安新区辖区内第一类医疗器械备案人自行生产第一类医疗器械的企业或事业单位。

# 申报条件

1. 有与生产的医疗器械相适应的生产场地、环境条件、生产设备以及专业技术人员；
2. 有能对生产的医疗器械进行质量检验的机构或者专职检验人员以及检验设备；
3. 有保证医疗器械质量的管理制度；
4. 有与生产的医疗器械相适应的售后服务能力；
5. 符合产品研制、生产工艺文件规定的要求。

# 申报材料

**（一）申请备案（产品备案、生产备案）**

1. **第一类医疗器械产品备案材料**：
   - 第一类医疗器械备案表（电子，填写规范完整）。
   - 关联文件（电子）：企业营业执照副本或事业单位法人证书的复印件；委托其他企业生产的，应当提供受托企业资格文件(营业执照副本复印件)、委托合同和质量协议复印件。
   - 产品技术要求（电子）：按照《医疗器械产品技术要求编写指导原则》编制，主要包括医疗器械产品的可进行客观判定的功能性、安全性指标和检测方法。
   - 产品检验报告（电子）：产品全性能自检报告或委托检验报告，检验的产品应当具有典型性。检验报告后随附产品实物照片（包括拆除所有内外包装后的样品实物照片，以及内外包装实样照片。多个型号规格的，提供典型产品的照片）。
   - 产品说明书及最小销售单元标签设计样稿（电子）：符合《医疗器械说明书和标签管理规定》《体外诊断试剂说明书编写指导原则》等相关要求，说明书中产品性能应当与产品技术要求中的相应内容一致。
   - 生产制造信息（电子）：对生产过程相关情况的概述。无源医疗器械应明确产品生产加工工艺，注明关键工艺。有源医疗器械应提供产品生产工艺过程的描述性资料，可采用流程图的形式，或生产过程的概述。体外诊断试剂应概述主要生产工艺，包括：固相载体、显色系统等的描述及确定依据，反应体系包括样本采集及处理、样本要求、样本用量、试剂用量、反应条件、校准方法(如果需要)、质控方法等；有多个研制、生产场地的，应当概述每个研制、生产场地的实际情况；委托其他企业生产的，应当列出受托企业名称、住所、生产地址。
   - 符合性声明（电子）：声明符合第一类医疗器械备案相关要求；声明本产品符合有关分类的要求及依据，包括《第一类医疗器械产品目录》或《体外诊断试剂分类子目录》的有关内容，应当注明确切的产品分类依据，明确所属子目录、一级、二级产品类别；声明本产品符合现行国家标准、行业标准并提供符合标准的清单；声明所提交备案资料的真实性(境内产品由备案人出具)。

2. **第一类医疗器械生产备案材料**：
   - 第一类医疗器械生产备案申请表（电子，填写规范完整，申请表内无需提供同步申报产品备案编号）。
   - 法定代表人(企业负责人)身份证明复印件（电子，身份证明文件在有效期内）。
   - 生产、质量和技术负责人的身份、学历、职称相关材料复印件（电子，生产管理部门和质量管理部门负责人不得互相兼任）。
   - 生产管理、质量检验岗位从业人员学历、职称一览表（电子，表内信息与备案表填报信息一致，满足企业生产、检验需求）。
   - 生产场地的相关文件复印件（电子）：包括租赁协议、房产证明(或使用权证明)，规划用途或设计用途不应为“住宅”“宅基地”等。有特殊生产环境要求的，还应当提交设施、环境的相关文件复印件：例如生产过程有净化要求的，应当提供有资质的检测机构出具的环境检测报告。
   - 主要生产设备和检验设备目录（电子）：检验设备包括进货检验、过程检验、出厂最终检验所需的设备；在净化条件下生产的，还应当提供环境监测设备。
   - 质量手册和程序文件目录：质量手册应当包括企业质量目标、组织机构及职责、质量体系的适用范围和要求；程序文件应当根据产品生产和质量管理过程中需要建立，例如文件控制程序、记录控制程序、设计控制程序、采购控制程序、质量控制程序、产品放行程序、顾客反馈处理程序、不合格品控制程序、数据分析程序、纠正措施程序、预防措施程序、产品信息告知程序、体系内部审核程序等。
   - 生产工艺流程图（电子）：注明主要控制项目和控制点，及关键工序和特殊过程的设备及工艺参数控制。
   - 证明售后服务能力的相关材料（电子）：应当提供包括售后服务制度、顾客反馈处理程序等；清楚表述售后服务责任单位或部门、相关流程等，并与提交的其他材料相一致。
   - 经办人的授权文件（电子，仅委托办理时提供）。

# 办理流程

1. **注册**：备案人登录国家药品监督管理局政务服务门户 https://zwfw.nmpa.gov.cn/web/index?sourceType=nmpa，完成注册并绑定“医疗器械地方行政许可备案信息系统”。
2. **填报**：备案人同步提交产品备案、生产备案（或生产备案变更）申请材料，并对提交的电子申请材料和数据的合法性、真实性、有效性负责。
3. **备案**：河北雄安新区营商环境局（行政审批局）对企业提交的备案资料进行核对。
   - 符合规定的，予以备案（备案变更），一次性告知第一类医疗器械产品备案、生产备案生产备案（或生产备案变更）需要补正的全部内容（如有），同步发放《第一类医疗器械备案编号告知书》《第一类医疗器械生产备案凭证》。
   - 备案资料不齐全或者不符合规定形式的，应当一次告知需要补正的全部内容。
   - 对不予备案的，应当告知备案人并说明理由。
