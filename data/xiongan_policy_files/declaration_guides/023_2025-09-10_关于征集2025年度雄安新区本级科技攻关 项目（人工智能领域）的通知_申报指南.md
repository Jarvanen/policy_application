# 政策简介

为深入贯彻落实高标准高质量建设雄安新区的重要指示精神，加快培育发展新质生产力，推动人工智能与雄安新区城市建设中创新场景深度融合，现公开征集2025年度雄安新区本级科技攻关项目（人工智能领域）。项目围绕雄安新区高质量建设、高水平管理、高质量疏解发展中的紧迫性、前瞻性科技需求，吸引和集聚国内外优秀科研力量和高端人才开展科研工作，推动人工智能技术研发、场景落地与产业集聚，助力雄安打造具有全球影响力的人工智能创新发展高地。

# 奖励额度/补贴标准

每个支持项目给予资金支持原则上不超过300万元。形成的具有转化潜力的科技成果，可优先推荐新区相关产业基金进行投资孵化。

# 注意事项

1. **信用记录与真实性**：项目申报单位须无不良信用记录，对申报材料内容的真实性、合法性负责，如存在虚假填报等情况，项目视为无效。
2. **禁止重复申报**：已获得财政经费支持的相关技术及研究内容，不得再重复申报。
3. **代理限制**：雄安新区工信科技数据局未指定、授权或委托任何中介机构或个人从事科研项目申报的代理活动，代理机构或个人的行为与工信科技数据局无关。
4. **申报数量限制**：各申报单位承担本次本级科技攻关项目数量原则上不超过2项（不含雄安新区“揭榜挂帅”科研课题、重大科技专项、软科学研究项目）。
5. **项目负责人限制**：同一个项目负责人在研项目不得超过2个；申报用户必须为项目负责人，且必须进行实名，保障信息的真实性，要求年龄不超过60周岁。
6. **系统操作要求**：申报单位须仔细阅读相关科技计划的管理文件、办法，确认本单位具备相关科技计划的项目申报资格；申报书标题统一用黑体四号字，申报书正文部分统一用宋体小四号字填写，正文（包括标题）行距为1.5倍。

# 申请对象

1. 在雄安新区范围内注册的企业、科研机构和高等院校等具有独立法人资格的单位。
2. 根据非首都功能疏解安排部署，以非法人机构形式在新区落地的疏解单位。
3. 全国范围内的高等学校、科研院所、企业等可作为合作单位参与承担项目。

# 申报条件

**（一）单位条件**
1. 单位管理规范，业务及财务制度健全，具有相应的科研能力和条件，运行管理规范。
2. 在新区范围内有固定办公场所并持续运营，有常驻工作人员。
3. 项目单位具有满足项目正常实施所需的基本资金、基础和条件，有固定的场所，无知识产权纠纷。

**（二）项目负责人条件**
1. 项目负责人应具有高级专业技术职务（职称）。
2. 具备领导和组织开展创新性研究的能力。
3. 科研信用记录良好。

**（三）项目条件**
1. **领域方向**：重点聚焦人工智能领域，围绕大模型、机器感知与机器人、自主可控芯片应用方向。
   - **大模型方向**：着力突破高效训练、推理部署等关键技术，推动创新应用落地。
   - **机器感知与机器人方向**：重点围绕智能感知、灵巧操作、自主决策与人机共融等核心能力，提升机器人智能化水平与应用适应性。
   - **自主可控芯片应用方向**：聚力高性能高能效AI算力、空天信息、能源表计、智能交通等城市建设领域，构建自主开放的自主可控产业生态。
   - 鼓励科研攻关团队面向智能网联、AI算力等重点需求，开发高性能、低功耗、低成本的AI解决方案。
2. **资金要求**：项目投资总额和资金来源应明确。企业作为申报单位须对项目经费进行配套，自筹经费与申请财政科技经费比例原则上不低于2:1。
3. **时间要求**：项目需要有一定的研究基础，在2025年12月前启动，执行周期原则上不超过2年。
4. **目标要求**：项目需具有明确的技术创新性和示范应用性，具有清晰、可量化的目标及考核指标。

# 申报材料

1. 通过“雄安新区产业互联网平台-科技项目综合服务平台”在线填写并提交申报书。
2. 申报书内容包括：项目意义与必要性、研究目标及内容、申报单位研究基础、进度安排、经费预算、项目组织实施/保障措施及风险分析、研究团队、所要求的附件。
3. 附件材料包括：联合申报协议、财务审计报告、技术就绪度自评表、其它相关材料。
4. 盖章件包括：雄安新区本级科技项目申报诚信承诺书（申请人部分）、雄安新区本级科技项目申报诚信承诺书（申报单位部分）、申报书封面。

# 办理流程

1. **在线申报**：申报单位登陆雄安新区产业互联网平台-科技项目综合服务平台板块（https://techpro.xaiip.org.cn/webIndex），点击政策申报-申报指南-立即申报，提交所有申报材料。
2. **初审**：新区工信科技数据局组织相关部门对申报材料的规范性和齐备情况进行初审。符合条件的，予以受理；不符合条件的，一次性告知需要补充的全部内容。
3. **评审**：受理工作结束后，新区工信科技数据局根据项目受理情况，分批次开展评审，根据评审结果确定拟支持项目。
4. **公示**：新区工信科技数据局将拟支持项目名单在指定官方网站进行公示，公示期为5个工作日。
5. **立项与拨付**：公示期满无异议的项目，新区工信科技数据局将按照相关程序办理立项、资金拨付及后期项目管理等工作。

# 申报时间

征集时间自2025年9月12日9时至2025年9月22日17时止，逾期不予受理。
