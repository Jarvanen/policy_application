# 政策简介

本通知旨在做好雄安新区河北省“专精特新”中小企业申报，进一步完善新区“专精特新”培育体系，引领新区中小企业实现高质量发展。主要工作包括河北省“创新型”中小企业评价和河北省“专精特新”中小企业认定。

# 申请对象

1. **河北省“创新型”中小企业评价对象**：符合条件的中小企业。
2. **河北省“专精特新”中小企业认定对象**：
   - 2022年公布的河北省“创新型”中小企业；
   - 通过新区评价审核并推荐省厅的2023年第一批河北省“创新型”中小企业。

# 申报条件

**河北省“创新型”中小企业评价条件**：
- 依据《河北省优质中小企业梯度培育管理细则》要求，符合相关评价标准。

**河北省“专精特新”中小企业认定条件**：
- 依据《河北省优质中小企业梯度培育管理细则》要求，符合专精特新中小企业认定标准。
- 需满足附件4佐证材料清单中涉及的各项指标要求（如研发费用、知识产权、主导产品领域、质量认证、标准制修订、产学研合作等）。

# 申报材料

**一、河北省“创新型”中小企业评价申报材料**
1. 《河北省创新型中小企业自评表》（在培育平台填写后下载打印，相关数据须与培育平台申报系统保持一致，在“真实性声明”处由法定代表人签字，并在封面加盖公章）；
2. 企业营业执照复印件；
3. 2021、2022年度审计报告（含研发费用），如无年度审计报告，则提供从电子税务局下载的2021年度、2022年度纳税申报表（含研发费用，加盖企业公章）；
4. 自评标准条件的必要佐证材料（含直通条件佐证材料或评分指标相关佐证材料，需加盖企业公章）。

**二、河北省“专精特新”中小企业认定申报材料**
1. 纸质认定申报书（含《河北省专精特新中小企业认定申报表》、相关佐证等）；
2. 佐证材料清单（包括但不限于）：
   - 企业营业执照复印件（必须项）；
   - 2020年、2021年、2022年年度审计报告及2022年研发费用专项审计报告；其中符合“近两年研发费用总额均值在1000万元以上”条件的，需提供2021年、2022年研发费用专项审计报告（必须项）；
   - 主导产品所属领域情况证明材料（包含在产业链供应链关键环节及关键领域“补短板”“锻长板”“填空白”取得实际成效证明材料，属于工业“六基”领域、中华老字号名录证明材料，主导产品服务关键产业链重点龙头企业的合同、发票等证明材料）；
   - 在优质中小企业梯度培育平台完成数字化水平自测结果截图；
   - 获得省级以上质量奖荣誉的文件、证书或牌匾；
   - 获得相关质量管理体系认证及对应证书；
   - 拥有自主品牌相应的佐证材料（产品注册商标证或其他相关材料）；
   - 主持（参与）制修订标准等证明材料；
   - 企业为产业链企业配套或带动产业链企业发展证明材料，包含销售合同、发票等；
   - 市级以上相关部门出具的属于鼓励支持发展的主导产业、特色产业或相关领域证明（文件号及文件名称等）；
   - 坚持绿色发展，利用特色资源或技术进行研制生产，提供独具特色的优质产品或服务证明材料；
   - 获得市级及以上技术创新、品牌、专利等称号或认定的文件、牌匾、证书等证明材料；
   - 企业与高等院校和科研院所开展产学研合作证明；
   - 获得科技特派员服务支持有关证明；
   - 近三年在地方经济发展贡献或企业家精神实践方面获得荣誉的文件、牌匾、证书等证明材料；
   - 企业上市计划、股份制改造等上市规划、辅导协议、企业名称变更核准通知书等证明材料；
   - 知识产权证书，如属于Ⅰ类高价值知识产权需说明并提供证明材料；
   - 企业自建或与高校、科研机构联合建立研发机构的佐证资料；
   - 上年度营业收入在1000万元以下，但近2年新增股权融资总额（合格机构投资者的实缴额）达2000万元以上证明材料，包括银行到账凭证或融资报告；
   - 近两年新增股权融资总额（合格机构投资者的实缴额）6000万元以上证明材料，包括银行到账凭证或融资报告；
   - 近三年获得过国家级科技奖励或省级科技奖励及排名等证明材料；
   - 近三年进入“创客中国”中小企业创新创业大赛全国500强企业组名单等证明材料及获奖证书；
   - 其他相关佐证材料。

# 办理流程

**一、河北省“创新型”中小企业评价流程**
1. **企业自评**：符合条件的中小企业按照自愿原则登录工业和信息化部优质中小企业梯度培育平台（zjtx.miit.gov.cn）参与自评；自评通过企业下载打印《河北省创新型中小企业自评表》，提供（平台上传）必要证明资料，并制作纸质申报书。
2. **县级审核与推荐**：三县工信方面主管部门完成网上审核，并正式行文推荐，连同企业申报书（申请书）资料（一式两份）报送新区改革发展局（白洋淀大道办公区C485房间）。
3. **新区择优推荐**：新区改革发展局根据三县推荐企业情况进行择优推荐。

**二、河北省“专精特新”中小企业认定流程**
1. **企业申报**：符合条件的企业登录河北工业企业服务平台申报板块（http://gxt.hebei.gov.cn/main/）申报河北省专精特新中小企业认定，并依据通知要求规范制作纸质认定申报书（含申报表、相关佐证等）。
2. **县级审核与推荐**：三县工信方面主管部门完成网上审核，并正式行文推荐，连同企业申报书（申请书）资料（一式两份）报送新区改发局（白洋淀大道办公区C485）。
3. **新区择优推荐**：新区改革发展局根据三县推荐企业情况进行择优推荐。

# 申报时间

**一、河北省“创新型”中小企业评价**
- **企业自评及材料提交截止**：2月20日前。
- **县级网上审核截止**：2月20日前。
- **县级正式行文推荐截止**：2月22日前。

**二、河北省“专精特新”中小企业认定**
- **企业网上申报及材料提交截止**：3月1日前。
- **县级网上审核截止**：3月1日前。
- **县级正式行文推荐截止**：3月2日前。

# 注意事项

1. **自愿原则**：创新型中小企业评价按照自愿原则进行。
2. **数据一致性**：《河北省创新型中小企业自评表》相关数据须与培育平台申报系统保持一致。
3. **签字盖章要求**：
   - 创新型中小企业自评表需在“真实性声明”处由法定代表人签字，并在封面加盖公章。
   - 纳税申报表（如适用）需加盖企业公章。
   - 自评标准条件的必要佐证材料需加盖企业公章。
   - 专精特新中小企业认定申报表需企业盖章、推荐单位盖章。
4. **材料份数**：企业申报书（申请书）资料需一式两份报送。
5. **审核把关**：三县工信方面主管部门需指导企业做好申报表填报和材料整理，并认真审核把关。
6. **工作通报**：新区改革发展局将对工作情况进行通报。
