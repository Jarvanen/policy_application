# 政策简介

为深入贯彻落实习近平总书记关于把雄安新区打造成新时代创新高地和创业热土的重要指示精神，始终把创新作为引领雄安新区高质量发展的第一动力，瞄准雄安新区建设发展中的重大科技需求，加强科技项目攻关，推进科技成果转化，现拟公开征集首批雄安新区本级科技项目申报书。围绕雄安新区规划建设中的紧迫性、前瞻性科技需求，吸引和集聚国内外优势科研力量和高端人才开展科研工作，积极推动科技项目建设，加快科技成果转化，加速创新产业集聚。

# 奖励额度/补贴标准

每个支持项目给予资金支持原则上不超过100万元，分两次发放。支持资金用于研发创新，研发费用应纳入雄安新区“科学研究与实验经费支出”统计范围，形成的具有转化潜力的科技成果，可优先推荐新区相关产业基金进行投资孵化。

# 注意事项

1. 项目申报单位须无不良信用记录，对申报材料内容的真实性、合法性负责，如存在虚假填报等情况，项目视为无效。
2. 雄安新区改革发展局未指定、授权或委托任何中介机构或个人从事科研项目申报的代理活动，代理机构或个人的行为与改革发展局无关。
3. 已获得财政经费支持的相关技术及研究内容，不得再重复申报。
4. 申报平台要求：不要使用IE浏览器，请用谷歌浏览器。
5. 财务信息要求：本项政策需要提供“2021年全年”财务信息。
6. 材料格式要求：财务报表支持PDF格式文件，大小不超过30M；新区本级项目申报书需下载模板填写完毕扫描上传；新区本级科技项目汇总表需下载模板填写完毕扫描上传。

# 申请对象

1. 在雄安新区范围内注册的企业、科研机构和高等院校等具有独立法人资格的单位。
2. 鼓励科技型企业结合行业发展技术瓶颈提出需求。

# 申报条件

**（一）单位条件**
1. 单位管理规范，业务及财务制度健全，具有相应的科研能力和条件，运行管理规范。
2. 在新区范围内有固定办公场所并持续运营，有常驻工作人员。
3. 项目负责人应具有高级专业技术职务（职称），具备领导和组织开展创新性研究的能力，科研信用记录良好。

**（二）项目条件**
1. 重点聚焦新一代信息与软件技术、生物与人口健康技术、新材料与高端制造技术、前沿交叉和颠覆性技术、智能建造技术等领域。
2. 项目单位具有满足项目正常实施所需的基本资金、基础和条件，有固定的场所，无知识产权纠纷。
3. 项目投资总额和资金来源应明确。
4. 项目需要有一定的研究基础，在2024年启动，执行周期原则上不超过2年。
5. 项目需具有明确的技术创新性和示范应用性，具有清晰、可量化的目标及考核指标。
6. 企业作为申报单位须对项目经费进行配套，自筹经费与申请财政科技经费比例原则上不低于2:1。
7. 各申报单位承担项目数量原则上不超过2项，鼓励各单位、各项目间开展协同创新，联合攻关技术难题。

# 申报材料

1. 单位基本信息表（必填）
2. 企业财务信息表（必填，需提供2021年全年财务信息，含财务报表PDF及审计报告）
3. 新区本级项目申报书（必填，需下载模板填写后扫描上传）
4. 新区本级科技项目汇总表（必填，需下载模板填写后扫描上传）
5. 数字人民币对公(企业)钱包（选填）

# 办理流程

1. **提交材料**：申报单位根据申报通知的受理时间和条件等要求，登录雄安新区产业互联网平台提交所有申报材料。
2. **初审**：新区改革发展局组织相关部门对申报材料的规范性和齐备情况进行初审。符合条件的，予以受理；不符合条件的，一次性告知需要补充的全部内容。
3. **评审**：受理工作结束后，新区改革发展局会根据项目受理情况开展评审，根据评审结果确定拟支持项目，对未能予以支持的项目列入雄安新区本级科技项目储备库。
4. **公示**：新区改革发展局将拟支持项目名单在指定官方网站进行公示，公示期为5个工作日。
5. **立项与拨付**：公示期满无异议的项目，新区改革发展局将按照相关程序办理立项、资金拨付及后期项目管理等工作。

# 申报时间

自通知发布之日起至2024年1月3日17时止。逾期不予受理。
