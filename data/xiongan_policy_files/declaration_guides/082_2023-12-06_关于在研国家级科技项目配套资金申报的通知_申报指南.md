# 政策简介

为落实《河北雄安新区关于支持企业创新发展若干措施》（雄安政字〔2023〕29号）第七条相关内容，鼓励新区企业和科研机构积极申报各级科技项目，加快提升新区科技创新能力，河北雄安新区改革发展局对新区企业或研发机构成功申报的国家级科技项目，给予相应配套资金支持。

# 奖励额度/补贴标准

新区按照实际获得专项经费1:1配套资金支持，单个项目配套资金原则上不超过1000万元。

# 注意事项

1. **资金用途**：配套资助资金应当按照《河北雄安新区科技项目（课题）经费暂行管理办法》等相关资金管理规定使用，统筹安排用于科研活动的直接支出。
2. **材料要求**：项目申请单位对申请材料的合法性、真实性、准确性和完整性负责；扫描件及复印件应该清晰；提交的国家项目任务书中应该有相关单位正式签章；进账银行凭证、转拨银行凭证应该完整。
3. **重复享受限制**：申请项目国拨资助资金已到达申请单位账户且未获得过雄安新区财政配套资助。

# 申请对象

1. 在雄安新区内依法经营，具有独立法人资格的企业、科研机构等单位。
2. 申请单位应当为申请项目/课题的牵头单位。
3. 申请单位应当在雄安新区拥有开展项目的必要条件。

# 申报条件

**一、支持范围（优先支持同时满足以下条件的项目）**
1. 为科技部组织的国家科技重大项目（即国家科技重大专项、科技创新2030-重大项目、国家重点研发计划、国家自然科学基金委重大项目、重大研究计划项目等）；
2. 雄安新区管委会已就申请项目出具《自筹经费来源证明》；
3. 申请单位为申请项目/课题的牵头单位。

**二、申请条件**
1. 申请单位应当是在雄安新区内依法经营，具有独立法人资格的企业、科研机构等单位；
2. 申请单位应当在雄安新区拥有开展项目的必要条件；
3. 申请项目国拨资助资金（以下简称“国拨资金”）已到达申请单位账户且未获得过雄安新区财政配套资助；
4. 项目计划结题时间应在2024年5月30日后，国拨资金到账日期为2022年1月1日至2023年11月30日；
5. 项目申请单位、项目负责人均未被列入国家严重违法失信名单。

# 申报材料

1. 雄安新区科技计划国家和省计划配套资助申请书原件；
2. 2022年度纳税证明复印件（非事业单位提供）；
3. 国家项目下达文件、任务书或合同书（含项目和课题，至少包含封面、基本信息表、项目预算表、签字盖章页）复印件；
4. 国拨资金的银行回单等相关进账凭证复印件；给项目参与单位拨款的银行回单复印件；
5. 科研诚信承诺书原件；
6. 雄安新区出具的《自筹经费来源证明》、历史配套资助下达文件等相关资料复印件。
