# 政策简介
为增强企业技术开发和创新能力，完善和健全雄安新区技术创新体系，根据《雄安新区企业技术中心认定管理办法（修订）》，河北雄安新区管理委员会改革发展局会同新区税务局、雄安海关支持建设一批雄安新区企业技术中心。企业技术中心是指企业适应创新发展和市场竞争需要设立的技术研发与创新机构，负责研究制定和组织实施企业技术创新战略规划；开展新产品、新技术、新工艺、新模式研发和应用；创造运用知识产权；建立完善技术标准体系；凝聚培养创新人才；开展协同创新；推进技术创新全过程实施。

# 奖励额度/补贴标准
1. **新认定奖励**：对新认定的雄安新区企业技术中心，雄安新区财政给予企业10万元的资金奖励。
2. **评价优秀奖励**：对评价结果为优秀的雄安新区企业技术中心，雄安新区财政给予企业30万元的资金奖励。
3. **升级奖励**：对新认定为省级企业技术中心的，给予50万元的资金奖励；对新认定为国家级企业技术中心的，给予100万元的资金奖励。
4. **税收优惠**：积极落实国家有关技术创新税收优惠政策。

# 注意事项
1. **申报时限**：材料提交截止日期为2024年5月8日。
2. **申报方式**：申报实施不见面审批，企业需在雄安新区产业互联网平台注册申报，提交电子版材料。
3. **诚信要求**：企业需对报送资料的真实性、完整性做出承诺。
4. **限制条件**：已获得雄安新区企业技术中心认定的企业，原则上不再受理其下属子公司企业技术中心认定申请。
5. **违规后果**：
   - 因运行评价不合格或逾期未报送评价材料被撤销资格的，两年内不得重新申报。
   - 因提供虚假材料和数据、发生重大质量安全事故、严重违反海关/税收法律法规或严重违法失信被撤销资格的，三年内不得重新申报。
6. **直接认定情形**：对于整体迁入雄安新区、已开展正常生产经营的企业，已被认定为国家或省级企业技术中心或分中心的，经必要材料审核后，直接认定为雄安新区企业技术中心或分中心。
7. **整改要求**：对于一次评价得分65分（含）至60分的，需限期整改，自评价结果发布之日起半年内将整改情况汇总报河北雄安新区改革发展局。

# 申请对象
1. 在雄安新区辖区内进行登记注册和纳税的企业法人。
2. **认定重点领域**：
   - 大数据与物联网、信息技术制造、人工智能与智能装备、生物医药健康、新能源与智能电网装备、新能源汽车与智能网联汽车、高端装备制造、新材料、先进环保、未来产业等省战略性新兴产业发展专项行动领域优势企业；
   - 产值规模较大、成长性好、竞争力强的高新技术企业；
   - 新区传统产业和重点发展产业中影响大、基础好、带动作用强的行业骨干企业；
   - 较大产值规模和创新优势的软件和信息服务业企业。

# 申报条件
1. **基本资格**：申请企业必须是在雄安新区辖区内进行登记注册和纳税的企业法人。
2. **行业地位**：企业在雄安新区同行业具有显著的发展优势和竞争优势，研究开发与创新水平在雄安新区同行业中处于领先地位。
3. **创新机制**：企业具有较为完善的技术创新机制和产学研合作机制，企业技术中心组织体系健全，发展规划和目标明确，技术创新成果丰富，成效显著。
4. **硬性指标**：
   - 年度研究与试验发展经费支出额不低于300万元；
   - 专职研究与试验发展人员数不少于30人；
   - 技术开发仪器设备原值不低于300万元。
5. **合规要求**：企业在申请受理截止时间前三年内，不得发生下列情况：
   - 因违反税收征管理法及有关法律、行政法规，构成偷税、骗取出口退税等严重税收违法行为受到行政、刑事处理；
   - 因违反海关法及有关法律、行政法规，构成走私行为，受到刑事、行政处罚，或因严重违反海关监管规定受到行政处罚；
   - 司法、行政机关认定的其他严重违法失信行为。

# 申报材料
1. 《雄安新区企业技术中心申请报告》；
2. 《雄安新区企业技术中心认定数据表》；
3. 《雄安新区企业技术中心评价指标附表》；
4. 评价指标必要证明材料（主要包括：技术中心高级专家、博士、硕士、专利信息、主持和参加制定的标准、省级和新区级研发平台、实验室和检测机构、科技奖励等方面的内容）；
5. 相关财务报表和统计报表（主要包括企业利润表；规模以上工业法人单位研发项目情况表、研发活动及相关情况表等，未列入统计范围的企业参照格式填报；大型企业集团需合并填报）；
6. 企业对报送资料的真实性、完整性承诺。

# 办理流程
1. **上报**：企业根据管理办法及申报通知编写申请材料，向三县（雄县、容城、安新县）改革发展部门提出申请并上报申报材料。
2. **初审**：三县改革发展部门对申请材料进行初审，对报送材料的合规性、真实性负责，在规定时间内将推荐企业齐全的申请材料（一式五份）和推荐意见上报雄安新区改革发展局。
3. **初评**：雄安新区改革发展局委托第三方机构，按照认定评价指标体系对企业申请材料进行形式审查和初步评审，提出通过初评的企业技术中心名单。
4. **终评**：
   - 雄安新区改革发展局会同雄安新区税务局、雄安海关对通过初评的企业是否存在涉税违法情况进行核查，出具审查意见。
   - 根据初评结果和涉税违法情况审查意见，由受委托的第三方机构组织进行专家评审，形成专家评审结果。
   - 雄安新区改革发展局根据专家评审结果进行综合终审，形成拟认定名单并公示，公示期为5个工作日。
5. **认定**：对公示无异议的企业技术中心，雄安新区改革发展局联合雄安新区税务局、雄安海关发文认定其为雄安新区企业技术中心。
