# 政策简介

为健全雄安新区上市后备企业培育机制，推动雄安新区资本市场持续健康发展，特设立“雄安新区上市后备企业资源库”（以下简称“上市后备库”）。上市后备企业管理工作坚持政府引导与企业自愿相结合，市场运作与政策扶持相结合，促进企业上市与优化产业结构相结合，利用金融资本与发展实体经济相结合。上市后备库分基础层和培育层，由企业自愿提出申请，按照细则筛选入库。由雄安新区推进企业上市工作领导小组办公室（设在新区改革发展局）做好日常管理及协调服务工作。

# 奖励额度/补贴标准

1. **入库奖励**：首次纳入上市后备企业库基础层的企业，一次性给予 20 万元奖励；首次纳入上市后备企业库培育层的企业，一次性给予 30 万元奖励。
2. **股改中介费补助**：对于纳入培育层的企业，已经完成股份制改造的，其所需中介费可根据股改财务顾问协议费用的 80%申请补助，最高不超过 50 万元。
3. **毕业生引进待遇**：上市后备企业库企业引进的毕业生，符合《雄安新区支持企业引进毕业生实施办法（试行）》相关规定的，由上市领导小组办公室推荐其享受相应待遇。

# 注意事项

1. **奖励限制**：同一企业每项奖励只能享受一次。
2. **入库前提**：利用多层次资本市场融资发展的入库企业，按雄安新区本级出台的上市相关政策给予扶持奖励，未入库企业原则上将不予以奖励。
3. **动态管理**：对上市后备企业库实施动态管理，每年进行调减。
4. **调减出库情形**：
   - 已经成功上市；
   - 企业或企业高管出现重大违法违规或失信行为；
   - 连续两年财务指标等不再符合本细则第四条所规定的入库标准；
   - 企业注册地已迁出雄安新区或停止经营活动；
   - 企业主营业务与国家、新区现行产业和环保政策不符；
   - 企业主动申请出库；
   - 企业拒不配合信息更新管理；
   - 新区推进企业上市工作领导小组认定的其他应出库情形。
5. **培育层调整**：列入上市后备企业库培育层的企业两年内未完成股份制改造，将调减出培育层，再次申请进入培育层，需完成股份制改造。
6. **信息报送**：入库企业应在每年 3 月底前向新区上市领导小组办公室提交上一年度完整年度财务报表及企业整体发展情况。

# 申请对象

注册地在雄安新区内，依法设立且合法存续两个会计年度以上（含两年）的股份有限公司或有限责任公司。

# 申报条件

**基础层入库条件：**
1. 注册地在新区内，依法设立且合法存续两个会计年度以上（含两年）的股份有限公司或有限责任公司；
2. 有明确的境内外上市意愿和计划；
3. 企业主营业务突出，符合国家产业政策和环境保护政策；
4. 企业运作规范，财务会计报告无虚假记录，近两年无重大违法违规行为和不良信用记录，不存在企业或法定代表人失信情况；
5. 企业财务指标应满足下列条件之一：
   - 最近两年连续盈利，且近两年净利润累计不低于1600万元；
   - 最近一年盈利，且营业收入不低于5000 万元；
   - 最近一年营业收入不低于 4000 万元，且最近两年累计研发投入占最近两年累计营业收入的比例不低于8%；
   - 最近两年营业收入连续增长，且年均复合增长率不低于 20%，年均营业收入不低于3000 万元；
   - 对于列入雄安新区“专精特新”企业库的企业，在科技创新方面有较强能力和示范效应的，可适当放宽上述财务指标要求，经上市领导小组办公室批准后列入上市后备库基础层。

**培育层入库条件（需在基础层企业中筛选）：**
结合基础层企业的行业、规模、财务等情况，对其中计划未来两年内完成股份制改造且符合下列条件之一的，经新区上市领导小组办公室审核后，列入培育层：
1. 最近两年持续盈利,且累计净利润不低于 3000 万元或者累计营业收入不低于 3 亿元；
2. 最近两年研发投入合计不低于4000 万元；
3. 企业已完成股份制改造并与保荐机构（包含保荐券商、会计事务所、律师事务所）签订上市辅导协议；
4. 已被认定为国家级专精特新中小企业和专精特新“小巨人”企业。

# 申报材料

**首次参与申报基础层的企业应提供：**
1. 公司章程；
2. 有资质的会计师事务所审计的财务报表和审计报告；
3. 雄安新区上市后备企业备案表。

**在基础层的企业申报培育层，除上述材料外，还需提供：**
1. 与保荐机构签订的两年内完成规范化股份制改造的财务顾问协议；
2. 以下材料之一：
   - 已完成股份制改造的相关材料和与保荐机构签订的上市辅导协议；
   - 被认定为国家级专精特新中小企业和专精特新“小巨人”企业的相关证明材料。

# 办理流程

1. **申报平台**：依托雄安新区产业互联网平台进行管理，企业入库申请在该平台填报。
2. **材料审核**：
   - 三县注册的企业入库申请材料由三县推进企业上市工作领导小组办公室负责初审；
   - 新区本级注册的企业入库申请材料由新区上市领导小组办公室负责审核。
3. **材料复核**：新区上市领导小组办公室对三县初审合格的企业进行复核。
4. **公示**：复核通过后在新区产业互联网平台公示 5 个工作日。公示有异议的企业，由上市领导小组办公室提出处理意见。
5. **入库**：公示结束后列入“雄安新区上市后备企业资源库”。

# 申报时间

入库企业应在每年 3 月底前向新区上市领导小组办公室提交上一年度完整年度财务报表及企业整体发展情况。
