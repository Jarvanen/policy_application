# 政策简介

为贯彻落实《雄安新区机器人产业创新发展三年行动计划（2024—2026年）》，加快推动雄安新区机器人产业布局和发展，融入京津冀重点产业链条，打造成为新时代的创新高地和创业热土，参照《河北省支持机器人产业发展若干措施》等文件，提出以下措施。

# 奖励额度/补贴标准

**一、打造高水平科技创新能力**
1. **支持科技创新平台建设**：对在雄安新区设立独立法人单位的国家级、省级机器人产业研究院、重点实验室、产业创新中心、技术创新中心、制造业创新中心、工程技术研究中心等各类科技创新平台，按年度研发投入的50%进行补助，最高不超过1000万元。
2. **支持关键技术产品攻关**：支持企业通过“揭榜挂帅”解决机器人产业短板问题和“卡脖子”技术难题，最高按实际攻关投入（包括与该项目相关的人员人工费、研发设备及软件购置费；小试、中试工艺设备购置、开发及制造费；设计费、材料费、检测检验费以及委托外单位开展相关研发活动的费用）的50%进行补助，最高3000万元。

**二、拓展未来之城场景应用**
3. **支持机器人场景创新应用**：
   - 场景提供主体（非政府投资主体）：按机器人产品及服务部分投资采购额20%给予奖励，最高200万元。
   - 机器人产品及服务提供商：按实际研发投入20%给予奖励，最高100万元。
4. **鼓励企业采用灵活方式打造机器人场景**：
   - 租赁方式：对采用租赁等方式推动机器人场景应用的企业，按年租赁合同金额的10%予以支持，每家企业最高100万元。
   - “机器人+”解决方案咨询或设计服务：按每个方案服务费用的50%，给予每家企业最高不超过30万元奖励。
   - “机器人+”集成项目：按项目实际发生额的5%奖励系统集成商，单个项目最高50万元。

**三、培育高质量产业生态**
5. **支持机器人龙头企业落地**：对在雄安新区新设立且实缴注册资本在1000万元以上且实际经营的机器人企业，给予最高5%的开办补贴，相关经费分3年按40%、30%、30%比例拨付。对投资总额大、产业关联度高、带动性强的重点企业，可采取“一企一策”“一事一议”的方式给予支持。
6. **引进培育优质中小企业**：
   - 高新技术企业、“专精特新”企业、专精特新“小巨人”企业：按现行政策予以认定奖励、科技创新券等支持。
   - 制造业单项冠军：新获评国家级制造业单项冠军企业、单项冠军产品的，分别给予200万元、100万元奖励；新获评省级制造业单项冠军企业、单项冠军产品的，分别给予100万元、50万元奖励。
7. **支持机器人重大产业项目落地**：
   - 重大产业项目：对当年新增固定资产投资额（土地购置费用除外）超过500万元的机器人产业项目，按照新增固定资产投资额的20%比例，给予最高2000万元的资金奖励，单个项目补贴年限不超过3年。
   - “机器人生产机器人”标杆工厂：对实现机器人生产全流程无人化、智能化的机器人企业，按照不超过建设项目投资的30%予以奖励，最高3000万元。
   - 重点项目转化：对其他已获得国家、省级科技计划支持的重点项目，在雄安新区转化的，按照不超过建设项目投资的30%予以奖励，最高3000万元。

**四、建立高水平公共服务体系**
8. **支持机器人产业公共服务平台建设**：支持建设中试平台、概念验证中心、共享加工中心、检验检测中心等公共服务平台，每个项目按照经评审核定总投资的30%予以奖励，最高不超过5000万元。
9. **支持机器人科普中心建设**：
   - 建设奖励：对经认定的科普中心给予软件系统及硬件设备投资10%的资金奖励，最高2000万元。
   - 运营奖励：科普中心运营前三年，每年最高按照年度运营经费50%的标准给予运营方资金奖励，累计不超过200万元。

# 注意事项

1. **政策重叠处理**：本政策中涉及的具体政策与其他既有政策重叠的，按照“就高不就低、不重复”的原则执行。
2. **实施细则**：由雄安新区改革发展局牵头，会同有关部门制定有关支持政策的具体实施细则。
3. **有效期**：本政策自印发之日起施行，有效期至2026年12月31日。

# 申请对象

1. **科技创新平台**：在雄安新区设立独立法人单位的国家级、省级机器人产业研究院、重点实验室、产业创新中心、技术创新中心、制造业创新中心、工程技术研究中心等；雄安新区企业、高校、科研院所新认定的各类上述科技创新平台。
2. **攻关企业**：通过“揭榜挂帅”解决机器人产业短板问题和“卡脖子”技术难题的企业。
3. **场景应用主体**：
   - 场景提供主体（非政府投资主体）。
   - 机器人产品及服务提供商。
   - 采用租赁等方式推动机器人场景应用的企业。
   - 开展“机器人+”解决方案咨询或设计服务的雄安新区企业。
   - 实施“机器人+”集成项目的系统集成服务商。
4. **落地企业**：
   - 在雄安新区新设立且实缴注册资本在1000万元以上且实际经营的机器人企业。
   - 投资总额大、产业关联度高、带动性强的重点企业。
   - 机器人行业领军企业。
   - 机器人企业（申报高新技术企业、“专精特新”企业、专精特新“小巨人”企业、制造业单项冠军等）。
5. **公共服务平台建设方**：建设中试平台、概念验证中心、共享加工中心、检验检测中心等机器人公共服务平台的企业或机构。
6. **科普中心建设及运营方**：建设并提供科普、实训、技术推广及体验等服务的企业。

# 申报条件

1. **科技创新平台**：
   - 引进类：国家级、省级机器人产业研究院、重点实验室、产业创新中心、技术创新中心、制造业创新中心、工程技术研究中心等，且在雄安新区设立独立法人单位。
   - 新认定类：雄安新区企业、高校、科研院所新认定的各类上述科技创新平台。
2. **关键技术攻关**：
   - 通过“揭榜挂帅”方式。
   - 聚力解决机器人产业短板问题和“卡脖子”技术难题。
   - 实际攻关投入包括：与该项目相关的人员人工费、研发设备及软件购置费；小试、中试工艺设备购置、开发及制造费；设计费、材料费、检测检验费以及委托外单位开展相关研发活动的费用。
3. **场景创新应用**：
   - 入选定期遴选的雄安新区机器人应用场景示范项目。
   - 场景提供主体须为非政府投资主体。
4. **灵活方式打造场景**：
   - 采用租赁等方式推动机器人场景应用。
   - 开展“机器人+”解决方案咨询或设计服务（需符合条件）。
   - 实施“机器人+”集成项目。
5. **龙头企业落地**：
   - 在雄安新区新设立。
   - 实缴注册资本在1000万元以上。
   - 实际经营。
   - 重点企业需具备投资总额大、产业关联度高、带动性强等特点。
6. **优质中小企业**：
   - 申报高新技术企业、“专精特新”企业、专精特新“小巨人”企业。
   - 新获评国家级或省级制造业单项冠军企业、单项冠军产品。
7. **重大产业项目**：
   - 机器人行业领军企业布局的具有生态主导力和国际竞争力的重大产业项目。
   - 当年新增固定资产投资额（土地购置费用除外）超过500万元。
   - “机器人生产机器人”标杆工厂：实现机器人生产全流程无人化、智能化。
   - 重点项目转化：已获得国家、省级科技计划支持的重点项目，在雄安新区转化。
8. **公共服务平台**：
   - 聚焦机器人产业技术研发和成果转化共性需求。
   - 建设中试平台、概念验证中心、共享加工中心、检验检测中心等。
   - 提供样机试制、加工工艺和高精部件生产、检验检测等公共服务。
9. **科普中心**：
   - 经认定。
   - 提供科普、实训、技术推广及体验等服务。

# 办理流程

1. **制定细则**：由雄安新区改革发展局牵头，会同有关部门制定有关支持政策的具体实施细则。
2. **遴选/认定**：
   - 定期遴选雄安新区机器人应用场景示范项目。
   - 新认定各类科技创新平台。
   - 新获评制造业单项冠军等资质。
   - 经评审核定总投资（针对公共服务平台）。
   - 经认定科普中心。
3. **拨付/奖励**：
   - 开办补贴分3年按40%、30%、30%比例拨付。
   - 其他奖励按政策规定标准执行。

# 申报时间

本政策自印发之日起施行，有效期至2026年12月31日。具体申报时间需关注后续发布的实施细则或通知。
