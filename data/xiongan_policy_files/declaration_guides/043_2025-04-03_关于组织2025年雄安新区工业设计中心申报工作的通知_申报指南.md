# 政策简介

为加快工业设计发展，加强企业工业设计能力建设，河北雄安新区工信科技数据局支持建设一批雄安新区工业设计中心。申报分为两类：
1. **雄安新区工业设计中心（企业工业设计中心）**：指制造业企业等单位专门成立的，相对独立运行的，主要为本单位提供工业设计服务的内设部门。重点围绕电子信息、机器人、生物制造、医疗健康、新能源、新材料、智能装备等战略性新兴产业以及服装、制鞋、毛绒玩具、纸塑包装等传统产业领域的优势企业开展培育创建。
2. **雄安新区工业设计中心（工业设计企业）**：指面向市场需求提供工业设计服务的设计企业。重点围绕服务制造业设计创新的产品设计、品牌设计、时尚设计、服务设计等领域优势设计企业开展培育创建。

# 申请对象

1. **企业工业设计中心申报主体**：雄安新区依法注册、具有独立法人资格、运营管理规范的制造企业。
2. **工业设计企业申报主体**：雄安新区依法注册、具有独立法人资格、运营管理规范的工业设计企业。

# 申报条件

**（一）通用基础条件**
1. 工业设计中心稳定运营两年以上（截至申报日期）。
2. 有固定的工作场所、必要的设计软件和试验设备、健全的管理制度、稳定的人员配置。
3. 满足《雄安新区工业设计中心主要评价指标》要求。
4. 遵守国家法律法规，两年内未发生重大环保、质量和安全事故，未被列为严重失信主体，没有重大违法行为或涉嫌违法正在接受有关部门审查的情况。

**（二）企业工业设计中心专项条件**
根据《雄安新区工业设计中心主要评价指标》（表1），需满足以下定量条件：
1. **设计团队**：设有首席设计师岗位，团队人员数量应达到10人以上，具有工业设计及相关专业大学本科及以上学历或取得相关高级专业技术职称（高级职业资格）的人员比例不低于50%。
2. **知识产权**：近两年，获得国内外专利及版权年均5项以上，或累计30项以上。
3. **其他**：近两年有稳定持续的设计费用投入；产出一批优秀设计成果；获得国家级、省部级或国内外知名设计奖项；牵头或参与制定设计标准等。

**（三）工业设计企业专项条件**
根据《雄安新区工业设计中心主要评价指标》（表2），需满足以下定量条件：
1. **设计团队**：团队人员数量应达到10人以上，具有工业设计及相关专业大学本科及以上学历或取得相关高级专业技术职称（高级职业资格）的人员比例不低于50%。
2. **业务规模**：近两年，工业设计服务年均营业收入应不低于200万元，占企业总营业收入的比例不低于50%。
3. **服务企业情况**：累计服务制造企业不少于20家，有一批高质量的设计服务案例。
4. **其他**：近两年有稳定持续的设计投入；获得国家级、省部级或国内外知名设计奖项；管理规范，有健全的管理制度、知识产权保护制度等。

**（四）申请表填写硬性指标（附件要求）**
*注：附件申请表中部分栏目备注了更严格的填报要求，申报时需同时满足：*
1. **设计团队人员数量**：申请表备注要求设计团队人员数量20人以上。
2. **高学历/高职称人员比例**：申请表备注要求具有工业设计及相关专业大学本科及以上学历或取得相关高级专业技术职称（高级职业资格）的人员比例不低于70%。
3. **专利版权数量**：申请表备注要求达到近两年年均10项或累计50项以上。
4. **服务企业数量**：工业设计企业申请表备注要求服务企业名单50家以上。

# 申报材料

1. **雄安新区工业设计中心申请表**：
   - 企业工业设计中心填写《雄安新区工业设计中心申请表（企业工业设计中心）》。
   - 工业设计企业填写《雄安新区工业设计中心申请表（工业设计企业）》。
   - 声明页末尾务必请申请单位法定代表人签名，并加盖单位公章。
2. **必要证明材料**（需扫描清晰，按申请表要求提供佐证）：
   - **基础材料**：企业营业执照、法定代表人身份证复印件；近两年财务审计报告复印件（含审计报告正文、资产负债表、利润表、现金流量表）；工业设计中心成立的佐证材料（如公司决定成立通知、董事会决议等，仅企业工业设计中心需提供）；企业相关管理制度、知识产权保护制度材料（仅工业设计企业需提供）。
   - **人员佐证**：首席设计师/团队带头人证明材料；设计团队人员学历证书或高级专业技术职称（高级职业资格）佐证材料；设计团队人员隶属关系证明（社保清单、个人社保证明或劳动合同等）。
   - **知识产权佐证**：专利证书复印件、版权证书复印件。
   - **成果与项目佐证**：项目立项书复印件；产品效果图和实物图（彩色）；项目产品销售相关佐证（合同、发票及有关凭证）；近两年设计服务案例证明材料和相关图片。
   - **荣誉与标准佐证**：标准复印件（体现标准名称、标准号及主持或参与信息）；获奖证书复印件；有效期内的新区级及以上荣誉图片。
   - **其他**：申报单位认为有必要的事项佐证材料。

# 办理流程

1. **平台注册与提交**：实施不见面审批，申报主体需在雄安新区产业互联网平台注册，提交电子版材料。
2. **初步审核**：
   - 如申报主体为雄安新区本级注册企业，由雄安新区工信科技数据局负责组织对申报材料进行初步审核。
   - 如申报主体为三县（雄县、容城县、安新县）注册企业，由三县工业和信息化主管部门负责组织对申报材料进行初步审核，确定推荐名单。
3. **评审与认定**：满足定量条件基础上按得分高低择优入围送审名单。
