# 政策简介

本细则旨在贯彻落实《关于促进“专精特新”企业和数字经济核心产业高质量发展的若干措施》，明确科技企业库具体管理措施，引导和培育新区企业高质量发展。政策主要包含科技企业库入库管理及奖补政策，资金从新区改革发展局产业发展资金列支。

# 奖励额度/补贴标准

1. **入库补助**：首次入库，一次性给予2万元补助。
2. **租金补贴**：对在新区孵化载体内租赁且自用办公用房的企业，给予1.5元/平方米/天租金补贴，每年不超过50万元。
   * 新区孵化载体指：国家级众创空间、新区级科技企业孵化器、国家级科技企业加速器。

# 注意事项

1. **申报平台**：企业须依托雄安新区科创服务区块链平台（新区产业互联网平台：http://www.xaiip.org.cn）进行入库及奖补申报。
2. **租金补贴限制**：
   * 企业工商注册地址须在孵化载体内。
   * 须提供房租支付凭证证明实际租赁。
   * 本措施原则上与《河北雄安新区关于支持企业创新发展的若干措施》(雄安政字〔2021〕5号)第十二条补贴同期申报，每半年一次。
3. **资金发放方式**：由新区财政部门通过财政对公钱包将数字人民币发放至企业数字人民币对公钱包。
4. **违规处罚**：如有伪造证件材料、虚报冒领等欺瞒行为，将追回补贴资金，给予警告，没收违法所得，并处被骗取补贴资金10％以上50％以下的罚款或者被违规使用补贴资金10％以上30％以下的罚款；对直接负责的主管人员和其他直接责任人员处3000元以上5万元以下的罚款；在新闻媒体上曝光；情节严重触犯法律的，移交司法机关处理。

# 申请对象

1. **入库对象**：新区企业（含在京高新技术企业、科技型中小企业整体迁入新区或在新区设立子公司）。
2. **租金补贴对象**：在新区孵化载体内租赁且自用办公用房的企业。

# 申报条件

1. **入库基础材料要求**：
   * 提供大专以上人员学历证明。
   * 提供企业社保证明（近6个月内任一月数据）。
   * 提供企业上一年度审计报告。
   * 签署并上传企业申请入库/补贴承诺书（盖企业公章）。
   * 原公司（或母公司）在京营业执照，在京高新技术企业、科技型中小企业审核证明（仅在京高新技术企业、科技型中小企业整体迁入新区或在新区设立子公司直接入库申报时提供）。
2. **租金补贴条件**：
   * 企业工商注册地址须在孵化载体内。
   * 提供入孵协议扫描件及企业房屋租赁信息等材料。
   * 提供房租支付凭证证明实际租赁。

# 申报材料

1. **入库申报材料**：
   * 大专以上人员学历证明；
   * 企业社保证明（近6个月内任一月数据）；
   * 企业上一年度审计报告；
   * 企业申请入库/补贴承诺书（盖企业公章）；
   * 原公司（或母公司）在京营业执照，在京高新技术企业、科技型中小企业审核证明（仅在京高新技术企业、科技型中小企业整体迁入新区或在新区设立子公司直接入库申报时提供）；
   * 审核部门要求的其他相关证明材料；
   * 平台所需其他基础信息。
2. **租金补贴申报材料**：
   * 入孵协议扫描件；
   * 企业房屋租赁信息等材料；
   * 企业申请入库/补贴承诺书（附件1）；
   * 房租支付凭证。

# 办理流程

1. **入库流程**：
   * **申报**：企业通过平台“科技企业库入库申请”模块进行申报，系统校验通过后予以受理。
   * **审核**：审核部门（三县人民政府确定的科技企业库相关管理部门）安排专人对企业申请进行审核，不符合条件的予以退回并告知企业。审核工作须在收到企业申请后7个工作日内完成。
   * **汇总、备案及公示**：申报结束后，审核部门形成《科技企业库202X年上/下半年入库企业审批汇总表》，报县政府研究同意后在平台进行公示（不少于5日），并同步将盖有县政府公章的汇总表报新区改革发展局备案。
   * **入库**：公示无异议后，新区改革发展局在平台对相应企业进行确认操作，完成企业入库。
2. **入库补助发放流程**：
   * 企业入库公示无异议后，审核部门直接导出《XX县科技企业库企业开办补助资金汇总表》加盖县政府公章报新区改革发展局备案。
3. **租金补贴流程**：
   * **申报**：企业在平台上传入孵协议扫描件及企业房屋租赁信息等材料，签署并上传承诺书，发起申请。
   * **初审**：孵化载体对本单位入驻企业上报申请材料的完整性、真实性和有效性以及入驻企业运营状态和信用安全等情况进行审核勘察并严格把关。全部审核完成后盖章并上传《XX(孵化载体名称)入驻企业办公用房租金补贴资金汇总表》等材料。
   * **审核**：新区科技部门根据企业填写信息及上传材料进行审核，同时要求通过审核企业将有关纸质版申报材料备案。
   * **汇总、备案及公示**：申报阶段结束后，新区科技部门将通过审核的企业名单报新区改革发展局研究同意后，在平台进行确认操作，并同步在平台进行公示（不少于5日）。
4. **资金拨付**：新区科技部门将报账资料报新区财政部门审核。审核通过的，由新区财政部门通过财政对公钱包将数字人民币发放至企业数字人民币对公钱包。

# 申报时间

1. **入库申报**：每年6月、12月为科技企业库入库申报窗口期。
2. **租金补贴申报**：每半年一次。
