# 政策简介

本通知旨在组织开展2023年度第二批高新技术企业申报认定工作。依据河北省高新技术企业认定管理工作领导小组办公室相关要求，由三县科技管理部门积极组织辖域内企业开展申报，经专家预评审及严格审查后，将符合条件的企业推荐至河北省认定管理机构。

# 申请对象

1. 辖域内企业。
2. 2023年未提交复审的高企（需通知到位）。

# 申报条件

1. 申报企业需具备国家级或河北省科技型中小企业资质，或已在平台申报相关资质。

# 申报材料

1. 电子版申请材料：需在国家“高新技术企业认定管理工作网”提交。
2. 纸质版申请材料：需逐项打印、装订成册，交三县科技管理部门。
3. 一致性要求：电子版材料需与纸质版一致。
4. 其他材料：资质、证明材料、告知承诺书等。

# 办理流程

1. **提交申请**：7月7日前，申报企业在国家“高新技术企业认定管理工作网”提交电子版申请；同时纸质版材料逐项打印、装订成册，交三县科技管理部门。
2. **专家预评审**：7月11日举办专家预评审。
3. **意见反馈与修改**：7月12日反馈专家预评审意见，申报企业修改后于7月15日前重新提交电子版。
4. **审查与推荐**：雄安新区改革发展局组织专家对申报企业的资质、证明材料、告知承诺书等条件进行严格审查，并核实纸质材料与国家高新技术企业认定管理工作网中信息一致，将符合条件的企业推荐至河北省认定管理机构。

# 申报时间

1. 电子版申请提交及纸质材料递交截止：7月7日前。
2. 专家预评审：7月11日。
3. 专家预评审意见反馈：7月12日。
4. 修改后重新提交电子版截止：7月15日前。

# 注意事项

1. 电子版材料需与纸质版一致。
2. 雄安新区改革发展局将对申报企业的资质、证明材料、告知承诺书等条件进行严格审查。
3. 需核实纸质材料与国家高新技术企业认定管理工作网中信息一致。
