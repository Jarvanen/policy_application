# 政策简介

本措施旨在加快推进承接北京非首都功能疏解，大力引进培育创新主体，促进企业创新发展，提升自主创新能力，支撑雄安新区高质量发展。政策涵盖新产品新技术新服务应用示范补贴、创新产品推荐、标准制定修订资助、首台(套)设备推广奖励、研发项目与平台配套、高新技术企业补助、关键共性技术研发奖励、孵化器及众创空间建设奖励、在孵企业租金补贴、金融服务支持、政务服务优化及重大创新项目“一事一议”支持等方面。

# 奖励额度/补贴标准

1. **应用示范补贴**：对于购买新区企业示范应用产品、技术、服务的新区企业，根据实际合同额给予不超过20%的补贴，最高不超过200万元。
2. **标准制定资助**：
   - 参与国际标准、国家标准及行业标准制定的，按项目分别给予不超过100万元、50万元、30万元的资助。
   - 参与国际标准、国家标准及行业标准修订的，按项目分别给予不超过40万元、20万元、10万元的资助。
3. **首台(套)重大技术设备推广奖励**：
   - 对通过省级以上认定的项目，给予不超过100万元的一次性奖励。
   - 对获得工业和信息化部或河北省补贴的项目，按照实际投保年度保费的15%给予补贴，最高不超过100万元。
4. **研发项目与平台配套**：新区企业成功申报国家级重大研发项目或省级以上创新平台的，新区及相关企事业单位按照实际获得专项经费1:1配套资金支持。
5. **高新技术企业补助**：
   - 初次认定为国家高新技术企业的，一次性给予30万元补助。
   - 重新认定通过的国家高新技术企业，一次性给予10万元补助。
6. **关键共性技术研发奖励**：对新基建、智能城市建设及高端产业发展等领域的技术攻关项目，给予最高不超过500万元的研发补贴。
7. **孵化器及众创空间奖励**：
   - 对认定(备案)为国家级、省级科技企业孵化器、众创空间的，一次性给予50万元奖励。
   - 对未认定为国家级、省级但经认定为新区科技企业孵化器的，第二年满足认定标准及承诺条件的，可享受一次性50万元补助。
   - 按照孵化器毕业企业数量，每毕业一家给予10万元奖励。
8. **在孵企业租金补贴**：对孵化器内的在孵企业，以当年在新区办公的正式在职员工人数为基数，按照人均10平方米办公场所，2年内每年给予办公用房租金1.5元/平方米/天补贴，最高不超过100万元。

# 注意事项

1. **有效期**：本措施自印发之日起施行，有效期2年。
2. **适用范围**：高校、科研院所及其他创新机构支持政策参照本措施执行。
3. **认定要求**：部分补贴需经新区认定（如创新产品推荐名单、孵化器认定等）。
4. **重复享受限制**：首台(套)设备补贴针对获得工业和信息化部或河北省补贴的项目进行保费补贴，需注意与上级补贴的衔接。

# 申请对象

1. **新区企业**：符合《河北雄安新区规划纲要》产业发展重点方向，在新区辖区范围工商注册、税务登记并开展实际经营活动的北京非首都功能疏解企业和三县本地企业。
2. **新区企事业单位及创新机构**：参与标准制定、修订的主体。
3. **科技企业孵化器及众创空间**：包括国家级、省级及新区认定的孵化器、众创空间。
4. **在孵企业**：入驻经新区认定的孵化器内的企业。

# 申报条件

1. **主体资格**：
   - 符合《河北雄安新区规划纲要》产业发展重点方向。
   - 在新区辖区范围工商注册、税务登记并开展实际经营活动。
   - 属于北京非首都功能疏解企业或三县本地企业。
2. **具体事项条件**：
   - **应用示范**：需定期征集新产品、新技术、新服务，组织行业专家结合新区智能城市建设研究开发应用场景，开展应用示范。
   - **创新产品**：具有自主知识产权的创新产品和服务，经新区认定。
   - **标准制定/修订**：参与国际标准、国家标准及行业标准的制定或修订。
   - **首台(套)**：通过省级以上认定，或获得工业和信息化部或河北省补贴。
   - **研发项目/平台**：成功申报国家级重大研发项目或省级以上创新平台（包括省级学科重点实验室、省级企业重点实验室、省级技术创新中心、省级产业技术研究院、省级企业技术中心、省级工程研究中心、省级工业设计中心、省级中小企业公共服务平台）。
   - **高新技术企业**：初次认定或重新认定通过的国家高新技术企业。
   - **研发奖励**：围绕新基建、智能城市建设及高端产业发展等领域开展技术攻关。
   - **孵化器/众创空间**：认定(备案)为国家级、省级，或经认定为新区科技企业孵化器且第二年满足《河北雄安新区科技企业孵化器认定管理办法》认定标准及承诺条件。
   - **在孵企业**：在经新区认定的孵化器内，以当年在新区办公的正式在职员工人数为基数。
