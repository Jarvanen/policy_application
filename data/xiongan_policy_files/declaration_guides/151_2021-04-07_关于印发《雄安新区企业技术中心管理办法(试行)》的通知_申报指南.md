# 政策简介

本办法旨在强化企业在技术创新中的主体地位，支持、促进和规范雄安新区企业技术中心的建设和成长。企业技术中心是指企业适应创新发展和市场竞争需要设立的技术研发与创新机构，负责研究制定和组织实施企业技术创新战略规划；开展新产品、新技术、新工艺、新模式研发和应用；创造运用知识产权；建立完善技术标准体系；凝聚培养创新人才；开展协同创新；推进技术创新全过程实施。通过充分发挥政府引导作用，对技术创新能力强、创新业绩显著、具有重要示范作用的企业技术中心予以认定，并给予政策支持，鼓励企业不断提高自主创新能力，为升级为省级或国家级企业技术中心奠定基础。

# 奖励额度/补贴标准

1. **新认定奖励**：对新认定的雄安新区企业技术中心，雄安新区财政给予企业人民币10万元的资金奖励。
2. **运行评价奖励**：对评价结果为优秀的雄安新区企业技术中心，雄安新区财政给予企业人民币30万元的资金奖励。
3. **资金用途**：资金用于支持企业改善研发设施条件，购置科研仪器设备，开展新技术、新产品、新工艺研发、成果转化等相关工作。

# 注意事项

1. **认定频率与截止**：认定原则上每年组织一次，申请认定材料受理截止时间为当年4月30日。
2. **重复认定限制**：已获得雄安新区企业技术中心认定的企业，原则上不再受理其下属子公司企业技术中心认定申请。
3. **直接认定情形**：对于整体迁入雄安新区的企业（尤其是北京非首都功能疏解类企业），已被认定为国家或省级企业技术中心或分中心的，经必要材料审核后，直接认定为雄安新区企业技术中心或分中心。
4. **运行评价要求**：已认定2年及以上的雄安新区企业技术中心，须参加每两年组织一次的运行评价。
5. **材料份数**：上报的申请材料及评价材料均需一式五份。
6. **公示期**：拟认定名单公示期为5个工作日。

# 申请对象

1. 在雄安新区辖区内进行登记注册和纳税的企业法人。
2. 整体迁入雄安新区的企业（尤其是北京非首都功能疏解类企业），且已被认定为国家或省级企业技术中心或分中心的。

# 申报条件

1. **行业地位**：企业在雄安新区同行业具有显著的发展优势和竞争优势，研究开发与创新水平在雄安新区同行业中处于领先地位。
2. **机制与体系**：企业具有较为完善的技术创新机制和产学研合作机制，企业技术中心组织体系健全，发展规划和目标明确，技术创新成果丰富，成效显著。
3. **硬性指标**：
   - 年度研究与试验发展经费支出额不低于300万元；
   - 专职研究与试验发展人员数不少于30人；
   - 技术开发仪器设备原值不低于300万元。
4. **合规要求**：企业在申请受理截止时间前三年内，不得发生下列情况：
   - 因违反税收征管理法及有关法律、行政法规，构成偷税、骗取出口退税等严重税收违法行为受到行政、刑事处理；
   - 因违反海关法及有关法律、行政法规，构成走私行为，受到刑事、行政处罚，或因严重违反海关监管规定受到行政处罚；
   - 司法、行政机关认定的其他严重违法失信行为。

# 申报材料

1. **认定申请**：申请报告、认定数据表及必要的证明材料。
2. **运行评价**：年度工作总结、评价数据表及必要的证明材料。

# 办理流程

**认定流程：**
1. **上报**：企业根据管理办法及当年申报通知编写申请材料，向三县改革发展部门提出申请并上报。
2. **初审**：三县改革发展部门对申请材料进行初审，在规定时间内将推荐企业齐全的申请材料（一式五份）和推荐意见上报雄安新区改革发展局。
3. **初评**：雄安新区改革发展局委托第三方机构，按照认定评价指标体系对企业申请材料进行形式审查和初步评审，提出通过初评的企业技术中心名单。
4. **终评**：
   - 雄安新区改革发展局会同雄安新区税务局、雄安海关对通过初评的企业是否存在涉税违法情况进行核查，出具审查意见。
   - 根据初评结果和审查意见，由受委托的第三方机构组织专家评审，形成专家评审结果。
   - 雄安新区改革发展局根据专家评审结果进行综合终审，形成拟认定名单并公示（5个工作日）。
5. **认定**：对公示无异议的企业技术中心，由雄安新区改革发展局联合雄安新区税务局、雄安海关发文认定。

**运行评价流程：**
1. **上报**：企业根据管理办法及当年评价工作通知编写评价材料，向三县改革发展部门上报。
2. **初审**：三县改革发展部门对评价材料及附件完整性、真实性进行审查，在规定时间内将评价材料（一式五份）和审查意见上报雄安新区改革发展局。
3. **核查**：雄安新区改革发展局委托第三方机构，按照评价指标体系对评价材料进行评价，形成评价结果和评价报告。
4. **确认**：雄安新区改革发展局会同雄安新区税务局、雄安海关对评价结果和评价报告进行审核确认，对结果予以公布。
