# 政策简介

为贯彻落实河北雄安新区党工委管委会党政办公室印发《关于应对疫情支持服务业领域纾困发展的若干措施》的通知要求，加强企业用工保障，鼓励用人单位吸纳当地劳动力就业，实施用人单位一次性新增就业补贴。

# 奖励额度/补贴标准

1. **新区注册服务型企业**：补贴标准1000元/人。
2. **新区注册的其他类型企业**：补贴标准500元/人。
3. **参建企业(含劳务分包企业)**：补贴标准500元/人。

# 注意事项

1. **申领限制**：用人单位一次性新增就业补贴一年内同一人只可申领一次。
2. **排除对象**：不包括从事个体经营或创办小微企业和在民政部门注册的社会组织人员。
3. **时间要求**：
   - 政策有效期至2022年12月31日。
   - 申报结束日期为2023年4月10日。
   - 用人单位可在每月的前5个工作日内申报。
4. **材料补充**：申报材料与内容不完备的，一次性告知原因并退回补充申报。申请单位在2个工作日内补充申报材料与内容完备的，予以受理；逾期未补充申报或补充申报材料与内容仍不完备的，不予受理。
5. **违规处罚**：如有伪造证件材料、虚报冒领等欺瞒行为，将追回补贴资金，给予警告，没收违法所得，并处以罚款；在新闻媒体上曝光；情节严重触犯法律的，移交司法机关处理。
6. **资金拨付**：新增就业补贴拨付至用人单位数字货币账户。

# 申请对象

新区注册服务型企业、其他类型企业和参建企业(含劳务分包企业)。

# 申报条件

1. **平台注册与发布**：在产业互联网(http://120.53.124.140:10001)注册并通过平台发布岗位需求。
2. **招用时间**：在2022年5月1日-2022年12月31日期间新招用。
3. **人员户籍**：新招用新区三县户籍人员（以签订劳动合同时间为准）。
4. **合同与凭证要求**：
   - **新区注册服务型企业**：需与劳动者签订6个月以上期限劳动合同并提供3个月工资发放记账凭证或银行代发工资对账单。
   - **新区注册的其他类型企业**：需与劳动者签订6个月以上期限劳动合同并提供3个月工资发放记账凭证或银行代发工资对账单。
   - **参建企业(含劳务分包企业)**：需将合同、考勤、3个月以上的工资发放记账凭证或银行代发工资对账单录入雄安新区劳务实名制监管系统，并推送到河北省农民工工资支付监控预警平台。

# 申报材料

1. 《用人单位申请一次性新增就业补贴承诺书》；
2. 《用人单位一次性新增就业补贴申请表》；
3. 《用人单位一次性新增就业补贴人员花名册》；
4. 新增就业人员身份证、户口本复印件；
5. 为新增就业人员发放工资的凭证材料；
6. 劳动合同复印件(除参建企业、劳务分包企业外，劳动合同需在注册地劳动保障监察部门备案)。

# 办理流程

1. **申报**：用人单位通过产业互联网平台发布用工岗位，签订合同后，录入产业互联网平台，通过产业互联网平台按实际新增招用就业的人数，向注册地所在人社部门提出新增就业补贴申请。
2. **受理**：申报材料与内容完备的，予以受理；不完备的退回补充。
3. **审核**：核查申报用人单位新增就业补贴人员花名册、招聘岗位发布、新增就业人员劳动合同等信息，必要时通过电话、上门等方式核实。
4. **公示**：对审核通过的，三县人社部门公示5天。公示结束后，将《申请表》、《汇总表》报新区公共服务局。
5. **复核**：新区公共服务局复核新增就业情况，汇总审核后将加盖公共服务局公章的汇总表报送至新区改革发展局。
6. **拨付**：新区改革发展局根据公共服务局审核认定结果，将所需资金列入公共服务局部门预算，根据公共服务局资金申请按程序将新增就业补贴拨付至用人单位数字货币账户。

# 申报时间

- 政策有效期：至2022年12月31日
- 申报结束日期：2023年4月10日
- 每月申报窗口：每月的前5个工作日内
