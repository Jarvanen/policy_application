# 政策简介

为支持雄安新区创新驱动发展引领区建设，切实调动优势创新资源向雄安集聚支撑高质量发展，带动雄安各类创新主体创新能力实现跨越式提升，现公开征集科研储备任务。支持方向包括新一代信息与软件技术、生物与人口健康技术、新材料与高端制造技术、清洁能源与生态环保技术、前沿交叉和颠覆性技术等重点领域。

# 奖励额度/补贴标准

申请财政资金支持金额原则上不超过1000万元。其中，企业作为申报单位须对任务经费进行配套，自筹经费与申请财政科技经费比例原则上不低于1:1。

# 注意事项

1. 已获得财政经费支持的相关技术及研究内容，不得重复申报。
2. 申报单位须无不良信用记录，申报材料内容须真实客观有效，如存在虚假申报等情况，视为申报无效。
3. 雄安新区改革发展局未指定、授权或委托任何中介机构或个人从事科研储备任务的代理申报活动，代理机构或个人的行为与改革发展局无关。
4. 涉密任务，只报送纸质版。
5. 申报纸质版材料需按顺序装订成册，并加盖封面页公章与骑缝章，一式二份。
6. 电子申报材料需扫描成PDF文件，文件命名方式：单位名-任务名。

# 申请对象

原则上应为在雄安新区范围内注册具有独立法人资格的单位。

# 申报条件

1. 申报单位管理规范，业务及财务制度健全，具有相应的科研能力和条件，运行管理规范。
2. 任务单位具有满足任务正常实施所需的基本资金、基础和条件，有固定的场所，无知识产权纠纷。
3. 任务投资总额和资金来源应明确。
4. 申报任务2022年、2023年、2024年启动均可，任务执行周期原则上不超过3年。
5. 任务需具有明确的技术创新性和示范应用性，具有清晰、可量化的目标及考核指标。

# 申报材料

《雄安科技创新储备任务申报方案》。

# 办理流程

1. 线上申报：通过雄安新区科创服务区块链平台（雄安新区产业互联网平台：http://www.xaiip.org.cn）线上申报。
2. 邮件申报：将加盖公章后的《雄安科技创新储备任务申报方案》扫描成PDF文件，发送至雄安新区改革发展局邮箱：gfjggfwz@126.com。
3. 纸质申报：申报纸质版材料报送至雄安新区改革发展局公共服务组。

# 申报时间

通过平台长期开放申报，定期集中受理。2022年第一批任务自通知发布之日起至2022年7月15日，逾期不予受理。
