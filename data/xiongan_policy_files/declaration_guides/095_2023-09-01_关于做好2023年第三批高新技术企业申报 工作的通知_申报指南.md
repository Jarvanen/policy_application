# 政策简介

本通知旨在组织开展2023年第三批高新技术企业申报认定工作。根据河北省高新技术企业认定管理工作领导小组办公室要求，本年度受理企业申报材料截止日期为9月15日。政策涉及新申报企业认定及已认定高新技术企业的复审工作，要求三县科技管理部门积极组织辖域内企业开展申报认定。

# 申请对象

1. 拟申报2023年第三批高新技术企业的企业。
2. 2023年需进行复审的高新技术企业（名单见附件2）。

# 申报条件

1. 申报企业需具备河北省科技型中小企业资质或国家科技型中小企业资质，或已在平台申报相关资质。

# 申报材料

1. **电子版材料**：需在国家“高新技术企业认定管理工作网”（http://www.innocom.gov.cn/）提交。
2. **纸质版材料**：需逐项打印、装订成册，电子版材料需与纸质版一致。
3. **重新推荐说明**：第一批未通过企业再次申请的，需按照附件1填写修改说明，与纸质材料一并提交。
4. **未复审情况说明**：需复审高企如因企业经营状况、搬迁等原因未提交复审的，需出具情况说明（见附件3格式）。

# 办理流程

1. **网上申报**：申报企业需在国家“高新技术企业认定管理工作网”提交电子版申请。
2. **材料提交**：将装订成册的纸质版材料交至三县科技管理部门。
3. **市级审核**：县科技管理部门审核通过后，提交至新区改发局进行市级审核。
4. **复审通知**：三县科技管理部门需对2023年需复审高企通知到位。

# 申报时间

本年度受理企业申报材料截止日期为2023年9月15日。

# 注意事项

1. **中介机构选择**：申报企业自行选择符合条件的专项审计中介机构。
2. **告知承诺制**：根据《科技部关于高新技术企业认定有关证明事项实行告知承诺制的通知》有关要求，从今年起，高新技术企业申报材料中营业执照等注册登记证件以及专利证书等企业知识产权证件实行告知承诺制，申请企业自主选择是否适用告知承诺制办理。
3. **材料一致性**：电子版材料需与纸质版一致。
4. **复审要求**：需复审的高企名单见附件2，未提交复审的需出具情况说明。
