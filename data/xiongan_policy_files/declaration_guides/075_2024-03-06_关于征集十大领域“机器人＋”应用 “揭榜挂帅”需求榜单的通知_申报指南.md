# 政策简介

为落实《河北省支持机器人产业发展若干措施》，打造“机器人+”典型应用场景，推动机器人产业更好赋能于各行各业，省工信厅联合十部门组织开展十大领域“机器人＋”应用“揭榜挂帅”需求榜单征集活动。

# 申请对象

1. 在河北省依法设立的独立法人单位。
2. 雄安新区范围内有意提交相关“机器人＋”应用需求的单位。

# 申报条件

1. 经营状态正常、信用记录良好。
2. 具备相应的项目实施能力，具有完成项目所需的必要设施设备和良好的财务状况，能够为项目实施提供必要的资金保障。
3. 明确具体应用场景、技术要求和预期目标，具有可操作性和可评估性。
4. 应用场景应具有创新性和代表性，能够体现“机器人＋”的融合应用，可带动相关产业链企业创新发展，具有较好的社会效益和经济效益。
5. 愿意主动配合参与供需对接、公开宣传、展览展示等配套活动，推广案例经验。
6. 近3年无不良信用记录和重大违法行为。

# 申报材料

1. 《河北省“机器人+”应用“揭榜挂帅”需求榜单申报书》（WORD版、PDF盖章版）。
2. 申报单位营业执照复印件（必备）。
3. 申报单位“信用中国”查询记录（必备）。
4. 其他佐证材料。

# 办理流程

1. **分领域动员征集**：各市相关领域主管部门广泛动员各类主体积极参与申报，加强对申报单位的指导和支持，并对申报项目进行跟踪服务。
2. **申报单位填写申报书**：需求榜单申报单位根据征集范围、要求，填写《河北省“机器人+”应用“揭榜挂帅”需求榜单申报书》。
3. **汇总报送**：各市工信局牵头组织本地区需求榜单征集工作，汇总申报情况，对申报单位提交材料进行形式审查后，以公文形式将《河北省“机器人+”应用“揭榜挂帅”需求榜单汇总表》报省工信厅，并将《河北省“机器人+”应用“揭榜挂帅”需求榜单申报书》WORD版、PDF盖章版发至指定电子邮箱。

# 申报时间

1. 申报单位提交材料截止时间：3月15日前。
2. 各市工信局汇总报送截止时间：3月20日前。

# 注意事项

1. **材料格式要求**：
   - 申报书需如实、详细填写，除另有说明外，栏目不得空缺。
   - 文字排版要求：正文仿宋4号；一级标题黑体3号；二级标题楷体4号；纸张规格A4；页边距：上3.7cm，下3.5cm，左2.8cm，右2.6cm。
   - 需提交WORD版及PDF盖章版。
2. **所属领域范围**：制造业、农业、建筑、能源、商贸物流、医疗健康、养老服务、教育、商业社区服务、安全应急和极限环境应用10大应用重点领域。
3. **真实性承诺**：申报单位需对报送的资料真实性负责，自愿接受行业主管部门的检查、指导，报送资料可用于行业主管部门的分析研究与宣传报道。
4. **雄安新区特定要求**：新区范围内单位请将申报材料发送至邮箱：gfjxxcyz@163.com。
