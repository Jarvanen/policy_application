# 政策简介
本细则旨在支持雄安新区服务业中小微企业纾困发展，对新区范围内注册并依法纳税的服务业中小微法人企业（行业代码前两位为51-97，且符合中小微企业界定）在新区范围内银行机构申请的销售订单、应收账款等相关流动资金贷款给予贴息支持。贴息按照“先付后贴”、“免申即享”的方式实施，即企业按合同约定支付银行机构利息后，新区政府定期对符合条件的企业贴息。以个人名义贷款（包括企业法定代表人及配偶、企业主要股东、实际控制人等）用于经营的不列入贴息范围。

# 奖励额度/补贴标准
1. **贴息利率**：按照贷款合同签订当日执行的人民银行公布的一年期贷款市场报价利率（LPR）予以贴息；实际贷款利率低于LPR的，按照实际贷款利率予以贴息。
2. **贴息期限**：最多不超过2年。
3. **贴息总额**：每户企业贴息总额不超过30万元。
4. **适用时间段**：2022年4月29日至2022年12月31日之间发生的新增贷款（以提款日期为准）。

# 注意事项
1. **排他性规定**：各家企业同时只可享受一项贷款贴息政策。
2. **贷款类型限制**：仅支持销售订单、应收账款等相关流动资金贷款；以个人名义贷款（包括企业法定代表人及配偶、企业主要股东、实际控制人等）用于企业经营的不列入贴息范围。
3. **资金拨付方式**：通过数字人民币钱包完成资金拨付。
4. **政策有效期**：本细则有效期至2022年12月31日。
5. **动态调整**：贴息资金在预算安排总额下，视贴息进度和资金使用情况，新区改发局有权顺延贴息资金拨付或暂停、终止贴息。

# 申请对象
1. **主体类型**：新区范围内注册并依法纳税的服务业中小微法人企业。
2. **行业范围**：国家标准《国民经济行业分类》(GB/T4754-2017)中，行业代码前两位为51-97。
3. **规模界定**：符合《统计上大中小微型企业划分办法(2017)》中小微企业界定。
4. **信用要求**：企业应守法经营，未被相关行政主管部门列入失信联合惩戒名单。

# 申报条件
1. 企业需在新区范围内注册并依法纳税。
2. 企业属于服务业中小微企业（行业代码51-97，符合中小微企业界定）。
3. 贷款为在新区范围内银行机构申请的销售订单、应收账款等相关流动资金贷款。
4. 贷款发生时间在2022年4月29日至2022年12月31日之间（以提款日期为准）。
5. 企业未被列入失信联合惩戒名单。

# 申报材料
1. **线上提交**：企业登录雄安新区产业互联网平台，自主完善贷款信息，并将贷款合同等有关材料进行上传。
2. **无需额外报送**：完成上述信息录入后，企业无需再报送其他申报材料。
3. **银行及审核部门内部材料**（企业无需直接提交，但需配合提供）：
   - 贷款贴息明细表（附件1）
   - 付息有关证明材料
   - 《贷款贴息资金申请表》（附件2，由三县责任部门汇总填写）
   - 综合审计报告（由第三方审计机构出具）

# 办理流程
1. **完善信息**：企业主体登录雄安新区产业互联网平台（http://www.xaiip.org.cn），通过“用户中心”-“企业数据账户”-“贷款信息”或“惠企政策”-“服务业中小微企业贷款贴息”自主完善贷款信息，并上传贷款合同等有关材料。
2. **银行核准**：各银行定期对企业客户填写信息进行核对。核对无误的，提交产业互联网平台推送至三县进行审核；核准有错误的，请企业客户重新填写后提交。
3. **三县审核**：三县审核部门根据企业贷款信息协请对应贷款银行出具贷款贴息明细表、付息有关证明等材料，并聘请第三方会计师事务所开展审计工作。
4. **汇总及公示**：三县责任部门定期对本县各部门审核通过的企业拟贴息情况进行汇总，填写《贷款贴息资金申请表》，请第三方审计机构出具综合审计报告，报县政府研究同意，同步在新区产业互联网平台上进行公示（不少于5日）。
5. **资金拨付**：三县责任部门根据县政府研究情况，形成明确书面意见，以县政府名义向新区改发局提起贴息申请，附《贷款贴息资金申请表》、综合审计报告等有关材料，新区改发局按程序通过数字人民币钱包完成资金拨付。

# 申报时间
- **贷款发生时间**：2022年4月29日至2022年12月31日（以提款日期为准）。
- **政策有效期**：至2022年12月31日。
- **具体申报节点**：企业需定期登录平台完善信息，银行定期核对推送，三县定期审核汇总及公示。
